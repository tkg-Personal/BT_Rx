# BT Audio Rx — 実装方法ガイド

本ドキュメントでは、PCB設計から部品実装・テストまでの全手順を説明する。

---

## 1. 必要な工具・材料

### はんだ付け工具
- はんだごて（温度調節付き、推奨 300-350°C）
- はんだ（0.5mm径、鉛フリー推奨）
- フラックス（ペンタイプ or 液体）
- ピンセット（精密、ESD対策品）
- はんだ吸取線 or 吸取器（リワーク用）
- 作業用拡大鏡 or 顕微鏡（0402部品のため推奨）

### 検査・テスト用
- デジタルマルチメータ（テスター）
- USB-UART変換アダプタ（**3.3Vレベル品必須**、FT232RL系等）
- USB-Cケーブル
- LiPoバッテリー（3.7V、保護回路内蔵品、JST PHコネクタ付き）
- 3.5mmオーディオケーブル
- イヤホン or スピーカー
- Bluetoothオーディオ送信デバイス（スマートフォン等）

### ソフトウェア
- Autodesk Fusion Electronics（回路図 + PCBレイアウト）
- Microchip BM83 Config Tool（UART経由設定用）
- ウェブブラウザ（JLCPCB発注用）

## 3. PCB設計手順 (Fusion Electronics)

### 3.1 プロジェクト作成
1. Fusion 360を起動 → **ELECTRONICS** ワークスペースに切り替え
2. **New Electronics Design** でプロジェクト作成
3. プロジェクト名: `BT_Rx`

### 3.2 回路図作成
1. **New Schematic** で回路図ドキュメントを作成
2. ライブラリから部品を配置:
   - BM83SM1: カスタムライブラリとして作成が必要（50pin モジュール、フットプリント含む）
   - その他部品: Fusion標準ライブラリ or LCSC統合ライブラリから検索
3. `BT_Rx_仕様書.md` の「回路接続詳細」セクションに従って全ネットを接続
4. 電源ネット名を統一: `VBUS`, `BAT+`, `GND`, `SYS_PWR`
5. **ERC（Electrical Rules Check）** を実行し、エラーを全て解消

### 3.3 BM83カスタムライブラリ作成
BM83SM1はFusion標準ライブラリにないため、カスタム作成が必要:
1. **New Library** → シンボル作成（50pin、データシート Table 9-1 参照）
2. フットプリント作成（32×15mm モジュール、データシート Section 7 参照）
3. 3Dモデル: データシートの外形図から作成 or STEP入手
4. デバイス作成（シンボル + フットプリント紐付け）

### 3.4 PCBレイアウト
1. 回路図から **Switch to PCB** でPCBドキュメントへ移行
2. 基板外形を描画: **50mm × 30mm**
3. 部品配置（仕様書「部品配置ガイドライン」参照）:
   - U1 (BM83): 中央〜右寄り、アンテナ端が基板右端
   - J1 (USB-C): 基板上端
   - J2 (3.5mm): 基板下端
   - J3 (バッテリー): 基板左端
4. **アンテナクリアランス確認**: PCBアンテナ端から15mm以内に銅箔・部品を配置しない
5. 配線:
   - 電源トレース: 0.3mm以上
   - 信号トレース: 0.15mm以上
   - オーディオ配線: L/R対称、短く、他信号から離す
6. GNDベタ: B.Cu全面、F.Cuもできる限り
7. ビアスティッチング: モジュール直下のGNDプレーンに分散配置

### 3.5 DRC・検証
1. **DRC（Design Rules Check）** を実行
2. クリアランス違反・未接続ネットを全て解消
3. BM83リファレンス回路 (`docs/datasheets/BM83_reference_circuit.pdf`) と照合

### 3.6 製造ファイル出力
1. **CAM Processor** を開く
2. JLCPCB用ガーバーファイルを生成:
   - F.Cu, B.Cu, F.SilkS, B.SilkS, F.Mask, B.Mask, Edge.Cuts, Drill
3. 全レイヤーをZIPファイルにまとめる

## 4. JLCPCB発注手順

### 4.1 PCBのみ発注
1. [jlcpcb.com](https://jlcpcb.com/) にアクセス → **Order Now**
2. ガーバーZIPファイルをアップロード
3. 設定値:
   - Layers: 2
   - PCB Thickness: 1.6mm
   - PCB Color: 任意（緑が最安）
   - Surface Finish: HASL (Lead Free) or ENIG
   - Copper Weight: 1oz
   - PCB Qty: 5 (最小ロット)
4. レビュー画面でパターンを確認 → **Save to Cart** → 注文

### 4.2 PCBA（PCB + SMT実装）サービス
JLCPCBのSMT実装サービスを利用すると、SMD部品の実装を工場で行える。

1. ガーバーファイルアップロード後、**SMT Assembly** を有効化
2. 必要な追加ファイル:
   - **BOM (Bill of Materials)**: CSV形式。列: `Comment, Designator, Footprint, LCSC Part #`
   - **CPL (Component Placement List)**: CSV形式。列: `Designator, Mid X, Mid Y, Rotation, Layer`
3. Fusion Electronicsから BOM/CPL をエクスポート
4. LCSC品番を各部品に割り当て（LCSC在庫の **Basic Parts** は実装費無料）
5. 実装面を選択（Top / Bottom / Both）
6. プレビューで部品配置を確認 → 注文

**注意:**
- U1 (BM83SM1) はLCSC品番 C1558669 でBOMに含まれている
- 手はんだ部品（J2, JP2, C6, C7）はBOM/CPLに含めない
- DNP部品（D3, D4）はBOMに含まれるがJLCPCBで「未実装」指定する

## 5. はんだ付け手順

### 実装順序（推奨）
JLCPCB PCBAサービスでSMD部品は実装済み。手はんだが必要な部品のみ以下の順序で実装する。

**PCBA実装済み部品** (工場で実装):
R1-R4, C1-C5, C8, D1-D4, LED1, LED2, SW1, J1, JP1, U1

**手はんだ部品** (以下の順序で実装):
1. **C6, C7 (100µF電解コンデンサ)**: 極性注意（マイナス側=帯マーク）
2. **J2 (35RASMT4BHNTRX 3.5mmジャック)**: 基板裏面からはんだ付け
3. **JP2 (ピンヘッダ 1×6)**: 基板裏面からはんだ付け

### BM83実装の注意点
- BM83モジュールの裏面には大型GNDパッドがある
- ホットエアガン推奨（温度: 350°C、風量: 弱〜中）
- 手はんだの場合: 各ピンを個別にはんだ付け → GNDパッドはビアからはんだを流す
- **アンテナ部分にフラックスやはんだを付着させない**

## 6. BM83 Config Tool設定

### 6.1 UART接続
1. USB-UART変換アダプタ（**3.3Vレベル品**）を用意
2. 接続:
   - アダプタTXD → J4:pin2 (BM83 RXD)
   - アダプタRXD → J4:pin1 (BM83 TXD)
   - アダプタGND → J4:pin3 (GND)
   - J4:pin4 (BAT_IN) は接続しない（バッテリーまたはUSBから電源供給）
3. ボーレート: 115200bps (デフォルト)

### 6.2 Config Tool起動
1. MicrochipウェブサイトからBM83 Config Toolをダウンロード
2. PCにインストール → 起動
3. COMポートを選択 → **Connect**

### 6.3 必須設定項目
| 設定項目 | 推奨値 | 説明 |
|---------|--------|------|
| Audio Output Mode | **Single-ended** | Caplessではなく必ずSingle-endedに |
| Charge Current | 200-350mA | バッテリー容量に合わせて調整 |
| LED1 Pattern | BT Status | 接続状態表示 |
| LED2 Pattern | Charge Status | 充電状態表示 |
| Device Name | 任意 | Bluetooth表示名 |
| Audio Gain | デフォルト | 必要に応じて調整 |

### 6.4 Test Mode進入（ファームウェア書き込み時のみ）
1. J4:pin5 (P3_4) と J4:pin3 (GND) をジャンパピンまたはワイヤで短絡
2. J4:pin6 (RST_N) を一瞬GNDに接触させてリセット（またはJ4:pin6とpin3を一瞬短絡）
3. Test Modeに入ったことを確認
4. ファームウェア書き込み完了後、**J4:pin5のジャンパを必ず外す**

## 7. テスト手順

### 7.1 電源系テスト（部品実装直後、BM83実装前でも可）
1. USB-Cケーブルを接続
2. テスターでADAP_IN (pin 22パッド) の電圧を測定 → **4.75V〜5.25V** であることを確認
3. バッテリーを接続
4. BAT_IN (pin 23パッド) の電圧を測定 → **3.2V〜4.2V** であることを確認
5. **短絡チェック**: VBUS-GND間、BAT-GND間の抵抗を確認（短絡していないこと）

### 7.2 BM83動作テスト
1. バッテリーを接続（またはUSB-C接続）
2. MFBボタンを長押し（約3秒）→ LED点滅で起動確認
3. スマートフォンのBluetooth設定画面で **BM83のデバイス名** が表示されることを確認
4. ペアリング実行

### 7.3 オーディオテスト
1. 3.5mmイヤホンを接続
2. スマートフォンから音楽を再生
3. 左右チャンネルが正しく出力されていることを確認
4. ノイズ・歪みがないことを確認
5. 音量調整が機能することを確認

### 7.4 充電テスト
1. バッテリー接続状態でUSB-Cケーブルを接続
2. LED2（赤）が充電中パターンで点灯/点滅することを確認
3. 充電完了後、LED2パターンが変化することを確認
4. テスターでBAT_IN電圧が上昇していくことを確認（→ 最終4.2V）

## 8. トラブルシューティング

| 症状 | 考えられる原因 | 対処法 |
|------|-------------|--------|
| MFB長押しでも起動しない | バッテリー未接続/電圧不足、はんだ不良 | BAT_IN電圧確認、BM83のはんだ付け確認 |
| Bluetooth検出されない | ファームウェア未書き込み、アンテナ近傍に金属 | Config Toolで確認、アンテナクリアランス確認 |
| 音が出ない | オーディオモード未設定、C6/C7極性逆、はんだ不良 | Config ToolでSingle-ended設定確認、コンデンサ確認 |
| 片チャンネルのみ出力 | R3/R4/C6/C7のはんだ不良 | 該当チャンネルの部品確認 |
| ノイズが多い | GNDベタ不良、デカップリング不足 | GNDビア追加、C1/C2確認 |
| 充電されない | ADAP_IN未接続、USB-Cはんだ不良 | VBUS電圧確認、CC抵抗確認 |
| Config Tool接続不可 | UART TX/RX逆、電圧レベル不一致 | TX-RX入れ替え、3.3Vアダプタ確認 |
| Test Modeから抜けない | P3_4がLOWのまま | P3_4のジャンパを外してリセット |
| LED点灯しない | 極性逆、SYS_PWR未接続 | LED方向確認、SYS_PWR配線確認 |
