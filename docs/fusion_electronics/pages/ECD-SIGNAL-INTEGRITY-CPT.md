# 信号の整合性

Fusion Signal Integrity Extension

 この機能は拡張機能の一部です。拡張機能は Fusion の追加機能にアクセスするための柔軟な方法です。[詳細情報](https://help.autodesk.com/view/fusion360/JPN/?contextId=EXT-OVERVIEW)。

![信号の整合性 img](../images/concepts/ecd-signal-integrity.png)

信号の整合性は、USB、Bluetooth、wi-fi、RAM などの GHz の領域で動作する高速エレクトロニクスの主要な要素です。この領域で作業する場合、信号の整合性はデザイン上不可欠なものです。

信号の整合性を解析する最適なタイミングは、回路図をレイアウトし、コンポーネントを 2D PCB に配置した後です。これでボードのルーティングを開始する準備が整いました。この段階で、信号の整合性について検討を開始します。この機能をルーティング アクティビティ中に使用すると、作業の中でデザイン要件を維持することができるようになります。重要なフィードバックを得るために、ボードが完全にデザインされるまで待つ必要はありません。また、重要な変更があった場合は、いつでも解析を更新してその効果を確認することができます。

> **Note:** **注:** 主要な信号パスの一部であるポリゴンは、信号の整合性解析ではサポートされません。

## 解析結果を使用する

テーブルには、解析するパスの階層ビューが表示され、結果がレポートされます。パス行には、パス内のセグメントの集計値が表示されます。各セグメント行には、1 つのセグメントの結果が表示されます。

キャンバスでは、セグメントが右側のカラー チャートに示すように、セグメントのインピーダンス値に応じて色付けされています。

- 緑は、入力パラメータによって定義された目標値を表します。

- 青などの寒色系の色は、低いインピーダンスを示します。

- 赤などの暖色系の色は、高いインピーダンスを示します。

テーブル内で:

- パスまたはセグメントをダブルクリックして、キャンバス内でハイライト表示します。ビューがズームされ、パスまたはセグメントの範囲が表示されます。選択したオブジェクトが一時的に境界オーバーレイで囲まれます。

- キャンバス オーバーレイ ツールには次のものがあります。

![インピーダンス オーバーレイのアイコン](../images/icon/ecd-pcb-impedance.png) は、インピーダンス オーバーレイと凡例を表示します。

- ![オーバーレイ アイコンなしのアイコン](../images/icon/ecd-si-no-overlay.png) は、オーバーレイ表示をオフにします。

- テーブル ツールには、左から順に次のものがあります。

  ![結果テーブル ツール](../images/how-to/ecd-analyze-results-tools.png)

**解析を停止**: 解析を停止します。

- **テーブルを更新**: すべての結果が破棄され、テーブル内のパスを再解析します。ボードに変更が加えられたかどうかを表示します。

- **結果を削除**: テーブルの内容を削除します。

- **テーブルを展開**: 解析したパスのすべての結果行を表示します。

- **テーブルを折りたたむ**: 解析したパスのすべての結果行を非表示にします。

- **結果をエクスポート**: 結果データをファイルとして保存し、Fusion の外部で使用できるようにします。

- [結果]テーブルは次の情報を提供します。

**長さ**: セグメントまたはパスの直線的(ミル単位)な計測値です。

- **インピーダンス**: パスの場合、値は最小セグメント値から最大セグメント値までの範囲です。セグメントの場合、値は選択したセグメントのみに適用されます。矢印は、インピーダンス値が目標インピーダンスより高いか低いかを示します。

- **時間遅延**: 信号が一方の終了点からもう一方の終了点に移動するのにかかる時間を説明します。この値は、ピコ秒単位です。

- **抵抗**: 電気回路の電流に対する抵抗の指標です。この値は、オーム(Ω)で表されます。

- **静電容量**: 回路またはコンポーネントが電気電荷としてエネルギーを収集および蓄積する能力を表します。この値は、マイクロファラド(μF)で表されます。

- **インダクタンス**: 電流の流れによって生じるエネルギーを磁場として蓄積する能力を表します。この値は、ナノヘンリー(nH)で表されます。

## 解析の通知

解析をキャンセルすると、キャンセル時までのデータがテーブルに入力されます。不完全な解析が含まれるパスの横には、黄色の三角形のバッジが表示されます。解析を再実行するには、信号パスを右クリックし、**[解析を再実行]**をクリックします。コンテキスト メニューで**[信号をクリア]**を使用して結果を削除し、**[すべての子を折りたたむ]**を使用してテーブル内の絶対パスのみを表示します。

解析でエラーが返された場合は、エラーが発生したパスの横に赤いひし形のバッジが表示されます。パス行をダブルクリックして、キャンバスにパスを表示します。
