# ドキュメントの属性を管理する

このコマンドは、ユーザ定義のドキュメント属性に対して機能します。ドキュメント属性はスタンドアロンのテキスト文字列です。つまり、より大きなテキスト文字列にインラインで埋め込むことはできません。

- 回路図または PCB ドキュメント ツールバーの[属性]パネルで、 **[ドキュメント属性]** ![ドキュメント属性のアイコン](../images/icon/ecd/attributes-doc.png) をクリックします。

- [ドキュメント属性]ダイアログで、**[新規作成]**をクリックして属性を作成するか、属性を変更する場合は、属性を選択して**[変更]**をクリックします。このダイアログでドキュメント属性を削除することもできます。

![ドキュメント属性](../images/how-to/ecd-doc-attributes.png)

- 属性名、値、および表示設定を指定し、**[OK]**をクリックします。

- 必要な数の属性に対してこのプロセスを繰り返します。

- **[OK]** をクリックして、ドキュメント属性の追加を終了します。
