# 3D パッケージの作成を終了する

これは、3D パッケージを生成し、仕様パネルで詳細を確認した後で行います。

- [パッケージ 3D]ツールバーで、 **[終了]** ![終了のアイコン](../images/icon/ecd/finish.png) をクリックし、開いているライブラリにパッケージを追加します。

- [保存]ダイアログで、自動的に生成されたコンポーネント名を確認します。これには、技術的な詳細が含まれています。その名前を保持するか、必要に応じて変更し、**[保存]**をクリックします。パッケージおよび関連するフットプリントがライブラリに作成されます。

![SOIC のフットプリント](../images/how-to/SOIC-footprint.png)

- オブジェクトを表示するには、[コンテンツ マネージャ]パネルで[フットプリント]または[パッケージ]をクリックし、コンポーネント名をダブルクリックします。

![コンテンツ マネージャ](../images/how-to/content-mgr-head.png)

新しいパッケージのライブラリ コンテンツを完成させる場合は、記号を作成してから、記号、フットプリント、およびパッケージを結合するデバイスを作成します。
