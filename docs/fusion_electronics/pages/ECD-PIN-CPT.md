# [ピン]ツール

[ピン]ツールは、シンボル エディタで独自に使用されます。記号にピンを配置して、接続を定義します。

## Swaplevel の設定

- **0** は、回路図でピンを他のピンと入れ替えることができないことを意味します。

- **1** 以上の整数は、回路図でピンを同じ整数値の他のピンと入れ替えることができることを意味します。これには、異なるデバイス間での入れ替えも含まれます。
