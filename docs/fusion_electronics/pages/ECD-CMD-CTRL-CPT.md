# コマンド ライン制御

Fusion ツールバー上のツールをクリックする代わりに、コマンド ラインからツールを呼び出すこともできます。入力では、大文字と小文字は区別されません。コマンドを入力するときに、他のキーワードと間違えられない限り、キーワードを省略することができます。例えば、次の通りです。

CHANGE WIDTH 0.024

値の実際の単位は[GRID]メニューで設定します。現在設定されているグリッドを変更せずに、コマンド ラインで直接単位を指定することもできます。

CHANGE WIDTH 0.6MM

または

cha wid 24mii

ほとんどのコマンドは、コマンド ラインで座標値を宣言しながら実行できます。

例

MOVE IC1>VALUE (2.50 1.75) ;

パーツ IC1 の値のプレースホルダ テキストは、レイアウトでは位置 2.50 1.75 に移動します。

MIRROR U1;

パーツ U1 は基板の下側にミラー化されます。

HOLE 0.15 (5 8.5) ;

ドリル直径が 0.15 の穴を位置 5 8.5 に配置します。

VIA 'GND' 0.070 round (2.0 3.0) ;

信号 GND に属する直径 0.070 の丸い形状のビアが、位置 2.0 3.0 に配置されます。

## 履歴機能

[Crsr-Up] ()または[Crsr-Down] ()を押して、直前に入力したコマンドを呼び出し、コマンドを編集します。[Esc]キーを押すと、コマンド ラインの内容が削除されます。

## ファンクション キー

[Alt]、[Ctrl]、および[Shift] (Mac OS-X ではさらに[Cmd])を使用するファンクション キーおよびそれらのキーの組み合わせに、テキストを割り当てることができます。ただし、オペレーティング システムまたは Linux ウィンドウ マネージャで使用されていない場合に限ります(たとえば、[F1]はヘルプに使用します)。ファンクション キーを押した場合、キーボードからテキストを入力した場合に対応します。すべてのコマンドはテキストとして入力できるため、特定のパラメータを持つ各コマンドをファンクション キーに割り当てることができます。この方法で、一連のコマンド全体をファンクション キーに割り当てることもできます。

コマンド

ASSIGN

は、現在のファンクション キーの割り当てを表示します。キー割り当てに対する変更は、割り当てウィンドウで実行できます。[新規]ボタンを使用すると、新しいキー割り当てを定義できます。[削除]をクリックするとマークされたエントリが削除され、[変更]をクリックすると既存の定義が変更されます。[OK]をクリックするとダイアログが閉じ、定義が保存されます。[キャンセル]をクリックするとダイアログが中止されます。

これらの設定は、[回路図エディタ]または[レイアウト エディタ]の[オプション]/[割り当て]メニューでも行うことができます。

例:

[Ctrl]+[Shift]+[G]を押すと、0.127 mm のグリッドが表示されます。

ASSIGN CS+G 'GRID MM 0.127 ON;';

[Alt]+[F6]を押すと、画層が Top に変更され、ROUTE コマンドが起動します。

ASSIGN A+F6 'LAYER TOP; ROUTE';

[Alt]+[R]を押すと、画層 Top、Pads、Vias、および Dimension のみが最初に表示され、既定のプリンタを使用して印刷が開始されます。

ASSIGN A+R 'DISPLAY NONE 1 17 18 20; PRINT;';

[A]、[C]、[M]、[S]は、[Alt]、[Ctrl]、[Cmd] (Mac OS-X のみ)、および[Shift]キーの修飾キーです。

[Alt]+[0]を押すと、コントロール パネルが前景に表示されます。[Alt]+[1]～[9]の組み合わせは、各タイトル バーに表示されるウィンドウ番号に従って、さまざまなエディタ ウィンドウに割り当てられます。

## スクリプト ファイル

スクリプト ファイルは強力なツールです。スクリプトには、一連の多数のコマンドを含めることができます。たとえば defaultcolors.scr では、すべての画層に対する特定の色と塗り潰しパターンを指定できます。一方、他のプログラムのデータから変換されたネットリストが含まれている場合もあります。

SCRIPT コマンドは、スクリプト ファイルを実行するために使用されます。

多くのユーザ言語プログラム(ULP)が、レイアウトや回路図を修正するために読み込むことができるスクリプト ファイルを作成します。

Electronics は、たとえば EXPORT コマンド([スクリプト]オプション)を使用したスクリプト ファイルとして、ライブラリ全体を出力します。このファイルをテキスト エディタで修正してから、再度読み込むことができます。これにより、ライブラリに簡単に変更を加えることができます。

スクリプト ファイルとエクスポート コマンドの詳細については、この章の後半で説明します。

## 混合入力

コマンドを指定するさまざまな方法を組み合わせることができます。

たとえば、CIRCLE コマンドのアイコン(コマンド ラインに CIRCLE と入力した場合に対応)をクリックし、次のように、コマンド ラインで円の中心と円周上の点の座標を入力します。

(2 2) (2 3) ←

上記の値を使用すると、単位が現在インチに設定されている場合、中心座標が(2 2)の半径 1 インチの円になります。CIRCLE コマンドをアイコンまたはコマンド ラインのどちらで入力するかは関係ありません。

> **Note:** 
注:

 一部の Electronics コマンドは、[Shift]キー、[Alt]キー、または[Ctrl]キーと組み合わせて使用されます。Mac OS-X 用の Electronics を使用している場合は、[Ctrl]ではなく[Cmd]キーを使用してください。
