# 別のライブラリからコンポーネントをインポートする

コンポーネントは、他のアクティブなライブラリからカスタム ライブラリにインポートすることができます。 

- ツールバーの **[インポート]** グループで、 **[コンポーネントをインポート]** 、 **[記号をインポート]** 、または **[フットプリントをインポート]** を選択して、[インポート]ダイアログを表示します。

3D パッケージをインポートする場合は、 **[フットプリントをインポート]** を選択し、ダイアログの右下にある **[接続された 3D モデルを読み込み]** がオンになっていることを確認します。

- 1 つまたは複数のオブジェクトを選択し、 **[OK]** をクリックして開いているライブラリに追加し、ダイアログを閉じます。

- ライブラリで名前が重複している場合は、アセットに新しい名前を付けるよう求められます。
