# ポリゴンを描く

このコマンドは、ソリッド ポリゴンを作成します。

- [回路図ドキュメント]ツールバーで、 **[ポリゴン形状]** ![ポリゴンのアイコン](../images/icon/ecd/polygon.png) をクリックします。

- ダイアログ ボックスで、コーナーのスタイル、半径、およびマイタを確認して調整します。

- クリックして描画を開始し、クリックして各コーナーを配置し、ダブルクリックして描画の終端をマークし、ポリゴンを閉じます。
