# ドキュメントを同期する

2 つのデザインの違いを検出して処理します。注: 現時点では、シンクロナイザはモジュールではサポートされていません。

- Electronics で両方のドキュメントを開きます(通常は回路図と回路基板)。

- 回路図ドキュメントをクリックし、[回路図検証]ツールバーで、**[同期]** ![同期のアイコン](../images/icon/ecd/synchronize.png) をクリックします。

- [シンクロナイザ]ダイアログで、回路図がソース ドキュメントであり、回路基板がターゲットであることに注意してください。

- 同期が実行された場合にターゲット ドキュメントで追加、削除、または修正されるコンポーネントとネットを示す列を確認します。

- チェック ボックスを使用して、ターゲットに追加しないコンポーネントの選択を解除するなどの、同期に必要な変更を加えます。

- 変更リストに問題がなくなったら、ダイアログで[実行]をクリックして、変更をターゲットに反映します。
