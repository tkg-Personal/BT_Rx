# ライブラリ エディタ

[新しいライブラリを作成する](ECD-CREATE-CSTM-LIB-TSK.htm)か、[既存のライブラリを開く](ECD-EDIT-CSTM-LIB-TSK.htm)と、ライブラリ オーサリング作業スペースが表示されます。ここで、**コンポーネント**、**記号**、**フットプリント**、**3D パッケージ**などの新しいアセットの作成を開始できます。また、他のライブラリまたはローカルの .lbr ファイルから現在のライブラリに[コンポーネントをインポートする](ECD-IMPORT-COMPONENTS-CPT.htm)こともできます。

## コンテンツ マネージャ

アセットを作成、表示、編集するには、ライブラリ オーサリング作業スペースの左側にある **[コンテンツ マネージャ]** パネルを使用します。任意のアセットを選択すると、[コンテンツ マネージャ]パネルの下部に詳細が表示されます。アセットをダブルクリックすると、対応するエディタが作業スペースの中央に表示され、編集を行うことができます。
