# アセンブリ バリアントを管理する

回路図ドキュメントでは、次の方法でアセンブリ バリアントにアクセスし、修正または削除することができます。

## デザイン マネージャの固定パネル

   ![アセンブリ バリアントへのアクセス](../images/how-to/ecd-assembly-variant-access.png)

- キャンバスの右側で、[デザイン マネージャ]固定パネルに切り替えます。

- **[アセンブリ バリアント]** リストからバリアントを選択します。**レイアウト エディタ**では、このフィールドは情報の表示のみに使用します。

- アセンブリ バリアントを管理するには、 **[アセンブリ バリアント]** ![バリアントのアイコン](../images/icon/variant-icon.png) をクリックします。

## ドキュメント ツールバー

- [回路図ドキュメント]の **[ドキュメント]** ツールバーで、 **[アセンブリ バリアント]** ![バリアントのアイコン](../images/icon/variant-icon.png) をクリックします。

> **Note:** 注: **[アセンブリ バリアント]** ドロップダウン リストの一番上の項目を選択して、既定のバリアント、または元のバリアントを使用します。
