# バスを追加する

- [回路図デザイン]ツールバーで、**[接続] > [バスを追加]** ![バスを追加のアイコン](../images/icon/ecd/add-bus.png) をクリックします。

- [バスを追加]ダイアログで、曲げ、線種、半径、およびマイタの図面オプションを選択します。  

- 回路図デザインでバス配線を描画します。ダブルクリックして図面を終了します。

- [バス]ダイアログで、事前定義されたバスを選択するか、**[デザイン内]**を選択してデザインで使用可能なネットから新しいバスを設定します。

- バスの**[エイリアス]** (短い名前)を確認し、必要に応じて変更します。

- ダイアログの下部パネルで、ネットを**[使用可能]**セットから**[含む]**セットに移動するか、逆方向に移動して、バスを使用するネットを指定します。含めるネットが**[仕様]**フィールドに表示されます。完了したら、[OK]をクリックします。

- ネットをバスからピンに接続するには、バスの端で、または水平または垂直セグメントに沿った任意の場所で、右クリックして**[バスをブレークアウト]**を選択します。

- すべてのネット、選択したネット、または単一のネットのいずれかを配線するかを選択し、適切なピン(1 つまたは複数)に接続します。
