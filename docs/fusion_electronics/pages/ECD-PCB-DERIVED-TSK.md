# PCB を派生する

アクティブなデザインのスケッチ プロファイルから PCB を派生するには、 **[PCB を派生]** ツールを使用して、3D PCB のアウトライン スケッチの代わりにこの外部スケッチを使用します。  

- **[デザイン]** 作業スペースで、デザインを保存します。

- **[作成] > [PCB を作成] > [PCB を派生]** をクリックします。

 **[PCB を派生]** ダイアログが表示されます。

- **[PCB を派生]** ダイアログで、スケッチ プロファイルを選択します。  

- スケッチ プロファイルから点をクリックして、原点を選択します。

- スケッチ プロファイルからスケッチ線分またはエッジをクリックして、**X** 軸を選択します。

- **[X 軸を反転]** をクリックして、**X** 軸の方向を反転します。

- **[Z 軸を反転]** をクリックして、**Z** 軸の方向を反転します。

> **Note:** 注: **Y** 軸は直接選択されず、**X** 軸および **Z** 軸から推測されます。
