# レイヤー セット マネージャを使用する

カスタム画層セットを作成または修正します。レイヤー セットには次の 3 つのタイプがあります。

- **[読み取り専用のレイヤー セット]**は、便利に使用できるように用意されている定義済みのレイヤー セットです。

- **[カスタム レイヤー セット]**は、いくつかの例が用意されている編集可能なレイヤー セットです。ワークフローのニーズに合わせてレイヤー セットを作成します。

- **[編集可能なレイヤー セット]**は、レイヤーを追加または除去できる定義済みのシルクスクリーンのレイヤ セットです。2 つのレイヤー セットである Top Silkscreen と Bottom Silkscreen には、CAM Preview および 3D PCB ドキュメントに表示される可視コンテンツを持つレイヤーのみが含まれます。これらのセットにレイヤーを追加すると、追加されたレイヤー内の表示オブジェクトが表示されます。

> **Note:**  注: CAM プレビューと CAM プロセッサは異なるツールであり、1 対 1 に対応するものではありません。CAM プロセッサ テンプレートは、ジョブに書き出される内容を定義します。

![画層ツール](../images/how-to/ecd-layer-sets-mgr.png)

## 新しいレイヤー セットを追加する

- [画層を表示]パネルで、 **[画層セット マネージャ]** ![画層ツール](../images/icon/ecd-lyr-sets-mgr.png) をクリックします。

- ダイアログのペインの下部で、[+]をクリックしてレイヤー セットを追加します。

- レイヤー セットをダブルクリックして、既定の名前を変更します。

- [使用可能なレイヤー]ペインで、セットに追加するレイヤーをクリックします。

- 完了したら、[完了]をクリックします。

## レイヤー セットを修正する

- [画層を表示]パネルで、 **[画層セット マネージャ]** ![画層ツール](../images/icon/ecd-lyr-sets-mgr.png) をクリックします。

- ダイアログで、修正するカスタム レイヤー セットを選択します。

- [セット]ペインの[レイヤー]で、レイヤー行の[X]をクリックして、セットからレイヤーを削除します。

- [使用可能なレイヤー]ペインで、セットに追加するレイヤーをクリックします。

- 完了したら、[完了]をクリックします。

## レイヤー セットを削除する

- [画層を表示]パネルで、 **[画層セット マネージャ]** ![画層ツール](../images/icon/ecd-lyr-sets-mgr.png) をクリックします。

- ダイアログで、削除するカスタム レイヤー セットを選択します。

- [レイヤー セット]ペインの下部にある[X]をクリックし、プロンプトに対して[OK]をクリックします。選択したレイヤー セットが削除されます。
