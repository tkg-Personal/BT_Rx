# 電子デザイン ライブラリの概要

Fusion 電子デザイン ライブラリには、電子デザインの作成に必要なコンポーネントが含まれています。Fusion のパブリック ライブラリはすべてのユーザが利用できますが、ハブライブラリを作成して、独自に使用したり、チームと共有することもできます。

各ライブラリ コンポーネントは、同じオブジェクトの異なる表現の一部またはすべてで構成されるデザイン オブジェクトです。

- **記号**: 回路図デザイン ドキュメントで使用

- **フットプリント**: 2D PCB レイアウト ドキュメントで使用

- **3D パッケージ**: 3D PCB レイアウト ドキュメントで使用

これらの異なる表現に加えて、コンポーネントには、同じ回路図シンボルの代替となる 2D および 3D 表現である複数の**バリアント**が含まれる場合があります。

## パブリック ライブラリ

パブリック ライブラリは、ユーザ自身が所有しているものでない限り、読み取り専用であり、Fusion またはサードパーティによって提供されます。これらのライブラリからコンポーネントをデザインで使用することはできますが、編集することはできません。パブリック ライブラリは、リモート アクセスとライブラリの共有を容易にするオートデスクのサーバである Library.io に格納されています。

## ハブ ライブラリ

記号、フットプリント、またはパッケージを編集する場合は、それらを格納して編集するためのハブ ライブラリを作成する必要があります。ハブ ライブラリに新しいコンポーネントを作成したり、他のライブラリからコンポーネントをインポートしたりすることができます。デザインで使用するハブ ライブラリ ドキュメントは、データ パネル内のプロジェクト フォルダから簡単に開くことができます。
