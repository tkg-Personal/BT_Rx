# 関連付けのある PCB を作成する

- **[ファイル] > [新規デザイン]** をクリックします。

- [デザイン]作業スペースで、**[作成] > [スケッチを作成]**をクリックし、スケッチ ツールを使用して 2D ボードのアウトラインを作成します。

- ツールバーの右端にある**[スケッチを終了]**をクリックします。

- **[ファイル] > [保存]**をクリックし、スケッチをプロジェクトの場所に保存します。

- **[作成] > [PCB を作成] > [関連付けのある PCB を作成]** ![3D PCB を派生のアイコン](../images/icon/ecd/derive-pcb-sketch.png) をクリックして、スケッチを選択します。

- **[関連付けのある PCB を作成]**ダイアログで、必要に応じて原点を設定し、X 軸を指定するか、X 軸または Z 軸を反転します。

- [OK]をクリックして完了します。3D PCB が作成され、作業スペースが Electronics 3D PCB に変わります。ここで 2D PCB にリンクして、ボードに穴を追加できます。
