# 単一レイヤー ビューを切り替える

すべての表示レイヤーをフル カラーで表示する方法から、アクティブなレイヤーのみをフル カラーで表示し、他のすべてのレイヤーをグレースケールで表示する方法に切り替えます。また、元に戻します。

[PCB 設計]ツールバーで、**[単一レイヤー ビューを切り替え]** ![単一レイヤー ビューを切り替えのアイコン](../images/icon/ecd/single-layer-on.png) をクリックします。アクティブなレイヤーはフル カラーのまま、他のレイヤーはすべてグレースケールになります。再度クリックすると、すべてのグレースケール レイヤーがフル カラーに復元されます。
