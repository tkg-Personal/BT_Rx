# コンポーネントに名前を付ける

コンポーネントは、回路図に追加されたときに自動的に名前が付けられ、順番に番号が付けられます。たとえば、ヒューズ F1 と F2 です。コンポーネントの名前を変更した場合、後で追加したコンポーネントはすべて自動的にその命名スキームを使用し続けるため、完成したデザインのコンポーネント名を確認して調整することが必要になる場合があります。
