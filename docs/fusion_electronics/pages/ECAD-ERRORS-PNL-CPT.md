# 電子デザインのエラー パネル

**[エラー]** パネルはデザイン ペインの右側にあり、デザイン内のエラーを確認して解決することができます。パネルにはエラーと警告が一覧表示され、エラーをクリックするとボード上の場所を確認できます。

**[エラー]** パネルは、次の作業時に使用できます。

- 回路図かいろず

- 回路基板
