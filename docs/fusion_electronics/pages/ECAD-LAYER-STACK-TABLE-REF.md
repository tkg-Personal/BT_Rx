# レイヤー スタック テーブルの参照

レイヤー スタック テーブルは、PCB 設計と製造における重要なコンポーネントです。

## テーブル

**[レイヤー スタック]** と **[ビア ペア]** のいずれかを選択します。

### レイヤー スタック

PCB で使用されるさまざまなレイヤーについて、その材料、厚さ、用途などの詳細情報を提供します。

### ビア ペア

ビア ペアによって作成された接続をドキュメント化します。

## [タイトル]

レイヤー スタック テーブルのタイトルを入力できます。

## 列

次の特定の列の表示設定を切り替えます。

- **レイヤー #**

- **名前**

- **材料**

- **[タイプ]**

- **[厚さ]:**

- **[線幅]**

- **開始点**
