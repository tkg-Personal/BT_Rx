# デザイン ルール カテゴリ

Fusion では、デザイン ルールをそのタイプに基づいて分類します。各カテゴリには、**ライブ DRC** と**手動 DRC** の 2 つのチェック レベルがあります。既定では、Fusion は**手動 DRC** のすべてのルール カテゴリを有効にします。

Fusion はルール カテゴリを単項型および二項型に分割します:

- 単項ルール: 銅箔幅やドリル サイズなど、単一のオブジェクトに適用します。

- 二項ルール: 銅箔クリアランスや基板のアウトライン クリアランスなど、2 つのオブジェクトに適用します。

## ルール カテゴリ レベル

### ライブ DRC

有効にすると、Fusion はライブ DRC の実行中にルール カテゴリをチェックします。これは、データ モデルが変更されたときのインタラクティブなワークフローの最後に行われます。

### 手動 DRC

有効にすると、Fusion は**手動 DRC** の実行中にルール カテゴリをチェックします。手動 DRC は、次のいずれかの方法で実行できます。

- コマンド ライン インタフェース(CLI)に DRC と入力します。

- ルール ツールバーの**[デザイン ルールをチェック]**をクリックします。
