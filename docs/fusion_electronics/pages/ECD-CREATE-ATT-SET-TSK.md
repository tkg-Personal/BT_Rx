# コンポーネント バリアントの新しい属性セットを作成する

新しい属性セットを作成して、異なるテクノロジや調達オプションなど、コンポーネント バリアントの代替プロパティ設定を定義します。

## 前提条件

- [[属性]ダイアログを開きます](ECD-OPEN-ATT-DIALOG-TSK.htm)。

- 目的のコンポーネント バリアントの属性が表示されていることを確認します。

## 手順

- [属性]ダイアログで、左下隅にある **[新規属性セット]** をクリックします。

- 新規属性セットの名前を入力します。この名前は、コンポーネント名およびバリアント名との組み合わせにより、完全なパーツ名を形成します([既定]が選択されていない場合)。

- (省略可能)既存の属性セットから属性をコピーする方法を選択します。必要に応じて、属性名、属性値、またはその両方をコピーできます。

**属性値**: 現在のバリアントに既に存在する属性の属性値のみをコピーします。

- **属性名のみ**: 値はコピーせずに、別のセットから属性をコピーします。

- **属性名と値**: 現在のバリアントに存在しない属性を含め、すべての属性とその値をコピーします。

- [OK]をクリックして属性セットを作成します。

## 次のステップ

- **属性値を編集**: [属性]ダイアログのセルをダブルクリックして、属性値を変更します。
