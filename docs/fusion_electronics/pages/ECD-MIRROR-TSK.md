# ミラー

**[ミラー]** ツールは、オブジェクトを y 軸を中心に反転させます。

- 開かれている電子デザイン回路図または 2D PCB ドキュメントの **[デザイン]** ツールバーで、 **([修正] > [ミラー])** ![ミラーのアイコン](../images/icon/mirror-icon.png) をクリックします。

- キャンバスで、ミラー化するオブジェクトをクリックします(左から右に反転)。

> **Note:** 注: オブジェクトを再度クリックすると、元の方向に戻ります。

- 必要に応じて別のオブジェクトをクリックしてミラー化します。

- 終了したら、キーボードの[Esc]を押してツールを終了します。
