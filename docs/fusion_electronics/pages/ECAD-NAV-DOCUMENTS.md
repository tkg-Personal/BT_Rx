# 電子デザインのドキュメント間をナビゲートする

ドキュメント間をナビゲートして、単一の電子デザイン内で回路図、2D PCB、3D PCB ビューをシームレスに切り替えることができます。これにより、既に行った作業が失われたり、ドキュメント間の連携が途切れたりすることなく、さまざまなデザイン段階で作業を行うことができます。

Fusionの[電子デザイン]作業スペースでは、3 つのドキュメント タイプすべてのリンクが維持されます。これらのビューの間をいつでも移動して、デザインを更新することや、変更が他のビューにどのように影響するかを確認することができます。

## ドキュメントの切り替えの仕組み

ドキュメントを切り替えるには、次の 2 つの方法があります。

- **ドキュメント タブ:** Fusion ウィンドウの上部にあるタブをクリックします。

- **アクションを切り替える:** 各ドキュメント タイプのデザイン ツールバー ボタンを使用します。

まだ存在しないドキュメントに切り替えると、Fusion によって自動的にドキュメントが作成され、電子デザインにリンクされます。

## ドキュメント タイプ別にオプションを切り替える

### 回路図からの切り替え

- **2D PCB への切り替え:** リンクされている 2D PCB ドキュメントが開きます。2D PCB が存在しない場合は、Fusion によって自動的に作成され、リンクされます。

### 2D PCB からの切り替え

- **回路図への切り替え:** リンクされている回路図ドキュメントが開きます。

- **3D PCB への切り替え:** リンクされている 3D PCB ドキュメントが開きます。3D PCB が存在しない場合は、Fusion によって自動的に作成され、リンクされます。これは、従来の[3D PCB にプッシュ]ワークフローの代替となる機能です。

### 3D PCB からの切り替え

- **2D PCB への切り替え:** リンクされている 2D PCB ドキュメントが開きます。

## 切り替え時の動作

ドキュメントに切り替えると、次のようになります。

- **ドキュメントが既に開いている場合:** Fusion で、ドキュメントが前面に移動されます。

- **ドキュメントは存在するが、開いていない場合:** Fusion でドキュメントが自動的に開きます。

- **ドキュメントが存在しない場合:** Fusion で新しいドキュメントが作成され、電子デザインにリンクされます。
