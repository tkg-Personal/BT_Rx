# ポリゴン選択を使用してオブジェクトを選択する

- 電子デザイン ドキュメントの **[デザイン]** ツールバーで、**([選択] > [ポリゴン選択])** ![グループのアイコン](../images/icon/select/polygon-select.png) をクリックします。

- キャンバスで、ポリゴンを描画する開始点をクリックします。

- 開始点に戻るまで、含めるオブジェクトの周囲の点をクリックし続けます。

- 右クリックしてポリゴン選択を完了します。
