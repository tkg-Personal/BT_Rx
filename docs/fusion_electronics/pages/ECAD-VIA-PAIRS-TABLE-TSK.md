# ビア ペア テーブルを追加する

Fusion の **[図面]** 作業スペースに、 **[ビア ペア]** に基づくテーブルを追加します。

- **[図面]** 作業スペースの **[電子デザイン]** ツールバーで、 **[作成] > [レイヤー スタック テーブル]** ![レイヤー スタック テーブルのアイコン](../images/icon/layer-stack-table.png) をクリックします。

 **[レイヤー スタック テーブルを挿入]** ダイアログが表示されます。

- **[レイヤー スタック テーブルを挿入]** ダイアログで、 **[テーブル]** ドロップダウン リストから ** [ビア ペア]** を選択します。

- 省略可能:

ビア ペア テーブルのタイトルを変更します。

- 特定の列の表示設定を切り替えます。

- 必要に応じて、 **[レイヤー スタック テーブル]** ダイアログの **[上]** および **[下]** 矢印を使用して、列の順序を変更します。

- クリックしてビア ペア テーブルをシートに配置します。

 ビア ペア テーブルがシートに表示されます。
