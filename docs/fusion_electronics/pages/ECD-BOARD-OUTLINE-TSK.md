# 3D PCB でボード アウトラインを編集する

**ボード アウトライン スケッチ**は、**PCB** の外形寸法を定義するために使用されます。このレイヤーは、2D PCB の**ボード アウトライン レイヤー**と同期します。

- **[電子デザイン]** 作業スペースの開いている **3D PCB** ドキュメントで、次のいずれかを実行します。

**3D PCB** 作業スペースで、 **[作成] > [ボード アウトラインを編集]** をクリックします。

- タイムラインで、キャンバスの下にある**アウトライン** スケッチを見つけてクリックし、アクティブにします。

- **ブラウザ**で、**スケッチ** フォルダを展開して、**アウトライン** スケッチを見つけます。

- 初めてスケッチを編集する場合は、スケッチを開始する最初の平面または面を選択します。

 有効な選択肢には次のものがあります。

基準平面

- 作業平面

- ジオメトリ上の平面

- スケッチ ジオメトリを描画して、閉じたプロファイルを作成します。

- [スケッチを終了]をクリックします。

## ヒント

アセンブリ コンテキストからジオメトリを参照するには、 **[プロジェクト]** ツールを使用します。
