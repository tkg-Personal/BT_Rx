# オブジェクトをロックする

- [PCB 設計]ツールバーで、 **[修正] > [ロック] > [ロック]** ![ロックのアイコン](../images/icon/ecd/lock.png) をクリックします。

- アイテムをクリックすると、所定の位置にロックされます。その他の使用可能なコマンド:

ロックされたアイテムをロック解除するには、[Shift]キーを押しながらクリックします。

- グループをロックするには、[Ctrl]キーを押しながら右クリックします。
