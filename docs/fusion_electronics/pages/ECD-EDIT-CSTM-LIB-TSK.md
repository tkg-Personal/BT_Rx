# 既存の電子デザイン ライブラリを編集する

既存の電子デザイン ライブラリを開いて編集し、コンポーネントの変更、新しいパーツの追加、ライブラリ コンテンツの更新を行います。ライブラリには、データ パネルまたはライブラリ マネージャからアクセスできます。

## 前提条件

- **ハブまたはプロジェクトへのアクセス権:** 編集する電子デザイン ライブラリを含むハブまたはプロジェクトへのアクセス権を持っている必要があります

## 手順

- 次のいずれかの方法を選択して、ライブラリを開きます。

### 方法 1: データ パネルを使用する

- Fusion ウィンドウの左上隅にある **[データ パネルを表示]** をクリックします。

- ライブラリを含むプロジェクトとフォルダを見つけます。

- ダブルクリックして、**ライブラリ作業スペース**でライブラリを開きます。

### 方法 2: ライブラリ マネージャを使用する

- 電子デザイン ライブラリ、回路図、または 2D PCB ドキュメントを開きます。

- [電子デザイン ライブラリ マネージャを開きます](ECD-OPEN-LIBMGR-TSK.htm)。

- **[ハブ ライブラリ]** タブで、編集するライブラリを見つけます。

- ライブラリを選択して右上の **[編集]** をクリックするか、ライブラリを右クリックして **[ライブラリを編集]** を選択します。

## 次に行える操作

ライブラリ作業スペースでライブラリを開いたら、次の操作を実行できます。

- **コンポーネントの追加:** 新しい回路図記号、フットプリント、コンポーネント、3D パッケージを作成します。

- **既存のパーツの編集:** コンポーネントのプロパティ、記号、フットプリント レイアウト、3D パッケージ ジオメトリを変更します。

- **ライブラリ情報の更新:** ライブラリの名前または説明を変更します。

- **コンポーネントの削除:** ライブラリから古いパーツや未使用のパーツを削除します。
