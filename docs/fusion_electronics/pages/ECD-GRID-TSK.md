# 図面のグリッド設定を確認/修正する

- [デザイン]ツールバーで、 **[グリッド]** ![グリッドのアイコン](../images/icon/ecd/grid-settings.png) をクリックします。

- [グリッド]ダイアログで、グリッド表示とサイズのフィーチャを確認します。

> **Note:** 注: 既定の 0.1 インチのグリッドを使用することは業界標準であり、強くお勧めします。
