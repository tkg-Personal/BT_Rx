# コンポーネントを置き換える

古いコンポーネントと新しいコンポーネントが属性セット、ピン数、またはジオメトリに互換性がない場合、システムで置換が拒否される場合があります。同じコンポーネントの複数のインスタンスを別のコンポーネントに置換するには、グループを選択し、[検査]の置換クイック アクション機能を使用します。
