# アセンブリ バリアントについて

製品バリアントは、さまざまな機能をサポートするためのさまざまなコンポーネントを持つ、一連の類似した PCB デザインを記録するための便利な方法です。元のデザインが最も完全なバージョンです。バリアントは、入力されていない(削除された)コンポーネントや異なる値を持つコンポーネントを使用して作成されます。
