# QuickRoute ガイド パス

- [PCB 設計]ツールバーで、 **[QuickRoute Guided]** ![QuickRoute Guided のアイコン](../images/icon/ecd/quick-route-guided.png) をクリックします。

- [QuickRoute Guided]ダイアログで、配線するレイヤー、およびトレースとビアの属性を確認または選択します。

- クリックして、配線するパッド間を横切る線分を描画します。選択したパッドがハイライト表示されます。選択の終了時にもう一度クリックします。

- 右クリックして、選択を終了します。

- クリックすると、1 つの開始パッドから終了パッドまでのガイド パスが描画されます。ダブルクリックしてパスを終了します。

- [Enter]を押して、パスを作成します。

> **Note:** 注: トレースはガイドパスにできるだけ近づくようにします。
