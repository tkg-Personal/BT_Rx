# 電子デザインの基本設定

電子デザインの基本設定を調整して、デザインの効率を高めます。以下の基本設定は、Fusion Electronics において既定の設定をコントロールします。基本設定に加えた変更は、Autodesk ID を使用して保存され、別のコンピュータで Fusion にサインインするときにロードされます。

## 電子デザイン

### 回路図

- **テーマ**: 作業スペースの色のテーマを選択します。明または暗から選択します。

- **カーソル**: カーソルのサイズを選択します。

- **グループの注釈**: グループの注釈の動作を選択します。

### PCB

- **テーマ**: 作業スペースの色のテーマを選択します。明または暗から選択します。

- **カーソル**: カーソルのサイズを選択します。

- **ボード形状を検出**: 閉じたボード プロファイルを 2D から 3D にプッシュする場合は、この設定をアクティブにするか、選択する必要があります。既定値: オン。

### その他

- **常にベクトル フォント**: 組み込みのベクトル フォントを常に使用します。CAM プロセッサ出力に 1:1 解像度を使用できます。既定値: オフ。

- **倍率を制限**: 回路図および 2D PCB ドキュメントの最大ズーム倍率を指定します。既定値: オン。

- **従来のコマンド ライン フォーカス**: EAGLE のコマンド ラインをエミュレートするように Fusion のコマンド ラインを設定します。

- **推定配置を表示**: スケッチ ジオメトリの作成および編集中に、ジオメトリの推定配置をオンにします。頂点とエッジには、位置合わせを示す一時的なグラフィックが表示されます。幾何拘束は適用されません。タスクの作成または編集中に推定配置を一時停止するには、[Shift]を押します。

- **マウス ホイール ズーム**: マウス ホイールのズーム比率を設定します。マウス ホイールでズームする場合は、これを調整します。値が 0 の場合、このオプションはオフになります。

- **外部テキスト エディタ**: 別の外部テキスト エディタを指定します。

- **パーツの人気度のしきい値**: フィルタが動作するしきい値を設定します。フィルタは、POPULARITY という名前の属性を検索します。属性を作成し、値を割り当てます。これは、パブリック ライブラリで使用します。

- **縦書き文字**: 新しい図面で縦書き文字を表示する方法を選択します。**[上]**は右側から読み取り、**[下]**は左側から読み取ります。

## カラー

- **テーマ**

**暗**: 暗いテーマを使用する場合のボード、グリッド要素、およびキャンバスの色を指定します。

- **明**: 明るいテーマを使用する場合のボード、グリッド要素、キャンバスの色を指定します。

- **暗**: 単一レイヤー モードの場合に暗く表示するために使用する色を選択します。

- **インジケータ**: 配線インジケータに使用する色を選択します。

- **ハイライト表示**

**暗く - 明るく**: ハイライトの強度を指定します。既定値は 50% です。

- **高コントラスト**: 選択して、必要なコントラストの色合い設定を調整します。

- **リセット**: ハイライト表示設定を既定値に戻します。

- **PCB の拘束と配線**

**未完了配線に異なる塗り潰しスタイルを使用**: 未完了配線と完了配線を区別するために、別の色を使用する場合に選択します。次に、使用する色を選択します。

- **ライブ DRC**: 2D PCB で変更が行われた後にデザイン ルール チェックを自動的に実行する場合に選択します。

- **それぞれの銅箔レイヤーの色でビアとパッドを表示**: 既定値: オフ。

## ドリル

次の基本設定を指定します。

- ドリルの番号と直径

- 回路図シンボルの直径と幅

## その他

次の基本設定を行います。

- **[ビープ音]**は、確認ビープ音のオン/オフを切り替えます。既定値: オン。

- **[接続を確認]**は、回路図にコンポーネントを配置するときに、パッケージ チェックをアクティブ化します。既定値: オン。

- **[最適化]**を使用すると、直線の曲げが自動的に削除できます。既定値: オン。

- **Ratsnest** コマンドを使用するときに、**[Ratsnest 処理銅箔ベタ]**にはポリゴンの内容が含まれています。既定値: オン。

- **[パッド名を表示]**は 2D PCB または 3D PCB ドキュメントにパッド名を表示します。既定値: オフ。

- **[パッド上に信号名を表示]**は、2D PCB または 3D PCB ドキュメントに信号名を表示します。既定値: オン。

- **[トレース上に信号名を表示]**トレースは、2D PCB または 3D PCB ドキュメントに信号名を表示します。既定値: オン。

- **[ビア長を表示]**は、2D PCB ドキュメントに開始レイヤー名と終了レイヤー名を表示します。既定値: オフ。

- **[ドリルを表示]**は、パッドとビアにドリル穴を表示します。既定値: オン。

- **[ネットとバスを自動終了]**のアクティブなネット配置は、ピンまたはバスに配置されたときに終了します。既定値: オン。

- **[接合を自動設定]**は、別のネット上でネットを終端するときに接合を自動的に追加します。既定値: オン。

- **[配線幅とドリルを自動設定]**は、デザイン ルールまたはトレースのネット クラスで指定されたワイヤ幅とビア長を使用します。この値は、信号ワイヤをクリックすると自動的に設定されます。既定値: オフ。オフの場合、最後に指定した値が使用されます。

- **[グループ]コマンド**は 2D PCB ドキュメントで何も選択されていない場合は、既定のマウス アクションとしてグループを選択できます。既定値: オフ。

- **[最小表示文字サイズ]**は、最小文字サイズと同じか大きい文字を表示します。既定値: 3 ピクセル。

- **[最小表示グリッド サイズ]**は、画面からの近接度が最小グリッド間隔以上のグリッド ラインとドットを表示します。既定値: 5 ピクセル。

- **[キャッチ係数]**は、アクティブな表示ウィンドウのパーセンテージでキャッチの半径を定義します。半径内のオブジェクトに到達できます。この値を 0 に設定して、設定をオフにすると、遠くに配置されたオブジェクトに到達します。既定値: アクティブな表示ウィンドウの 5%。

- **[選択係数]**は、アクティブな作図ウィンドウのパーセンテージで、選択半径を定義します。この半径内のオブジェクトが選択されます。既定値: 2%。

- **[スナップ長]**は、スナップ半径を定義します。カーソルがパッドまたは SMD から半径距離内にあるときにクリックすると、トレースがオブジェクトに接続されます。

- **[外部参照ラベルの形式]**は、外部参照を表示するときに使用するラベル文字の形式を定義します。

- **[外部参照パーツの形式]**は、外部参照を表示するときに使用するコンポーネント名の文字の形式を定義します。

## グリッド

回路図ドキュメントおよびボード ドキュメントにグリッドを表示するかどうかと、グリッドの表示方法に関する基本設定。

- **グリッド回路図**

**グリッドを表示**: 新しい回路図ドキュメントを開始するときに、グリッドを表示します。グリッド単位を選択します。

- **サイズ**: グリッド点間またはグリッド線の中心間の距離を定義します。

- **複数**:

- **代替**: 代替グリッドを定義します。[Alt]キーを押したままにすると、このグリッドがアクティブになります。キーを押している間、このグリッドはアクティブになります。代替グリッドを使用すると、コンポーネントの位置を制限付きで調整できます。

- **グリッド 2D PCB**

**グリッドを表示**: 新しい回路図ドキュメントを開始するときに、グリッドを表示します。グリッド単位を選択します。

- **サイズ**: グリッド点間またはグリッド線の中心間の距離を定義します。

- **複数**:

- **代替**: 代替グリッドを定義します。[Alt]キーを押したままにすると、このグリッドがアクティブになります。キーを押している間、このグリッドはアクティブになります。代替グリッドを使用すると、コンポーネントの位置を制限付きで調整できます。

- **グリッドをリセット**

**回路図ドキュメントを開くときにグリッドを既定にリセットします**

- **ボード ドキュメントを開くときにグリッドを既定にリセットします**

## 既定

トレース幅、ビア直径とドリル サイズ、PTH パッドの直径とドリル サイズ、SMD パッドのサイズと丸み、ポリゴンの分離と間隔、マイタ半径、文字サイズ、寸法線の幅、寸法補助線の幅、寸法補助線の長さ、寸法補助線のオフセットに使用する既定値を設定します。

## フォルダ

ライブラリ、デザイン ルール、ULP、スクリプト、CAM ジョブ、シミュレーション、SPICEモ デルで使用するパスを設定します。既定のパスはインストール時に定義されます。
