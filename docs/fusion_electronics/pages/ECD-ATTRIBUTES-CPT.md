# アトリビュート

ボード内のコンポーネントに名前と値以外の情報を割り当てる場合は、 **[属性]** ツールを使用して割り当てることができます。

属性は、さまざまなドキュメントで定義することができます。属性を定義できるドキュメントは、次の通りです。

- ライブラリ: 最も移植性が高く、一貫性のあるオプションです。

- 回路図: これらは 2D PCB に転送され、一貫性が保たれます。

- 2D PCB: 回路図に戻されない別個の属性セットです。2D PCB ドキュメントで直接作成された属性は、2D PCB にのみ存在します。

グローバル属性は、ボード全体に対して有効です。これらは、2D PCB と回路図ドキュメントで個別に定義できます。
