# 空白

ユーザ言語プログラムを実行する前に、ファイルから読み込む必要があります。この読み込み処理中に、ファイルの内容が解析され、トークンとホワイトスペースになります。スペース(空白)、タブ、改行文字、およびコメントは、ホワイトスペースとみなされ、破棄されます。

ホワイトスペースを表す ASCII 文字が破棄されない場所は、string s = "Hello World"; のようにリテラル文字列の中のみです

「o」と「W」の間の空白文字は、文字列の一部のままです。

行の最後の改行文字の前に円記号(¥)が付いている場合、円記号と改行文字の両方が破棄され、2 行が 1 行として扱われます。

"Hello \
World"
これは、"Hello World" と解析されます。
