# チュートリアル: デザインを同期する

通常、回路図、基板デザイン、およびライブラリは定期的に更新され、同期されたままになります。しかし、ドキュメントが電子デザインから分離され、個別に修正される場合に、この同期が解除されることがあります。このような場合は、シンクロナイザを使用して同期を復元することができます。   

このチュートリアルの内容

- 同じ電子デザインの回路図と基板の間の同期を復元する

- 2 つのバリアント デザインで使用されるコンポーネントを比較する
