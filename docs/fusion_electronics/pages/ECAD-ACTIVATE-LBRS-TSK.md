# 配置パネルでライブラリを追加または削除する

**電子デザイン ライブラリ マネージャ**を使用して、コンポーネント ライブラリを[配置パネル]に追加し、Fusion 電子デザインでそれらのコンポーネントにアクセスして配置できるようにします。非表示にする必要があるライブラリを非アクティブにすることもできます。

## 前提条件

- [電子デザイン ライブラリ マネージャを開く](ECD-OPEN-LIBMGR-TSK.htm)

## 手順

- 電子デザイン ライブラリ マネージャで、 **[ハブ ライブラリ]** 、 **[パブリック ライブラリ]** 、または **[プライベート ライブラリ]** タブにナビゲートし、追加したいライブラリを探します。

- **[アクティブ]** 列で、[配置パネル]で使用するライブラリのトグルをクリックします。

- チェック マーク付きの円は、ライブラリが正常にダウンロードされたことを示します。

- [配置パネル]に追加する他のライブラリについても同様に繰り返します。

## 予想される結果

アクティブ化されたライブラリは、[配置パネル]に表示され、それらのコンポーネントを電子デザインに配置できるようになります。

## 次のステップ

- アクティブ化されたライブラリから回路図または PCB デザインに[コンポーネントを配置](ECAD-PLACE-CMPNT-SCMTC-TSK.htm)します。

- ライブラリ マネージャを使用して、アクティブなライブラリ全体でコンポーネントを検索およびフィルタします。
