# コンポーネントを別のコンポーネントに置換する

- [回路図デザイン]ツールバーで、**[置換]** ![置換のアイコン](../images/icon/ecd/replace.png) をクリックします。

- ライブラリで置換項目を選択し、[OK]をクリックします。

- 図面に戻り、置き換える項目をクリックします。

置換が承認された場合は、実行されます。新しいコンポーネントがカーソル上に残り、別の置換操作を実行できます。

- 置換が承認されない場合は、問題について説明するメッセージが表示されます。
