# シミュレーションのアナログ電源を設定する

電源は、SPICE 互換の電圧源または電流源である必要があります。

- [回路図デザイン]ツールバーで、 **[シミュレート] > [アナログ ソース設定]** ![アナログ ソースのアイコン](../images/icon/ecd/analog-source-setup.png) をクリックします。

- [ソースの設定]ダイアログで、[DC 値]または[AC 値]を指定し、オプションの[非定常関数]を指定します。
