# 電子図面

電子図面は製造データセットの一部であり、技術者やエンジニアがボードを診断し、適切な構造を検証するために使用されます。量産の場合、レイヤー数に関係なく、すべてのデザインに対して生成されます。

![電子デザインの図面作成](../images/overview/dwg-from-ecad.png)

## レイヤー スタック

**レイヤー スタック テーブル**を**レイヤー スタックの図**と並べて追加すると、材料、厚さ、タイプ、および特別な属性など、各レイヤーの詳細情報を確認できます。これらを組み合わせることで、PCB 設計の徹底的な検証が可能になり、生産開始前に必要な基準と性能要件を満たしていることを確認できます。また、PCB の物理的な構造と仕様を全員が理解できるようになります。
