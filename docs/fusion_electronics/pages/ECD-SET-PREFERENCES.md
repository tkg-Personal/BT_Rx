# 電子設計の基本設定を行う

- アプリケーション バーの右上隅で、**[マイ プロファイル]**をクリックします。

- **[基本設定]**を選択します。

- 次の**[一般]**設定を確認します。

**[既定のドキュメントの基本設定]**は**[電子デザイン]**に設定されています

- **[ドメインの対象]**は**[電子デザイン]**に設定されています。Fusion が起動されると、新しいドキュメント タイプは[電子デザイン]ドキュメントになります。

![一般基本設定](../images/how-to/ecd-pref-gen-docpref.png)

- **[電子デザイン]**基本設定リストを展開します。リストの各項目をクリックして、基本設定を管理します。
