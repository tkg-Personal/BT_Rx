# QuickRoute エアワイヤ

- [PCB 設計]ツールバーで、 **[QuickRoute Airwire]** ![QuickRoute Airwire のアイコン](../images/icon/ecd/quick-route-airwire.png) をクリックします。

- [QuickRoute Airwire]ダイアログで、配線するレイヤー、およびトレースとビアの属性を確認または選択します。

- 回路基板で、配線するエアワイヤをクリックします。デザイン ルール違反を起こさずに実行することができない場合を除き、トレースが描画されます。

- さらに多くのエアワイヤを配線するには、1 つずつクリックを続けます。

- 終了するには、[QuickRoute Airwire]ダイアログで[OK]をクリックします。
