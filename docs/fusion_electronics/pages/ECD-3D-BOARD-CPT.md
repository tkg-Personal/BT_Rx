# 機械アセンブリに PCB を統合する

Fusion で電子機器を操作する場合、プリント基板(PCB)を機械アセンブリと統合することができます。これにより、完成した製品とのコンテキストにおいて、電子機器の適合性と機能をデザインし、検証することができます。

- 機械アセンブリのコンテキストで PCB を作成する

- 3D 回路基板を機械的な筐体にフィットさせる

- 機械的ジオメトリを参照しながら、穴、コンポーネントの位置、およびボードのアウトラインを修正する

- 変更を 2D PCB 基板と同期する
