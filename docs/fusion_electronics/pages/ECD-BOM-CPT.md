# 電子デザイン部品表

この部品表は、デザインの完全性と存在するすべてのバリアントを確認するために使用されます。部品表(BOM)は、コンピュータ支援製造(CAM)ファイルに含まれています。

重要な部品表の 5 つのコンポーネントは次のとおりです。

- 数量: 部品表に記載されているコンポーネントの数が正しいことを確認し、正しい数量が購入されるようにします。

- 値: 各コンポーネントは、動作およびパフォーマンスの値の必要な範囲内にある必要があります。

- 参照指定子: 各コンポーネントは、PCB の回路内の位置に従って識別される必要があります。

- フットプリント: コンポーネントには複数のフットプリントが用意されており、注文するコンポーネントのフットプリントを指定することが重要です。

- 製造元のパーツ番号: ディストリビュータまたは製造元に注文する際に使用する、特定のコンポーネントの一意の識別子です。
