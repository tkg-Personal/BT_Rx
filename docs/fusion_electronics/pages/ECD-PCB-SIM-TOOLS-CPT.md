# PCB シミュレーション ツール

**[電子設計]**ワークスペースの**[シミュレーション] > [信号を解析]**パネルの次のコマンドを使用すると、信号の整合性の評価、インピーダンスの計算、伝播の遅延、および Fusion のその他のパフォーマンスの測定値を実行して、デザインを定義する際の外部テストの必要性を減らすことができます。

Fusion Signal Integrity Extension

 この機能は拡張機能の一部です。拡張機能は Fusion の追加機能にアクセスするための柔軟な方法です。[詳細情報](https://help.autodesk.com/view/fusion360/JPN/?contextId=EXT-OVERVIEW)。

このツールを使用して、2D PCB の配線段階におけるプリント基盤の信号の整合性を解析します。
