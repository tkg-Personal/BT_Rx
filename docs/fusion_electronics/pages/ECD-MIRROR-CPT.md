# オブジェクトをミラー化する

**[ミラー]** は、選択した記号、オブジェクト、文字、およびグループを垂直軸を中心に回転し、元の配置のミラーとして再配置します。

## 回路図ドキュメントでミラーを使用する

記号、オブジェクト、および文字は、同じレイヤーに戻されます。回路図で **[ミラー]** を使用すると、ネットの交差を回避しながらデザインを簡略化できます。

![回路図ミラー化の結果](../images/concepts/ecd-mirror-sch.png)

図 1: ミラー化されたオブジェクトの表示

## PCB ドキュメントでミラーを使用する

個々のコンポーネントはミラー化できませんが、代わりに **[反転]** を使用してボードの反対側に反転することができます。コンポーネントを含むグループがミラー化されると、そのグループはボードの反対側に反転されます。**[ミラー]** は、オーディオ アンプ回路などの対称的な PCB を作成するときに便利です。

![ボード ミラー化の結果](../images/concepts/ecd-mirror-brd.png)

図 2: グループ内でミラー化されたコンポーネントが、ボードの反対側に自動的に反転して表示される様子
