# 新しい回路図ドキュメントを作成する

既定のグリッド サイズで回路図を作成します。後で回路図の接続が回路基板に正しく転送されるように、既定のグリッド サイズを変更しないことが重要です。

- ウィンドウ上部の[アプリケーション]バーで、 **[ファイル]** ![新規ファイルのアイコン](../images/icon/common/file.png) > **[新規電子デザイン]** をクリックします。

- [電子デザイン]ツールバーの **[共通]** タブで、 **[作成] > [新しい回路図]** ![新しい回路図のアイコン](../images/icon/ecd/new-schematic.png) をクリックして空のキャンバスを開きます。

- ファイルを保存します。

- **[表示]** パネルで、 **[グリッド設定]** ![グリッドのアイコン](../images/icon/ecd/grid-settings.png) をクリックして[グリッド]ダイアログを開きます。

- **[グリッド]** ダイアログで、 **[サイズ]** が 0.1 インチに設定されていることを確認します。

- **[OK]** をクリックして設定を保存し、ダイアログを閉じます。
