# チュートリアル: 手動配線

2D PCB の作成中に行う主なタスクの 1 つは、ワイヤのルートを作成することです。ルートの作成に使用するツールは多数ありますが、手動配線は最も便利で強力な技法の 1 つです。

このチュートリアルでは、ツールで使用されるショートカット キーなど、**[手動配線]**ツールのコントロールの使用方法を練習します。
