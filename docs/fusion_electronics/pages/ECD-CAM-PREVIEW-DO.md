# CAM プレビューを実行する

CAM プレビューを実行する前に、デザイン ルール チェック(DRC)を実行することをお勧めします。

- キャンバスで回路基板を開き、[製造]ツールバーで**[CAM プレビュー]** ![CAM プレビューのアイコン](../images/icon/ecd/cam-preview.png) をクリックします。

- [製造プレビュー]ダイアログで、ボードのプロパティを確認します。

- ボードの詳細がすべて正しい場合は、 **[CAM エクスポート]** ![CAM エクスポートのアイコン](../images/icon/ecd/cam-export.png) を実行し、ZIP アーカイブ内の製造ファイルを指定のフォルダの場所にエクスポートします。
