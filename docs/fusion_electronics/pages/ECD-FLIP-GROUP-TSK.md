# PCB レイアウト内のオブジェクト グループを反転する

**[反転]** は、選択したグリッド ポイントを軸として、フットプリントなどのオブジェクトを水平方向に回転させ、両面 PCB の反対側に配置します。

- **[電子デザイン]** 作業スペースで、開いている電子デザインの 2D PCB ドキュメントのキャンバスで、反転するオブジェクトの近くを左クリックし、そのオブジェクトを囲むようにボックスをドラッグしてグループを作成します。

- **[デザイン]** ツールバーで、 **[修正] > [反転]** ![反転のアイコン](../images/icon/ecd/flip.png) をクリックして **[反転]** パネルを開きます。

- 必要に応じて、 **[反転]** パネルで **[ルート モード]** 設定を調整します。

- キーボードで **[Ctrl]** を押しながらキャンバスを右クリックして、コンテキスト メニューを開きます。

> **Note:** 注: キャンバス内のどこで右クリックするかが重要です。グループは、次のグリッド ポイントを通ってクリックした場所まで水平軸を横切って反転します。

- コンテキスト メニューで、 **[反転: グループ]** をクリックしてグループを反転します。

- 必要に応じて、 **[元に戻す]** をクリックして変更を元に戻すか、 **[Esc]** をクリックして **[反転]** パネルを終了します。

- **[完了]** をクリックして変更を承認し、ダイアログを閉じます。
