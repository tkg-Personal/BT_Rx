# シート間を接続する信号に外部参照ラベルを追加する

シート間を接続するネットに外部参照ラベルを適用して、ネット上の信号が別のシート上のネットにリンクする場所を示します。

![外部参照ラベル](../images/how-to/xref-labels.png)

- 回路図ドキュメントで、 **[デザイン]** タブの **[接続] > [ラベル]** ![ラベルのアイコン](../images/icon/ecd/label.png) をクリックして、 **[ラベル]** コマンド パネルを開きます。

- **[ラベル]** コマンド パネルで、 **[外部参照をオン]** をクリックします。

[外部参照をオン]アイコンが青色でハイライト表示されます。

- キャンバスで、ラベルを付けるネットをクリックします。

信号が存在する場合は、その名前を示すラベルが表示されます。

- **[ラベル]** コマンド パネルで、必要に応じてラベルを回転させます。

- キャンバスでクリックしてラベルを配置します。

- ラベル文字を変更する場合は、**([接続] > [名前])** ![名前のアイコン](../images/icon/ecd/name.png) をクリックし、ラベルの原点(+)をクリックします。

- **[名前]** ダイアログで、新しい名前を入力します。

- **[OK]** をクリックして新しい名前を保存し、ダイアログを閉じます。

- 接続信号が配置されているもう一方のシートで、上記の手順を繰り返して同じラベルを配置します。

2 番目のラベルを保存すると、各ラベルに文字が自動的に追加され、信号のもう一方の端の位置が表示されます。
