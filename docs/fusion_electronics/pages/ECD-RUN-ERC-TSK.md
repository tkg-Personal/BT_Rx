# 電気的ルール チェックを実行する

- [回路図検証]ツールバーで、**[ERC]** ![ERC のアイコン](../images/icon/ecd/erc.png) をクリックします。チェックは回路図内で即座に実行され、結果が[ERC エラー]パネルに報告されます。

- パネルで、エラーと警告を確認します。いずれかをクリックすると、デザイン内でハイライト表示されます。

- 次のオプションがあります。

エラーと警告を修正し、![[ERC 更新]アイコン](../images/icon/ecd/ecd-erc-refresh.png) をクリックします。

- 個々のエラーまたは警告を選択し、**[承認]**ボタンを使用して承認します。

- [エラーまたは警告]見出しを選択し、**[すべて承認]**をクリックして承認します。

- **[承認済み]**見出しを選択し、**[すべて不認可]**をクリックして不認可にします。
