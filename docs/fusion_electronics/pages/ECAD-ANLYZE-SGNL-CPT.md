# 電子デザインの信号を解析パネル

**[信号を解析]** パネルは、**Signal Integrity Extension** で使用することができます。このパネルでは、次の操作を実行できます。

- 信号の整合性の評価

- インピーダンスの計算

- 伝播遅延の計算

また、設計の完了後に外部テストの必要性を減らすのに役立つその他のパフォーマンス指標の評価を実行します。

**[信号の解析]** パネルは、次の作業時に使用できます。

- 2D PCB
