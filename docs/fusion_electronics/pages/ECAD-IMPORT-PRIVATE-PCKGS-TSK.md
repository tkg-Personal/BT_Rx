# Library.io 3D パッケージをハブに読み込む

プライベート ライブラリを読み込んだ後、このライブラリにまだ、Fusion ハブに読み込む必要があるプライベート パッケージが含まれている場合があります。

## 前提条件

- Library.io に保存されているパッケージを含むライブラリが[開いている](ECD-EDIT-CSTM-LIB-TSK.htm)必要があります。

## 手順

- 左側のパネルで **[コンテンツ マネージャ]** にナビゲートし、 **[パッケージ]** タブをクリックします。

- Library.io アイコン ![Libraryio パッケージのアイコン](../images/icon/ecd/libio-package.png) が表示されている、Library.io に保存されているパッケージを見つけます。

- パッケージをダブルクリックして、現在のハブに読み込みます。現在のハブへの読み込みを確認するダイアログが表示されます。

- **[確認]** をクリックして、読み込みを続行します。

- パッケージを保存し、現在のハブでパッケージを保存する場所を選択します。

## もう一つの方法:

ハブ メンバーに Library.io データへの編集アクセス権を付与しても構わない場合は、メンバーを Library.io Web サイトのフォルダ メンバーに[招待](ECAD-LIBIO-PERM-TSK.htm)することもできます。Library.io に追加されたチーム メンバーには、自動的に 3D パッケージへのアクセス権が付与されるため、Fusion ハブに読み込む必要はありません。
