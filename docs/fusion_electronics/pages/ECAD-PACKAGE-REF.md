# パッケージ

ライブラリに保存されている、コンポーネントの 3D 物理表現。関連用語として「コンポーネント」、「フットプリント」、「記号」を参照してください。

- **パッケージ**(必須)は、記号と、必要に応じて 3D モデルで構成されます。

- **3D パッケージ**: コンポーネントの 3D モデルです。

バリアントごとに、異なる 3Dモデル を持つことができます。

- 3D モデルとフットプリントは一致している必要があります。

- ライブラリに含めるコンポーネントには、3D モデルは必須ではありません。このような場合、デザインを 3D PCB にプッシュする際に、プレースホルダ形状が使用されます。
