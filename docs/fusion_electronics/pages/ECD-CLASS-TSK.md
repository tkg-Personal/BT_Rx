# ネット クラスを確認する

- [回路図検証]ツールバーで、**[クラス]** ![クラスのアイコン](../images/icon/ecd/class.png) をクリックします。**[ネット クラス]**ダイアログが開きます。

- **[クラス]**タブでは、複数のオプションを使用できます。

このクラスのメンバーを確認するには、差動ペアをクリックします。

- 個々のネットを**[メンバーなし]**リストと**[メンバー]**リストの間で移動するには、[>]ボタンと[<]ボタンを使用します。

- すべてのネットをリスト間で移動するには、[>>]ボタンと[<<]ボタンを使用します。

- 新しいネット クラスを追加するには**[追加]**をクリックし、ネット クラスを削除するには**[削除]**をクリックします。

- **[ルール]**タブで、既定のルールが回路基板のデザイン ルールに適合していることを確認します。通常は、[幅]と[クリアランス]の値は 0 です。
