# 筐体内の 3D PCB を編集する

## 電子デザインから

- 3D PCB に変換する 2D 回路基板デザインを開きます。

- **[切り替え] > [3D PCB]**をクリックします。

- 必要に応じて、 **[3D PCB]** ツールバーで、 **[修正] > [コンポーネントを移動]** をクリックして、ボード上のコンポーネントの位置を調整します。

- 必要に応じて、 **[3D PCB]** ツールバーで、**[修正] > [コンポーネントを反転]** をクリックして、コンポーネントをボードの反対側に反転します。

- **[PCB] > [2D PCB]**をクリックして、両方のドキュメントを同期したままにします。

> **Note:**  注: 3D 基板が個別に作成されている場合、適切なデザインがあれば、3D デザインを 2D PCB にリンクすることができます。

## 機械設計から

- ブラウザで基板図面名の横にある鉛筆のアイコンをクリックして、3D 基板の **[インプレイス編集]** をアクティブにします。

![インプレイス編集をアクティブ化](../images/how-to/eip/activate-eip.png)

- 図面ウィンドウの上部にある **[インプレイス編集]** メニューで、 **[自動調整]** を選択します。

![自動調整](../images/how-to/eip/associative.png)

この設定は、 **[インプレイス編集]** を使用する場合に、3D と 2D の回路基板間の自動更新をサポートするために必要です。

- **[3D PCB]** パネルの **[反転]** ツールを使用して、コンポーネントをボードの反対側に移動したり、 **[移動]** ツールを使用してコンポーネントと筐体内の開口部との位置合わせを改善することができます。

- 図面ウィンドウの上部にある **[インプレイス編集]** メニューの緑のチェック マークをクリックして、 **[インプレイス編集]** モードを終了します。ボードと筐体の **[デザイン]** 作業スペースを保存します。

![インプレイス編集を終了](../images/how-to/eip/end-eip.png)

- **3D PCB** 図面タブをクリックして**データ パネル**で再び開き、ボードおよび筐体の **[デザイン]** 作業スペースから **[インプレイス編集]** の更新を取得します。
