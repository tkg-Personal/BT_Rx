# ライブラリ オブジェクトに SVG ファイルを挿入する

**[SVG を挿入]** を使用して、スケーラブルなベクトル イメージを記号またはフットプリントに配置します。

- **[電子デザイン]** 作業スペースの電子デザイン ライブラリ ドキュメントで、**コンテンツ マネージャ**の記号またはフットプリントをダブルクリックしてキャンバスで開きます。

- **[記号]** ツールバー(記号の場合)または **[フットプリント]** ツールバー(フットプリントの場合)で、 **[配置] > [SVG を挿入]** ![SVG を挿入のアイコン](../images/icon/ecd/insert-svg.png) をクリックします。

- **[ファイルを開く]** ダイアログで、イメージに移動して選択します。

- **[SVG を挿入]** パネルで、設定を確認します。

選択したファイル: インポートするファイル

- ターゲット画層: 画像を配置する画層

- ストロークのみ: インポートした画像の曲線を塗り潰すかどうか

- 尺度係数: 元の画像に対する画像のサイズ

- 画像を配置するには、次の操作を実行します。

**[SVG を挿入]** パネルの **[配置原点]** フィールドに座標を入力して、原点を基準にした特定の位置に配置します。

- または、キャンバス内をクリックして、マウスの位置に画像を配置します。

- **[SVG を挿入]** パネルで **[完了]** をクリックして画像を配置し、パネルを閉じます。
