# 最適化

- ネットを別のネットに結合します。

ネット セグメントを移動して別のセグメントに結合した場合、名前を変更して別のネットと結合するかどうかを尋ねられます。

![[結合]ダイアログ](../images/how-to/optimize-merge.png)

- ネットを延長して別のネットに結合した場合、ネット セグメントを接続するかどうか、および接続する場合に使用するネット名を尋ねられます。

![[接続]ダイアログ](../images/how-to/optimize-connect.png)

- ダイアログの質問に答えて、最適化を完了します。
