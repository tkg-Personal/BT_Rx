# オートルータを使用する

オートルータを使用して、基板上にルート信号を自動的に設定します。1 つの信号、信号のグループ、またはすべての信号を含めることができます。オートルータは、トラック幅や最小クリアランスが異なるさまざまなネット クラスをサポートします。

オートルータを開始する前に、適切なデザイン ルールが基板に設定されていることを確認します。配線する前に、コンポーネントの配置と配置グリッドを確認します。オートルータがすべてのパッドと SMD を適切に接続できるようにするルーティング グリッドを考えてみます。

理論上、オートルータは、時間に制限がなければ基板を完全に配線できます。実際には、必ずしもこれが可能とは限りません。重要な信号を先に手動で配線し、残りをオートルータ用に任せることを考えてみます。

オートルータでは、リップアップ/再試行アルゴリズムを使用します。トラックを配線できないとわかると、すぐにそれまでに配線されたトラックを除去(リップアップ)し、もう一度試します(再試行)。除去できるトラックの数はリップアップ深さと呼ばれ、これが処理速度と結果に影響します。既に手動で配線されているトラックは変更されません。

オートルータのメイン ダイアログでは、TopRouter バリアントを選択できます。トポロジ アプローチによるグリッドレス アルゴリズムが使用されます。このアルゴリズムでは、最初に信号のコースを計算し、次に最適化実行を使用してデザイン ルールを満たします。通常、TopRouter では必要なビアが従来のオートルータよりも大幅に少なくなります。1 つのプロジェクトに対して両方の方法を選択し、最終的にいずれかの配線結果を選択することができます。

オートルータで問題なく作業を進めるには、コンポーネントを配置し、パラメータを慎重に設定して配線方法をコントロールします。

**基本的な機能**:

- 任意の配線グリッド(最小 0.02 mm)または配置グリッドで使用できます

- グリッドレス配線アルゴリズムを使用した TopRouter (事前にオートルータを使用することが可能)

- ファンアウト配線用の BGA ルータ

- 信号画層でのルーティング グリッドおよび優先方向の自動選択(任意)

- 複数の配線ジョブを同時に処理するためのマルチコア プロセッサのサポート

- SMD を両側に配線

- 作図領域全体を配線領域にすることが可能(十分なメモリが使用可能な場合)

- 制御パラメータを使用した方法の選択

- さまざまなトラック幅と最小クリアランスを持つさまざまな信号クラスを同時配線

- デザイン ルール チェックとオートルータに共通のデータ セット(デザイン ルール)

- 多層機能: ペアだけではなく、最大 16 個の画層を同時配線可能

- ブラインド ビアと埋め込みビアのサポート

- 優先トラック方向を画層ごとに個別に設定可能: 水平および垂直、45/135 度の実角度(内側の画層に対して重要)

- 100% 配線方法のためのリップアップと再試行

- ビアを削減してトラック パスを円滑化する最適化パス

- 事前配線トラックは変更されません
