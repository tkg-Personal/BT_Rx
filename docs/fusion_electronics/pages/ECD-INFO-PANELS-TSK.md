# 固定パネルをサイズ変更する

[電子デザイン]作業スペースの固定パネルは、必要な情報を見やすくするためにサイズ変更することができます。

- 垂直または水平のパネルの端にカーソルを合わせ、カーソルがサイズ変更用のカーソルに変わるまで待ちます。

- クリック アンド ドラッグしてパネルのサイズを変更します。
