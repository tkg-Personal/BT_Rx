# ネットを描画する

ネット描画セッションを開始すると、[ネット]ダイアログ ボックスが開きます。ここでは、セッション中はいつでも、ネット クラス、曲げスタイル、線種、曲線の半径、またはマイタの形状の値を調整できます。変更すると、その後の描画操作に影響します。
