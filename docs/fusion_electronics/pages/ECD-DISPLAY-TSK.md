# [レイヤーを表示]について

**[画層を表示]** パネルはデザイン ペインの右側にあり、次の操作を実行できます。

**[画層を表示]** パネルは、次の作業中に使用することができます。

- 回路図

- 2D PCB

- ライブラリ

- [デザイン]ツールバーで、 **[レイヤーを表示]** をクリックするか、サイドバーをクリックして **[レイヤーを表示]** パネルを展開します。

- **[レイヤーを表示]**パネルでは、次の項目を管理できます。

レイヤーを選択する

- [レイヤー セット マネージャ]を開き、新しいセットを定義したり、既存のセットを修正する

- レイヤーを表示/非表示にする

- レイヤーをロック/ロック解除する

- レイヤーをアクティブなレイヤーとして設定する

- レイヤーを追加する

- 選択したレイヤーを削除する

- レイヤーの外観(塗り潰し、パターン、ハイライト色)を変更する

- レイヤー マテリアルのプロパティを変更する

- レイヤー プロパティを既定値にリセットする
