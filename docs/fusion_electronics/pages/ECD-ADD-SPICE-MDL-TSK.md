# SPICE シミュレーション モデルを追加する

この操作を行う前に、必要な SPICE モデルがコンピュータ上のアクセス可能な場所にあり、使用できることを確認します。

## 回路図回路デザインの場合

- 回路デザイン全体、またはその一部を選択し、コンポーネント単位で確認できます。

- [回路図デザイン]ツールバーで、**[シミュレート] > [モデルを追加]** ![モデルを追加のアイコン](../images/icon/ecd/add-sim-model.png) をクリックします。

- [モデルを追加]ダイアログで、コンポーネントにマッピングされている属性と値を確認し、必要に応じて変更します。適切にマッピングされているすべてのコンポーネントには、[完了]列に緑のチェック マークが表示されます。

- 終了したら、**[完了]**ボタンをクリックします。

## 単一ライブラリ コンポーネントの場合

- [ライブラリ デバイス]ツールバーで、**[モデルを追加]** ![モデルを追加のアイコン](../images/icon/ecd/add-sim-model.png) をクリックします。

- [モデルを追加]ダイアログで、必要に応じて適切な SPICE タイプを選択します。

- [MAP]をクリックし、[モデルをロード]ボタンをクリックします。

- モデル ファイル(<name>.mdl)の場所を参照し、[開く]をクリックします。

- コンポーネント ピンを SPICE モデルの入力にマッピングします。
