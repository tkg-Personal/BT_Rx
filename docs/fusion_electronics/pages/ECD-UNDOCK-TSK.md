# デザイン ウィンドウのドッキング解除とドッキング

## タブ付きウィンドウのドッキングを解除する

- ドッキング解除するタブをクリックし、ボタンを押したまま、ドキュメントの縮小表示に変わるまでタブを下にドラッグします。

- マウス ボタンを放すと、シートが別の新しいウィンドウで開き、別のモニタなど、デスクトップの別の場所に移動できます。

## ドッキング解除されたウィンドウを再ドッキングする

- 再ドッキングするタブで、画面の右上にある湾曲した矢印 ![ドッキングのアイコン](../images/how-to/dock-icon.png) をクリックします。

ウィンドウがタブ付きウィンドウとして再ドッキングされます。

![タブをドッキング解除](../images/how-to/undock-dock2.gif)
