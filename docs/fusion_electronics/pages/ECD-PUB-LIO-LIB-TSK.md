# ハブ ライブラリを Library.io にパブリッシュする

既存のハブ ライブラリを Library.io にパブリッシュして、ハブ間でライブラリの同期を維持することや、より幅広い対象者と共有することができます。

## 前提条件

作業を開始する前に、以下のことを確認します。

- **既存のハブ ライブラリがあること:** 既存のライブラリを開くか、新しいライブラリを作成します。

- **ライブラリが保存され、アップロードされていること:** 初回保存後、アップロードに時間がかかることがあります。

- **ライブラリがまだ Library.io にリンクされていないこと:** リンクされているライブラリには、[コンテンツ マネージャ](ecad-contnt-mgr-panel.htm)でリンク アイコン ![リンクのアイコン](../images/icon/ecd/lbr-link.png) が表示されます。これを新しいライブラリとしてパブリッシュするには、最初に[リンク解除](ECD-UNLINK-LIB-IO-TSK.htm)します。

## 手順

- ツールバーの **[管理]** セクションで、 **[Library.io]** セクションに移動し、 **[Library.io で新しい管理ライブラリを作成]** ![ライブラリのプッシュ アイコン](../images/icon/ecd/lbr-push.png) をクリックします。

- Library.io でライブラリを保存する**管理フォルダ**を選択します。

管理フォルダは、Library.io の権限をコントロールします。

- ユーザによってアクセス権のあるフォルダが異なる場合があります。

- Library.io Web サイトで、どのユーザがどのフォルダにアクセスできるかを確認します。

- **[作成]** をクリックします。

## 重要な注意事項

- **アップロード時間:** ライブラリの Library.io へのアップロードには時間がかかることがあります。

- **サイズ制限:** ハブ ライブラリが Library.io ストレージに対して大きすぎる場合は、エラー メッセージが表示されます。
