# 選択配置ファイルを生成する

- キャンバスで回路基板を開いた状態で、[製造]ツールバーで、**[PickAndPlace]** ![選択配置のアイコン](../images/icon/ecd/pick-place.png) をクリックします。

- 開いたダイアログで、提案された MNT ファイルを保存するためのフォルダが適切であることを確認します。そうでない場合は、新しい場所を参照します。

- **[保存]**をクリックします。
