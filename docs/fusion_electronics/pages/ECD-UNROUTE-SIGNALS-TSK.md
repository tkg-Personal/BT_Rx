# 信号を配線解除する

銅線信号を未配線の信号に変換し、再配線の準備を整えます。

- [PCB 設計]ツールバーで、 **[配線解除]** ![リップアップのアイコン](../images/icon/ecd/ripup.png) をクリックします。

- [配線解除]ダイアログで、実行する配線解除操作のタイプを選択します。

**[オブジェクト]**: 既定では、配線されたセグメントを未配線の信号に変換し、ビアを削除し、ポリゴンがアウトラインになります。

- **[接続]**: 選択したセグメントに接続されているすべての銅線を、他のレイヤー上の銅線を含め、未配線の信号に変換します。

- **[同じレイヤー上の接続]**: 選択したセグメントを未配線の信号に変換します。

- **[信号]**: 選択したセグメントと同じ名前を持つすべてのセグメントを未配線の信号に変換します。これには、複数の分岐トレースが含まれる場合があります。

- **[コンポーネント間]**: 2 つのコンポーネント間の共有セグメントを未配線の信号に変換します。

- **[始点/終点]**: 接続された 2 つの任意の種類の端子間で共有されているセグメントを未配線の信号に変換します。

- 操作を実行するトレース、ビア、またはデバイスをクリックします。
