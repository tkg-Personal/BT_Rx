# ポリゴンベタのカットアウトを作成する

- [PCB 設計]ツールバーで、 **[ポリゴン] > [カットアウト]** ![ポリゴンのカットアウトのアイコン](../images/icon/ecd/polygon-cutout.png) をクリックします。

- 注入されたポリゴンの内側をクリックして、切り取るポリゴン形状の描画を開始します。

- ダブルクリックすると、銅箔ベタが閉じ、注入された銅箔ベタから除去されます。
