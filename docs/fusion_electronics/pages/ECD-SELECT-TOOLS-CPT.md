# Fusion Electronics でツールを選択する

**[選択]** ツールは、「[Fusion での選択](https://help.autodesk.com/view/fusion360/JPN/?guid=SLD-SELECTION)」で説明されているように、連続するコマンドでオブジェクトを選択するために使用します。選択は、1 つのオブジェクト、複数のオブジェクト、または図面全体で構成できます。既定では、 **[ウィンドウ選択]** がアクティブです。

## ウィンドウ選択

[ウィンドウ選択]では、オブジェクトを選択する長方形の領域を指定することができます。選択用長方形は、カーソルをクリック アンド ドラッグし、選択したいオブジェクトを含む長方形を作成することで定義します。選択用長方形は、選択するオブジェクトを完全に囲んでいる必要があります。選択したオブジェクトを含む領域の背景は色が変わり、選択されていることが示されます。カーソルをドラッグする方向によって、次のとおり選択方法が決まります。

- **ウィンドウ**: 左から右へクリック アンド ドラッグすると、選択用長方形で完全に囲まれたオブジェクトのみが選択されます。

- **交差**: 右から左へクリック アンド ドラッグすると、選択用長方形が接触しているすべてのオブジェクトが選択されます。

選択用長方形を使用してグループ化すると、その内側のオブジェクトがすべて含まれます。これには、回路図内のコンポーネントとネット、基板上のコンポーネント、ワイヤ、ビアなどがあります。[Ctrl]を押しながらクリックすると個別に選択でき、クリックしたコンポーネントだけが追加されます。

## ポリゴン選択

**[ポリゴン選択]** を使用すると、特定のオブジェクトの周囲に形状を描画して、そのオブジェクトを含めることができます。選択が完了すると、オブジェクトの周囲に長方形が描画され、選択したオブジェクトがハイライト表示されます。ポリゴンを閉じる最も簡単な方法は、マウスの右ボタンを使用することです。表示されている画層のオブジェクトのみを選択に含めることができます。

![ポリゴン選択](../images/workflow/polygon-select.png)

*図 1: ポリゴン選択と長方形内の選択されたオブジェクト*

選択には次のものが含まれます。

- 原点がポリゴンの内側にあるすべてのオブジェクト

- 少なくとも 1 つの端点がポリゴンベタの内側にあるすべてのワイヤ

- 中心がポリゴンベタの内側にあるすべての円

- いずれかのコーナーがポリゴンの内側にあるすべての長方形

より恒久的なグループを作成したい場合は、コンテキスト メニューの **[新規グループ]** ツールを使用します。グループには境界ボックスが表示され、上部にはラベルが表示されます。
