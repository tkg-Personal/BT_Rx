# 電子デザインの図面を作成する

Fusion の **[図面]** 作業スペースで、 **[電子デザイン]** をドキュメント化する図面を作成します。

- 次のいずれかの方法で図面の作成を開始します。

**[電子デザイン]** 作業スペースの **[電子デザイン]** ドキュメントにある**アプリケーション バー**で、 **[ファイル] > [新しい図面] > [電子図面]** をクリックします。

- **[電子デザイン]** 作業スペースの **[電子デザイン]** ドキュメントにある **[共通]** ツールバーで、 **[電子図面]** ![電子図面のアイコン](../images/icon/electronics-drawing.png) をクリックします。

- **[電子デザイン]** 作業スペースで、3D PCB 作業スペース スイッチャをクリックし、 **[図面] > [デザインから]** をクリックします。

- **データ パネル**で、 **[電子デザイン]** を右クリックし、 **[電子デザインから図面を新規作成]** をクリックします。

**[図面を作成]** ダイアログが表示されます。

- **[図面を作成]** ダイアログで、 **[書き出し先]** 設定を選択します。

- **[OK]** をクリックします。

 新しい図面が作成され、**[図面]**作業スペースの独自のタブに表示されます。

- 最初の**図面ビュー**を配置します。

- **[図面]**ツールバーで、次の操作を行います。

**[作成]**ツールを使用して、**図面ビュー**と**スケッチ**を作成します。

- **[修正]**ツールを使用して、図面内の**シート**上のオブジェクトの移動、回転、削除を修正します。

- **[ジオメトリ]** ツールを使用して、中心線、中心マーク、および拡張機能を追加します。

- **[寸法]** ツールを使用して、**図面ビュー**のジオメトリの寸法を記入します。

- **[文字]** ツールを使用して、注記を追加します。

- **[シンボル]** ツールを使用して、シンボルを追加します。

- **[挿入]** ツールを使用して、イメージを追加します。

- **[テーブル]** ツールを使用して、テーブルを追加します。

- **[エクスポート]** ツールを使用して、**シート**を異なるフォーマットにエクスポートします。

- **シート**で、**表題欄**を修正します。

- **[シート]**バーで、次の操作を行います。

図面に新しい**シート**を追加します。

- **[電子デザイン]** ツールバーでは、次の操作を行います。

**[作成]** ツールを使用して、 **[レイヤー スタックの図]** を作成します。

- **[テーブル]** ツールを使用して、 **[レイヤー スタック テーブル]** を追加します。

- 図面を**保存**します。
