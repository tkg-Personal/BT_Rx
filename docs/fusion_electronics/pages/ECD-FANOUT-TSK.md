# ファンアウト

デバイスまたは信号のビアを自動的に配置します。

- [PCB 設計]ツールバーで、 **[ファンアウト]** ![ファンアウトのアイコン](../images/icon/ecd/fanout.png) をクリックします。

- [ファンアウト]ダイアログで、タイプ、スタイル、デザイン ルール違反を許容するかどうか、およびトレースとビアの寸法を確認し、選択します。

- ファンアウトするコンポーネントまたは信号をクリックします。ビアが配置されます。

- [OK]をクリックして完了します。
