# ポリゴンで多層基板を自動配線する

複数の供給電圧に加えて個々のワイヤも含まれているポリゴンで供給レイヤーを作成することができます。

- オートルータを実行する前に、銅箔ベタを定義します。

- 銅箔ベタに適切な信号名を付けます。

- RATSNEST コマンドを使用して、Electronics が銅箔ベタを計算できるようにします。

- オートルータのセットアップでその画層の優先方向および基準コスト(cfBase)を選択します。銅箔ベタ レイヤーの cfBase の値が高いほど、オートルータはこのようなレイヤーを強く回避します。

- 配線後、銅箔ベタが引き続きすべての信号点を接続していることを確認します。信号を配置したためにポリゴンベタが分割された可能性があります。RATSNEST はポリゴンベタを再計算し、「*Ratsnest: Nothing to do! (必要な処理はありません)*」というメッセージを表示します(すべてが正常な場合)。

> **Note:** 
注:

 オートルータはマイクロ ビアを設定できません。画層設定で定義した最大深度よりも短いブラインド ビアを設定することができます。
