# ネット クラスと信号の色をカスタマイズする

- キャンバスで、カスタマイズするネットを選択します。

- **[検査]パネル**で、以下の手順を実行します。

- **[外観]**行で、**[ネット クラス]**または**[信号]**の色ボタンをクリックして、**[カスタム信号色]**ダイアログを開きます。

- **[カスタム信号色]**ダイアログで、プリセット オプションから色を選択するか、カスタム カラーを選択します。

 ![[検査]パネル](../images/how-to/custom-color-inspector.png)

または、以下の手順を実行します。  

- 選択したネットまたは信号を右クリックして、**[プロパティ]**をクリックします。

- 使用する色を選択します。

- **[外観]**行で、**[ネット クラス]**または**[信号]**の色ボタンをクリックして、**[カスタム信号色]**ダイアログを開きます。

- **[カスタム信号色]**ダイアログで、プリセット オプションから色を選択するか、カスタム カラーを選択します。

 ![プロパティ パネル](../images/how-to/net-custom-color-properties.png)

**[適用]**をクリックして、選択した色を設定します。この色が、後から追加される信号を含め、ネット クラスのすべての信号とビアに関連付けられます。

   ![プロパティ パネル](../images/how-to/net-custom-color-properties-sample.gif)

[[信号の色]ビューと[画層の色]ビューを切り替える](ECD-NET-CUSTOM-COLOR-SWITCH.htm)方法については、こちらを参照してください。
