# SMD パッドを配置する

- **[新規電子デザイン ライブラリ]**を開きます。

- **[新規電子デザイン ライブラリ]**作業スペースで、**[作成] > [新しいフットプリント]**をクリックします。

- **[フットプリントを追加]**ダイアログで、新しいフットプリントに名前を付けるか、既存のフットプリントをインポートします。新しいパッドをフットプリントに配置します。

- [ライブラリ フットプリント]ツールバーで、 **[配置] > [SMD パッド]** ![SMD パッドのアイコン](../images/icon/ecd/smd-pad.png) をクリックします。

- **[SMD パッド]**ダイアログで、各パッドのプロパティを設定します。

角度: カスタムの方向角度を設定するか、リストから角度を選択します。

- サイズ: カスタム サイズを設定するか、リストからサイズを選択します。

- 丸み: パッド コーナーの丸みを設定します。パーセンテージが高いほど、コーナー曲線が大きくなります。

- はんだマスクを適用:

自動: [デザイン基本設定]で指定された既定のはんだマスク値を使用します。

- オフ: はんだマスク開口部を無効にします。はんだマスクはパッド全体を覆います。

- オフセット: カスタム サイズを設定するか、リストからサイズを選択します。カスタム値が[デザイン基本設定]の制限を超える場合、2D PCB で許可されている制限に自動的に調整されます。

- ステンシルを適用 :

自動: [デザイン基本設定]で指定された既定のステンシル値を使用します。

- オフ: ステンシルを無効にします。

- パーセンテージ削減: パッド領域のパーセンテージで縮小して、ステンシルの適用範囲を設定します。大きなパッドの場合は、2x2 または 3x3 のグリッドから選択します。

- オフセット削減: ステンシル領域を縮小するカスタム オフセット サイズを設定するか、リストからサイズを選択します。

 オフセットおよびパーセンテージ値が[デザイン基本設定]の制限を超える場合、2D PCB で許可されている制限に自動的に調整されます。

- **[完了]**をクリックして、パッドを配置します。

**注**: パッドには、配置されたシーケンスに従って自動的に名前が割り当てられます。
