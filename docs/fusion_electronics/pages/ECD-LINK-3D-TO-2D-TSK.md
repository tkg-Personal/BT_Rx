# 3D ボードを 2D デザインにリンクする

ほとんどの 3D ボードは 2D ボードから生成され、したがって最初から 2D デザインにリンクされているため、このツールは通常は使用できません。

- **[3D PCB]** ツールバーで、 **[2D PCB にリンク]** ![2D にリンクのアイコン](../images/icon/ecd/link-2d-pcb.png) をクリックします。

- 2D デザインの場所を参照して選択します。

 3D ボードは 2D デザインにリンクされます。
