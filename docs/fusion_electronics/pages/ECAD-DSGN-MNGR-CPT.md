# 電子デザイン マネージャ パネル

**[デザイン マネージャ]** を使用すると、ドキュメント内のすべてのコンポーネント、信号、グループ、PCB スケッチを簡単に移動できます。この機能は、コンポーネントを表示して確認するのが困難な、密集したデザインを使用する場合に特に便利です。**[デザイン マネージャ]** は、情報や配線を抽出する場合に、プロジェクト シートやコンポーネント タイプでオブジェクトを選択する場合に便利です。

**[デザイン マネージャ]** パネルは、次の作業時に使用できます。

- 回路図かいろず

- 回路基板

**[ブラウザ]** タブの **[表示]** フィールドで、[コンポーネント]、[信号]、[グループ]のいずれかを選択できます。

![[デザイン マネージャ]ブラウザ](../images/how-to/ecd-design-mgr-browser.png)

コンポーネントを選択すると、**[項目]**テーブルにサブ要素の属性(信号、パッドの寸法、中心位置、トレースの長さなど)が表示されます。[コンポーネント]リストでは、コンポーネントの位置をロックすることができます。1 つの信号を選択し、そのすべての要素をリップアップ、QuickRoute、またはファンアウトします。要素のグループが作成された場合、グループ タイトルを選択することでグループ全体を移動できます。

**[フィルタ]**タブは、クエリーを作成し、名前でオブジェクト タイプを選択する場合に使用します(任意の数のコンデンサに対して C* など)。この機能を効率的に使用する例として、特定の値のすべてのコンデンサを選択し、[検査]パネルを数回クリックして、すべてのコンデンサを別のモデルのコンデンサに置き換えることができます。

![デザイン マネージャのフィルタ](../images/how-to/design-mgr-filter.png)

デザイン マネージャの右上に、オブジェクト選択に役立つ 2 つのボタンが表示されます。

**[ボード上を選択]** ![ボード上を選択](../images/how-to/select-on-board-icon.png) では、[結果]フィールドで選択したオブジェクトをハイライト表示し、周囲にフレームを描画します。

**[選択にズーム]** ![選択にズーム](../images/how-to/zoom-to-selection-icon.png) では、[結果]フィールドで選択したオブジェクトをより接近して見るためにビューを拡大します。
