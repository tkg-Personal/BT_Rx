# SPICE モデルについて

デザイン プロセスで電子部品シミュレーションを活用するには、シミュレーションで使用するすべてのコンポーネントが、デザイン内のコンポーネントの属性に一致する属性を持っているか、類似していて現実的な解析の基礎を提供する必要があります。このリアル感は、SPICE (Simulation Program with Integrated Circuit Emphasis)モデルをコンポーネントに正しくマッピングすることで実現します。

シミュレーション SPICE モデルは、ライブラリ内の個々のコンポーネントに追加することも、グループとして選択して回路図デザイン内の複数のコンポーネントに追加することもできます。ライブラリ内のコンポーネントに SPICE マップを追加すると、これらのコンポーネントはマッピングしなくても回路シミュレーションで使用できるようになります。提供されているライブラリ *ngspice-digital* および *ngspice-simulation* 内のすべてのコンポーネントには SPICE モデルがあるため、シミュレーションで使用する準備が整っています。その他のコンポーネントの場合、製造元またはインターネットから SPICE モデルを入手できます。
