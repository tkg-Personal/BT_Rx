# ウィンドウ選択を使用してオブジェクトを選択する

- 電子デザイン ドキュメントの **[デザイン]** ツールバーで、**([選択] > [ウィンドウ選択])** ![グループのアイコン](../images/icon/group-icon-16.png) をクリックします。

- キャンバスでクリック アンド ドラッグして、選択するコンポーネントの周囲に選択用長方形を描画します。

- マウスのボタンを放して選択します。
