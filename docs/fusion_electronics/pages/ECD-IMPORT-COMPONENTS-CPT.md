# カスタム ライブラリにコンポーネントをインポートする

**ライブラリ エディタ**では、他のライブラリから電子コンポーネント、記号、またはフットプリントをインポートできます。また、ローカルの .lbr ファイルからコンポーネントを直接インポートすることもできます。これにより、既存のコンポーネント アセットを再利用したり、パブリック ライブラリからアセットをコピーおよび修正したり、ライブラリを Fusion 電子デザイン内で統合することができます。
