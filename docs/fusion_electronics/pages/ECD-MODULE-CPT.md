# モジュールについて

デザインの一部を別のシートで開発する場合は、回路図デザインにモジュールを追加します。複数のモジュールを作成し、各モジュールに同じコネクタ ポートを追加することができます。これにより、各デザイン バリアント内のさまざまなモジュールを使用してデザインを容易にすることができます。
