# 電子デザイン部品表にアクセスする

**[電子デザイン]** 作業スペースで、次のいずれかの操作を行います。

- **[プロジェクト]** の **[共通]** タブで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) をクリックします。

- **[2D 回路図]** の **[ドキュメント]** タブで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) をクリックします。

- **[2D PCB]** の **[ドキュメント]** タブで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) をクリックします。

部品表が開きます。

![BOM ウィンドウ](../images/concepts/bom-access.png)

## 追加情報

- **自動更新:** 設計を変更すると、部品表が自動的に更新されます。

- **マルチタスク:** [部品表]ダイアログを開いたまま、デザインでの作業を行うことができます。

- **交差選択:** 部品表と Fusion キャンバスの両方から項目を選択することができます。一方で行った変更はもう一方に反映されます。
