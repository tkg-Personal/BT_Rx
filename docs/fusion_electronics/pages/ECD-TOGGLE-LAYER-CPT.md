# 単一レイヤー切り替えについて

高密度のデザインを配線する場合、このコントロールを使用して、作業中のレイヤーをフル カラーで、その他すべてのレイヤーをグレースケールで表示します。信号が別のレイヤーに遷移すると、単一レイヤー ビューはレイヤーの変更を追跡し、新しいアクティブなレイヤーのみを表示します。
