# CAM 出力ファイル

これらのファイルのいずれかまたは両方を、製造業者から要求されることがあります。これらを保存するには、ファイルを生成する必要があります。

**IPC-D-356 ネットリスト** ファイルは、各ネットまたはノードの始点と終点のネット名、ピン、XY 位置などの、CAM ソフトウェアに関する指示を含む ASCII テキスト ファイルです。製造業者はこのファイルを使用してテスト フィクスチャを作成し、生成された PCB を検証します。これは、特に大量生産の用途で重要です。

**選択配置**ファイルは図心ファイルとも呼ばれ、回路基板上のすべての表面実装型デバイス(SMD)の空間的な位置を記録します。この位置データには、レイヤー、XY 位置、および回転が含まれます。

選択配置ツールの代わりにユーザ言語プログラム(ULP) *mountsmd* を使用すると、上部レイヤーと下部レイヤーの両方での SMD の位置のファイルを生成できます(該当する場合)。
