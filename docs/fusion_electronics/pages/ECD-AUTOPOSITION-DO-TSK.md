# コンポーネント属性の再配置

このツールを使用して、コンポーネントの名前、値、またはその両方の位置を標準化します。

- 回路図 **[ドキュメント]** ツールバーで、 **[描画] > [属性を再配置]** ![属性を再配置のアイコン](../images/icon/ecd/ecd-reposition-attr-16.png) を選択します。

- [属性を再配置]ダイアログで、次の設定を指定します。

既定のコンポーネントの名前および値の位置の復元と、名前、値、またはその両方の再配置

- 属性ラベルの回転

- コンポーネントに関連した位置の設定(左上や右下など)

- コンポーネントの中央ハンドルをクリックして、属性の位置の変更を適用します。

- 同じ変更を適用するには、追加のコンポーネントをクリックします。

- ダイアログで、[完了]をクリックして操作を終了します。
