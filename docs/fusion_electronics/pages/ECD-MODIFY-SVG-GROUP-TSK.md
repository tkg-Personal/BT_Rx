# 電子デザイン ドキュメントの SVG グループを修正する

SVG グループを移動、回転、またはミラー化します。

- **[電子デザイン]** 作業スペースの開かれたドキュメントで、修正する SVG グループを見つけます。

- キャンバスで、SVG グループを左クリックし、グループの周囲にボックスをドラッグして選択します。

SVG グループの周囲にボックスが表示され、選択されていることが示されます。

- **[デザイン]** ツールバー、 **[記号]** ツールバー(記号の場合)、または **[フットプリント]** ツールバー(フットプリントの場合)で、 **[修正] > [移動]** ![移動のアイコン](../images/icon/common/move.png) をクリックして、 **[移動]** パネルを開きます。

- **[移動]** パネルで、必要に応じて **[回転]** 、**[スピン]** 、および **[ミラー]** の設定を調整します。

- キーボードで **[Ctrl]** を押しながら SVG グループを右クリックして、コンテキスト メニューを開きます。

- コンテキスト メニューで **[移動: グループ]** をクリックして SVG グループを移動し、マウス クリックで配置します。

- SVG グループの方向と位置が適切になるまで、最後の 3 つの手順を繰り返します。

- **[移動]** パネルで **[完了]** をクリックして変更を確定し、パネルを閉じます。
