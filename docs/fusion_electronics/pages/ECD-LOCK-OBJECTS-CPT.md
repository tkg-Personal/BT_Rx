# オブジェクトをロックする

**[ロック]** コマンドを使用すると、ボード上のコンポーネントを固定して、移動、回転、ミラー化できないようにすることができます。これは、特定の位置に取り付ける必要があって誤って移動してはならないコネクタなどの場合に役立ちます。

ロックは、ボード上のコンポーネント、トレース、ビア、穴、ラベル、およびその他のテキストに適用できます。また、グループと一緒に使用することもできます。

> **Note:** 注: 移動するグループにロックされたオブジェクトが含まれている場合、これらのオブジェクトはグループとともには移動しません。ロックされたパーツからアタッチ解除されたテキストは、個別に移動することはできますが、グループとともには移動しません。

ロックされたパーツの原点は「x」として表示され、パーツがロックされていることを視覚的に示します。ロックされたワイヤやビアの場合、ロックされた状態は南京錠の記号で示されます。

ロックは、意図的にロック解除されるまで、セッション間で維持されます。[Shift] + [Lock]で、コンポーネントが再び解放されます。また、 **[すべてロック解除]** ツールを使用して、現在の画層にあるすべてのロックされたオブジェクトのロックを解除することもできます。
