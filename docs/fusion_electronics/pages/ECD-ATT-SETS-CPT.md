# コンポーネント バリアントの属性セット

属性セットには、コンポーネント バリアントのプロパティ値のリストが格納されます。属性セットを使用すると、バリアントごとに複数の設定を作成できます。回路図にコンポーネントを配置するときは、使用するバリアントと属性セットを選択します。各バリアントは、既定という 1 つの属性セットで始まります。

## 属性セットの仕組み

新しい属性セットを作成すると、次のようになります。

- 属性セット名にコンポーネント名およびバリアント名を組み合わせて、完全なパーツ名が作成されます。

- 属性セット名が既定の場合は、コンポーネント名とバリアント名のみが使用されます。

- 各属性セットでは、同じ属性に異なる値を設定できます。

- ライブラリ内の他の属性セットから属性名と値をコピーできます。
