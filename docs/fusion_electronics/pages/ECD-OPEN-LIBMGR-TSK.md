# 電子デザイン ライブラリ マネージャを開く

電子回路図または 2D PCB ドキュメントでは、次のことができます。 

- [ライブラリ]タブのツールバーで、 **[ライブラリ] > [ライブラリ マネージャを開く]** ![ライブラリのアイコン](../images/icon/ecd/lbr.png) をクリックして、[ライブラリ マネージャ]を開きます。

- [配置パネル]で、 **[ライブラリ マネージャを開く]** アイコン ![ライブラリのアイコン](../images/icon/ecd/lbr.png) をクリックして、[ライブラリ マネージャ]を開きます。

または、配置パネルの任意の行を右クリックし、 **[ライブラリ マネージャで開く]** ![ライブラリ アイコン](../images/icon/ecd/lbr.png) を選択して、目的のライブラリが選択された状態のライブラリ マネージャを開きます。

電子デザイン ライブラリ ドキュメントでは、次のことができます。 

- 任意のタブのツールバーで、 **[開く] > [ライブラリ マネージャを開く]** ![ライブラリ アイコン](../images/icon/ecd/lbr.png) をクリックして、ライブラリ マネージャを開きます。
