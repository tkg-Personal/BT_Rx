# 回転

- [デザイン]ツールバーで、**[回転]** ![回転のアイコン](../images/icon/ecd/rotate.png) をクリックします。

- オブジェクトの原点を左クリックして回転します。既定では、反時計回りに 90 度回転します。

- 必要に応じて、オブジェクトを再度左クリックして追加で 90 度回転するか、別のオブジェクトをクリックして回転します。

- 終了したら、[OK]をクリックするか、[Esc]を押します。
