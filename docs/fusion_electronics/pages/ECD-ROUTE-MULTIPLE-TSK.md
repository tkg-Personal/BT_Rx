# 複数の信号を配線する

このオプションは、始点から終点まで平行またはほぼ平行なルートをたどることができる信号のグループに最適です。

- [PCB 設計]ツールバーで、**[複数配線]** ![複数配線のアイコン](../images/icon/ecd/route-multiple.png) をクリックします。

- [複数配線]ダイアログで、選択されているオプションを確認および修正します。

**[トレースの曲げ]**は、トレースの新しい各セグメントの角度または形状をコントロールします。[Ctrl]キーを押しながら右クリックすると、オプションが順に切り替わります。

- ダイアログの下半分にあるコントロールを使用して、トレースおよびビアの物理プロパティを指定します。トレース幅は、コマンド ライン フィールドに数値を入力することによっても変更できます。複数のビアをボード上にどのように配置するかをコントロールする**ビア パターン**のオプションに注意してください。

- クリックして、信号の一方の端にあるパッドまたはコンポーネントのセットを横切る線分をドラッグし、クリックして選択を完了します。オブジェクトがハイライト表示されます。

- カーソルを少し移動し、右クリックしてすべての信号の配線を開始します。

- クリックして、コーナーまたは曲げを手動で配置します。

- 任意の時点で**[QuickRoute Multiple]**を使用して配線を完了する場合は、[Enter]を押します。トレースをチェックして、完了していないトレースを手動で完了する必要があるかどうかを確認します。
