# 3D パッケージを生成する

これは、[ライブラリ作成]ツールバーで**[新しいパッケージ]**をクリックして[ソリッド]ツールバーを開いた後で行います。

- [ソリッド]ツールバーで、 **[パッケージ ジェネレータ]** ![パッケージ ジェネレータのアイコン](../images/icon/ecd/3d-pkg-generator.png) をクリックします。

- [パッケージ ジェネレータ]パネルで、サンプル パッケージのアルファベット順のリストをスクロールして、いずれかを選択して作業します。この例では、[SOIC、SOP]を選択します。

![パッケージ ジェネレータ](../images/how-to/pkg-generator.png)

仕様パネルが開き、コンポーネントの詳細が表示されます。

![SOIC の仕様](../images/how-to/SOIC-specs.png)

パッケージ、フットプリント、サーマル、および製造の詳細を表示するタブに注目してください。

- 仕様パネルでは、要件に基づいて寸法などの詳細を調整できます。この例では、ピンの数(# Pins)を 16 に変更します。

- **[更新]**をクリックして、3D コンポーネントの変更内容を確認します。

![パッケージ ビュー](../images/how-to/3d-pkg-view.png)

パッドの緑の点線のアウトラインや、属性ラベルやコンポーネントのアウトラインなどの他のフットプリントの詳細に注目してください。赤色の矢印で示される二重円は、ピン 1 を示します。ピン 1 は、ボード上のコンポーネントを正しい方向に設定するために使用されます。
