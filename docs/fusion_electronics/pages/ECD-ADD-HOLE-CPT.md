# 非めっき貫通穴(NPTH)

非めっき貫通穴(NPTH)とは、取り付けねじやその他のハードウェアなど、機械的な目的で回路基板に作成される穴のことです。非めっき貫通穴(NPTH)は、非 SMT(表面実装技術)抵抗器、コンデンサ、および DIP パッケージ IC の電気的接続に使用されるめっき貫通穴(PTH)とは異なり、導電コーティングが施されていません。

- **設計上の考慮事項**: NPTH を使用してボードを設計する際には、機械的要件、熱管理、およびボードの性能と信頼性に影響を与える可能性のあるその他の要因を考慮してください。

- **配置**: NPTH を配置する際には、コンポーネントやコネクタ、その他のフィーチャの位置を考慮してください。ショートやその他の問題を防ぐために、NPTH をトレースやパッドの近くに配置しないようにしてください。

- **機械的なサポート**: NPTH は、コンポーネントやその他のハードウェアに機械的なサポートを提供できます。また、ボードをシャーシや筐体に固定するためにも使用できます。

## PCB に NPTH を追加する

3D PCB に NPTH を追加すると、エッジやスケッチ点への参照が失われないように、ロックされている 2D PCB にも自動的に追加されます。2D PCB で NPTH をロック解除して移動または編集する場合は、3D PCB への影響を確認してください。
