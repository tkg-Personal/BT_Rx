# オブジェクトを回転する

**[回転]** は、回路図の記号や PCB レイアウトのフットプリントなどのオブジェクトを回転します。[回転]をクリックすると、[回転]ダイアログが開き、次の便利なオプションが表示されます。

- 回転角度として反時計回りで 0 度、90 度、180 度、270 度を選択します。

- [ミラー]をクリックすると、オブジェクトが左から右に反転します。このオプションは回転と組み合わせることができます。

オプションを使用する場合は、回転するオブジェクトをクリックする前に設定します。

> **Note:** 注: 3D モデルのビューの方向を変更する場合は、キャンバスの右上隅にある ViewCube、またはキャンバスの下部にあるビュー ナビゲーション ツールを使用します。
