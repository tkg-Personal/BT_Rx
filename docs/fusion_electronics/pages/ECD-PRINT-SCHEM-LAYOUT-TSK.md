# 回路図とレイアウトを印刷する

PRINT コマンドを使用すると、回路図、基板、およびライブラリ要素を印刷することができます。

- Fusion ウィンドウの上部で、印刷するドキュメントのタブをクリックします。これにより、画面上でそのドキュメントがアクティブになります。

- 任意の画層をドキュメント表示および印刷の対象から外す場合は、ドキュメントのコマンド ライン ウィンドウに DISPLAY と入力します。[表示画層]ダイアログで、[フィルタ]メニューまたは左側の目のアイコンを使用して、画層をオンまたはオフにします。選択内容がドキュメント ウィンドウに反映されます。完了したら、[OK]をクリックします。

- コマンド ラインに PRINT と入力し、[Enter]を押します。

- [印刷]ダイアログで、使用するプリンタを選択するか、[PDF に出力]などのオプションを選択します。詳細については、後述の「**[印刷]ダイアログでの設定**」を参照してください。

ドキュメント ウィンドウに表示される内容のほとんどが出力されます。例外は次のとおりです。

- テキストの原点クロス

- グリッド線またはグリッド ドット

- RATSNEST で計算できず、レイアウト エディタに輪郭のみが表示される銅箔ベタ

- デザイン ルール チェックのエラー銅箔ベタ

## [印刷]ダイアログでの設定

現在選択されているプリンタが、ウィンドウ上部の**[プリンタ]**行に表示されます。右側の矢印を使用すると、別のプリンタを選択したり、ファイル印刷オプションのいずれかをアクティブにすることができます。プリンタが選択されている場合に、3 つのドットがあるボタンをクリックすると、プリンタのプロパティが開きます。

ファイル印刷オプションを選択すると、**[出力ファイル]**行に出力ファイルへのパスが表示されます。変更する場合は、[...]ボタンをクリックします。

印刷の**[用紙]**サイズ、**[方向]**、および**[位置合わせ]**に関する設定が用意されています。

**[領域]**行で、印刷対象を決定します。**[ウィンドウ]**は、図面のうちエディタ ウィンドウに現在表示されている部分を印刷します。**[完全]**は、図面全体を印刷します。この場合、すべての図面オブジェクト(表示されているかどうかに関係なく)が、出力される印刷領域の計算の対象になります。

#### オプション

**[ミラー]**は Y 軸を中心に図面を左から右へ反転し、**[回転]**は図面を反時計回りに 90 度回転させ、**[上下反転]**は図面を 180 度回転させます。両方をアクティブにすると、270 度回転します。**[黒]**オプションを選択すると、白黒印刷になります。それ以外の場合、プリンタに応じて、カラーまたはグレー スケールで印刷されます。**[ソリッド]**を選択すると、各オブジェクトが完全に塗り潰されます。レイヤーごとに異なる塗り潰しパターンを表示する場合は、このオプションを非アクティブにします。**[キャプション]**オプションを選択すると、タイム スタンプとファイル パスを含むキャプションの表示/非表示を切り替えることができます。

**[尺度係数]**には、図面の尺度を指定します。値の範囲は 0.001 ～ 1000 です。**[ページ上限]**を 0 に設定すると、選択した尺度で出力を印刷するために必要な分だけページ数が使用されます。別の値を選択すると、その指定したページ数に合わせて図面の尺度が調整されます。つまり、あいにく、選択した尺度を維持できない場合があります。一方、[ページ上限]に 1 を選択した場合でも、選択する尺度係数によっては、印刷対象の複数のページが 1 ページに最大限収まることもあります。

[シート]ボックスを使用すると、回路図から印刷するシートを選択することができます。これは、回路図エディタにのみ表示されます。この選択によって、プレビューに表示されるシートも決まります。

印刷のエッジは、**[境界]**の下にある 4 つの入力ボックスを使用して定義できます。値は、ミリメートルまたはインチ単位で入力できます。値を変更した後で、プリンタ ドライバの標準設定を再度使用する場合は、0 を入力するだけで済みます。

**[位置合わせ]**により、印刷物の縦横比の補正係数を指定できます。これにより、印刷の寸法精度の線形誤差を補正できます。値は、0.1 ～ 2 の範囲で指定することができます。

カラー印刷に関する注意: Electronics では、常にカラー印刷物の基礎として白のパレットが使用されます。黒またはカラーの背景で作業していて、自分で定義した色を使用している場合は、これらの色を白のパレット用にも定義することをお勧めします。そうすると、プリンタでもその色を使用できるようになります。

レイアウトを印刷するときに、パッドやビアのドリル穴が表示されない場合は、[オプション]/[設定]/[その他]のメニュー項目を使用して、[表示モード]で[ドリルなし]オプションを選択します。

#### PDF ファイルを生成する

図面から PDF ファイル(解像度 1200 dpi)を生成する場合は、[プリンタ]行の小さな選択矢印をクリックし、[ファイルに出力(PDF)]オプションを選択します。[出力ファイル]行に移動し、PDF 出力ファイルのパスと名前を指定します。PDF ビューアを使用すると、PDF ファイル内の Electronics ベクトル フォントで書かれていないすべてのテキストを検索することができます。

PRINT コマンドは、コマンド ラインに直接入力することも、スクリプト ファイルで実行することもできます。
