# 新しい電子コンポーネントを作成する

## 前提条件

- [既存のライブラリを開く](ecad-edit-custom-lib.htm)か、[新しいライブラリを作成](ecad-create-custom-lib.htm)します。

## 手順

- ツールバーの **[作成]** グループで、 **[新規コンポーネント]** ![新規コンポーネントのアイコン](../images/icon/ecd/create-device.png) を選択します。これにより、コンポーネント エディタが開きます。

## 次のステップ

- コンポーネント エディタを使用して、記号とバリアントをコンポーネントに追加します。

- **[属性]ダイアログ**を使用して、コンポーネントまたはコンポーネント バリアントの属性を設定または作成します。
