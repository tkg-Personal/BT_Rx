# IPC ネットリストを生成する

IPC ネットリストは、ユーザ言語プログラム(ULP) *ipc-d-356* を実行して生成することもできます。

- キャンバスで回路基板を開いた状態で、[製造]ツールバーで、**[IPC-D-356 ネットリスト]** ![IPC ネットリストのアイコン](../images/icon/ecd/ipc.png) をクリックします。

- 開いたダイアログで、提案された IPC ファイルを保存するためのフォルダが適切であることを確認します。そうでない場合は、新しい場所を参照します。

- **[保存]**をクリックします。
