# QuickRoute コマンドを実行する

これらのコマンドは、ユーザによるコントロールを使用しない半自動配線方法を提供するため、配線操作で多くの時間を節約できます。

![QuickRoute メニュー](../images/how-to/quick-route-menu.png)

選択した信号を開始から終了まで配線するには、**[QuickRoute Airwire]**を使用します。

同じ名前の信号セグメントのグループを配線するには、**[QuickRoute Signal]**を使用します。このコマンドの制限は、16 より多いセグメントを持つ信号では実行できないことです。

関連するパッドのセットから信号のグループを配線するには、**[QuickRoute Multiple]**を使用します。

- メニュー オプションを選択します。ダイアログが開き、必要に応じてトレース幅を変更できます。

- クリックしてパッド間に線分を描画し、選択します。

- 右クリックすると、すべての信号が一度に配線されます。

![QuickRoute Multiple の GIF](../images/how-to/quick-route-multiple.gif)

基板上の既存のトレースを最適化または単純化し、不要な曲げやセグメントを除去するには、**[QuickRoute Smooth]**を使用します。

関連するパッドのセットから信号のグループの配線をガイドするには、**[QuickRoute Guided]**を使用します。このオプションを使用して、基板上の別のコンポーネントの周囲を移動する場合などに、トレース コーナーが発生する場所を詳細にコントロールできます。また、ある画層から別の画層にトレースを進めることもできます。 

- メニュー オプションを選択します。ダイアログが開き、関連するレイヤーの選択、トレースの幅とクリアランスの変更、およびビアの形状とサイズの修正を行うことができます。

![QuickRoute Guided のダイアログ](../images/how-to/quick-route-guided-db.png)

- クリックしてパッド間に線分を描画し、選択します。選択の終了時にもう一度クリックします。

- 右クリックして、選択を終了します。

- クリックすると、1 つの開始パッドから終了パッドまでのガイド パスが描画されます。ダブルクリックしてパスを終了します。

- [Enter]を押して、パスを作成します。 

次のアニメーションは、単一レイヤーのトレースと 2 つのレイヤーのトレースに対する QuickRoute Guided の使用例を示しています。

![QuickRoute Guided の GIF](../images/how-to/quick-route-guided.gif)

可能な限りガイド パスに近い位置にトレースが引き寄せられることに注意してください。
