# 2D PCB に非めっき穴を追加する

- **[電子デザイン]** 作業スペースにある 2D PCB ドキュメントの **[デザイン]** ツールバーで、 **[配置] > [穴(NPTH)]** ![穴のアイコン](../images/icon/ecd/hole-npth.png) をクリックし、非めっきの貫通穴を追加します。

- [穴(NPTH)]ダイアログで、 **[ドリル]** 設定(穴の直径)を確認および調整します。

- 目的の穴の位置にカーソルを移動します。

> **Note:** 注: キャンバスの上部に、デザインの原点からの X 距離と Y 距離が表示されます。

- クリックして穴を配置します。
