# コンポーネント

1 つまたは複数の記号とフットプリント、さらに必要に応じて 3D パッケージで構成される、ライブラリ内の完全に定義された要素。関連用語として「フットプリント」、「パッケージ」、「記号」を参照してください。

- **コンポーネント**: 完全に定義されたコンポーネントは、1 つの記号、1 つまたは複数のフットプリント、および必要に応じて 3D パッケージで構成されます。これらはライブラリに保存されます。

**コンポーネント**は、1 つまたは複数の記号とパッケージで構成されています。

- コンポーネントの 3D モデルは、STEP ファイル形式で割り当てられたモデルにすることができます。3D モデルはオンライン リポジトリで提供されます。また、ユーザが独自にアップロードした 3D モデルを使用することもできます。提供されているすべてのライブラリには、既定で単純な 3D ボックスが割り当てられています。これは、Web ベースのエディタを使用して 3D STEP ファイル モデルで置き換えることができます。
