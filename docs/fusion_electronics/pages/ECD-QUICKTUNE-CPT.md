# QuickTune

QuickTune はミアンダを追加することで、複数のトレース、差動ペア、またはバスの長さを自動的に調整します。

![[QuickTune]ダイアログ](../images/how-to/quick-tune-dialog.png)

[QuickTune]ダイアログには、既定値が自動的に入力されます。

- ターゲットの長さ: 選択した単一または複数のトレースの希望の長さを入力します。

- 許容差: カスタムの長さ範囲を設定するか、最小および最大の長さを自動(5 mil)のままにします。

- 最小振幅: ミアンダ波の最小距離を設定するか、自動(0 mil)のままにします。

- 最大振幅: ミアンダ波の最大距離を設定するか、自動(0 mil)のままにします。

- ギャップ: ミアンダの間隔を設定します。既定のギャップは、2.5 mil の係数で計算されます。ギャップを減らすことはできますが、銅箔クリアランスより小さくすることはできません。[デザイン ルール]エディタでギャップ係数を調整します。

**注**: 差動ペアまたは複数のトレースの場合、[ターゲット長さ]は長い方のトレースに一致します。単一トレースの場合は、現在の長さと一致します。
