# パッケージを作成する

ここでは、新しいカスタム ライブラリで 3D パッケージおよび関連するフットプリントを作成する方法について説明します。通常、このプロセスでは、製造元またはディストリビュータから取得したパーツの仕様シートで提供される詳細を使用します。

- [ファイル]メニューで、**[新規電子デザイン ライブラリ]**をクリックします。

- **[ファイル] > [保存]** をクリックし、[保存]ダイアログで、ライブラリの名前を指定し、既存のプロジェクトに保存するか、または、[保存先]フィールドの右端で下矢印をクリックし、 **[新規プロジェクト]** をクリックして名前を指定してから **[保存]** をクリックします。

- ウィンドウの左上で、 **[新しいパッケージ]** ![新しいパッケージのアイコン](../images/icon/ecd/create-package.png) をクリックします。リボンの[パッケージ]パネルがアクティブになります。

![パッケージ パネル](../images/how-to/ecd-package-panel.png)

パッケージを最初から作成する場合は、[ソリッド]をクリックし、Fusion の[図面]ツールを使用して 3D イメージを準備することができます。「*3D パッケージを生成する*」というトピックには、3D パッケージ ジェネレータで作業する、より一般的なプロセスの完全な例が示されています。
