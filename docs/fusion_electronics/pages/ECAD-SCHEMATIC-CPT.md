# 回路図ドキュメント

すべての電子デザインでは、回路図から始めて、デザインのロジックと機能をドキュメント化する必要があります。Fusion では、回路図は最大 999 枚のシートで構成できます。回路図には、次の 2 つの基本的な目的があります。

- 元のコンセプトの意図を伝えること

- プリント基板(PCB)のレイアウトの基礎を提供すること

回路図の基礎となるのはコンポーネントで、これはデザインにおける物理パーツを表します。コンポーネントはネットによって接続され、これはコンポーネント間の電気的接続を表します。Fusion 回路図エディタには、回路図の作成と編集に役立つさまざまなツールが用意されています。

PCB の意図を伝えるために、各コンポーネントに参照指定子が割り当てられます。これは、PCB 上の各コンポーネントを識別する英数字コードであり、コンポーネントごとの記号の数を表します。参照指定子は、部品表(BOM)のすべてのコンポーネントが基板上のどこに配置されているかも示します。最後に、部品表に加えて、回路図はネット リストの作成にも使用できます。ネット リストとは、すべてのコンポーネントとその接続のリストです。

ビデオ: 回路図の作成方法のデモンストレーション

主要な概念:

- コンポーネントを配置および移動する

- 接続を定義する

- 電力を追加する

- コンポーネントへの値の割り当て
