# レイヤー スタックの図を追加する

Fusion の **[図面]** 作業スペースに、 **[PCB レイヤー スタック]** に基づく図を図面に追加します。レイヤー スタックの図は、PCB とビア ペアのすべてのレイヤーを抽象的な図で示します。

- **[図面]** 作業スペースの **[電子デザイン]** ツールバーで、 **[作成] > [レイヤー スタックの図]** ![レイヤー スタックの図のアイコン](../images/icon/layer-stack-diagram.png) をクリックします。

- クリックして、レイヤー スタックの図をシートに配置します。

 レイヤー スタックの図がシートに表示されます。
