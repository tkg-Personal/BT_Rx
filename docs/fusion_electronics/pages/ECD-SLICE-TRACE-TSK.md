# ネットまたはトレースをスライスする

- [回路図デザイン]ツールバーで **[修正] > [スライス]** をクリックするか、[PCB 設計]ツールバーで **[やり直し] > [スライス]** ![スライスのアイコン](../images/icon/ecd/slice.png) をクリックします。

- [スライス]ダイアログで、スライスの幅とスライスの動作(リップアップするかどうか、およびどの側のセグメントをリップアップするか)を確認または選択します。

- パスの片側で、クリックしてスライス線の始点を設定します。

- パスの反対側にカーソルをドラッグし、クリックして終点を設定します。選択したスライスの動作が適用され、ボード トレースの場合、結果として生成されるエアワイヤ セグメントを再配線に使用できます。
