# CAM プロセッサについて

[CAM プロセッサ]ダイアログは、画層の追加やボードのセクションの削除などの、手動での変更の必要性を確認するために役立ちます。ここではまた、製造出力ファイルを検査することもできます。アセンブリ(TXT)、ドリル(XLN)、および Gerber (GBR)のファイル用に、個別のサブフォルダが作成されます。

既定のテンプレートの代わりに使用する CAM ジョブ テンプレートがある場合は、ジョブ テンプレート ファイルをロードしてアクティブな CAM ジョブにすることができます。

CAM ジョブ ファイルを圧縮パッケージとしてエクスポートする場合、エクスポートの保存先、およびエクスポートの単位を指定するオプションがあります。
