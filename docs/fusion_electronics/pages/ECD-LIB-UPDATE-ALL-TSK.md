# デザイン内のライブラリ コンポーネントを更新する

デザイン内のコンポーネントを、リンクされたライブラリの新しいバージョンで更新します。

## ライブラリからデザインを更新する

特定のライブラリからコンポーネントを更新します。

- **[ライブラリからデザインを更新]** ![ライブラリから更新のアイコン](../images/icon/ecd/update-library.png) をクリックします。

- [更新]ダイアログで、新しいバージョンを使用できる 1 つまたは複数のライブラリを選択し、 **[更新]** をクリックします。

## すべてのライブラリからデザインを更新する

デザインで使用されているすべてのライブラリからコンポーネントを更新します。

**[デザインで使用されているすべてのライブラリからデザインを更新]** ![ライブラリから更新のアイコン](../images/icon/ecd/update-all-libraries.png) をクリックします。更新がすべてのライブラリから自動的にロードされます。
