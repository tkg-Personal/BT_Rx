# 3D PCB コンポーネントを反転する

**[コンポーネントを反転]** は、コンポーネントを X 軸に沿って回転させ、両面 PCB の反対側に配置します。

- **[電子デザイン]** 作業スペースの開いている 3D PCB ドキュメントで、反転するコンポーネントをクリックするか、[Shift]キーを押しながら複数のコンポーネントをクリックして選択します。

- **[3D PCB]** ツールバーで、 **[修正] > [コンポーネントを反転]** ![3D を反転のアイコン](../images/icon/ecd/flip-3d.png) をクリックして、 **[コンポーネントを反転]** ダイアログを開きます。

単一または複数のコンポーネントが、ボードの反対側に瞬時に反転します。

- **[コンポーネントを反転]** ダイアログで、 **[OK]** をクリックして変更を適用するか、 **[キャンセル]** をクリックして変更を元に戻してダイアログを閉じます。
