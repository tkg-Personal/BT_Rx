# Forward&Back Annotation

回路図ファイルと基板ファイルの両方が開いている場合、Forward&Back Annotation により、両ファイルが自動的にリンクされ、一方のファイルで変更を加えると、もう一方のファイルにも自動的に変更が反映されます。

たとえば、新しいコンポーネントを配置すると、関連付けられているハウジングがレイアウトの基板の端に表示されます。ネットを配置すると、信号線が同時にレイアウト上に描画されます。

> **Note:** 重要: 回路図ファイルと基板ファイルは常に同時に開いていなければなりません。そうでないと、Forward&Back Annotation が機能せず、ファイルが同期しなくなります。

信号の配置や削除など、特定の操作は回路図でのみ可能です。コンポーネントの名前や値の変更など、その他の操作は、いずれのファイルでも実行できます。

> **Note:** 注: 一方のファイルを閉じた状態で他方のファイルを編集し、両方のファイルを開くと、電気的ルール チェック(ERC)によってファイルの整合性がチェックされ、不整合が見つかると適切な警告が表示されます。
