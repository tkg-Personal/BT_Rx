# 属性グループを削除する

電子デザイン ライブラリのコンポーネント バリアントから属性を削除します。

## 前提条件

- [[属性]ダイアログを開きます](ECD-OPEN-ATT-DIALOG-TSK.htm)。

- 目的のコンポーネント バリアントの属性が表示されていることを確認します。

## 手順

- [属性]ダイアログの左側の列で属性名を右クリックし、 **[属性を削除]** を選択します。

- **[削除]** をクリックして現在のバリアントから属性を削除します。テーブルに行が表示されなくなります。

- 値がコンポーネントまたはライブラリ スコープとして格納されていた場合は、コンポーネントまたはライブラリ全体の属性が削除されます。ダイアログ内のメッセージを参照して、削除されるスコープを確認します。

- (省略可能)削除の適用範囲を変更します。

**他のバリアントおよびコンポーネントの値を保持します。**: 属性を現在のバリアントからのみ削除し、ライブラリ内の他のバリアントおよびコンポーネントで引き続き使用できるようにします。これは、削除されるスコープ内に他のコンポーネントまたはバリアントがある場合にのみ表示されます。

- **ライブラリ全体から属性を削除します。**: スコープに関係なく、ライブラリ全体のすべてのコンポーネントおよびバリアントから、この名前を持つすべての属性を削除します。
