# DXF/DWG ファイルを挿入する

- [デザイン]ツールバーで、 **[配置] > [DXF/DWG を挿入]** ![回路図を挿入のアイコン](../images/icon/ecd/ecd-insert-dxf-dwg.png) を選択します。

- [ファイルを開く]ダイアログで、挿入する DXF/DWG ファイルを参照して選択します。[選択]をクリックします。  

- [DXF/DWG を挿入]ダイアログで、次を指定します。

[配置原点]: 明示的な XY 座標を使用する場合に使用します。もしくは、挿入値の完了後、キャンバス内でファイルの原点となる位置をクリックします。

- [ターゲット画層]: 図面が配置されるドキュメント内の画層を指定します。

- [単位]: ファイルをインポートするときに使用します。

- [入力画層]: 挿入されたファイルで使用する画層を指定します。

- [尺度係数](既定は 1): 挿入したファイルを配置する前に尺度を変更します。

- [線幅]: ドロップダウン リストから選択します。

- 挿入するファイルを拒否または変更するには、ダイアログ上部のファイル名の横にある **X** をクリックします。参照して別のファイルを探します。

- [完了]をクリックします。

インポートした図面の内容が現在のドキュメントに追加されます。
