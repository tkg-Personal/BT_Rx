# 電子デザインでのコンポーネントの配置

同期を維持するには、PCB ではなく回路図デザインにコンポーネントを追加する必要があります。回路図デザインはプロジェクトのマスター ドキュメントであり、PCB 設計はそこから派生します。回路図にコンポーネントを追加すると、自動的に PCB 設計にも追加されます。

> **Note:** 注: PCB 設計にコンポーネントを追加しても、リンクされた回路図デザインには追加されません。

2D PCB ドキュメントで作業している場合は、コンポーネントをボード上に直接配置することができます。ただし、パッドを含むフットプリントは、関連付けられた回路図がない限り、リンクされた回路図から配置する必要があります。
