# チュートリアル: 電子部品ライブラリの管理

電子デザインを作成する場合、特定のコンポーネントをベンダーから選択する必要があります。選択できるコンポーネントは何千種類もあります。ほとんどのコンポーネントは、Fusion とともにインストールされません。ただし、必要なコンポーネントは多くの場合、ダウンロードできます。

このチュートリアルの内容:

- 電子デザインに必要なコンポーネント ライブラリを見つけます。

- [ライブラリ マネージャ]ダイアログでライブラリを有効にします。

- 電子デザインにライブラリを追加します。

![ライブラリ マネージャのダイアログ](../images/tutorial/ecad-manage-libraries/ecd-tut-lib-fuselec-ic-interface.png)

*ライブラリが追加されると、[使用中]列のトグルが有効として表示されます。*

## 前提条件

- 回路図ドキュメントで[電子デザイン]作業スペースが開いていること。
