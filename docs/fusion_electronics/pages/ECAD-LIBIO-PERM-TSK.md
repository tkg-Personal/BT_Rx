# Library.io フォルダの権限を管理する

ライブラリ管理タスクを調べるには、任意の Web ブラウザで **library.io** を参照します。

![Library.io のコントロール](../images/how-to/library-io-panel.png)

- **[メンバーを管理]**ボタンをクリックして、チーム メンバーの電子メール リストを準備し、チーム メンバーを招待して、管理フォルダ内のライブラリへのアクセスを共有します。

- 管理フォルダへのアクセス権を持つすべてのユーザが Library.io のライブラリに変更をプッシュできるため、信頼できる作成者にのみ権限を付与してください。
