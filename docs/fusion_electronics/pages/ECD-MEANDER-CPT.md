# ミアンダ

[ミアンダ]ツールはミアンダ パターンを追加することにより、トレースの長さを一致させます。ミアンダを適用して、単一のトレースまたは差動ペアの長さを調整します。

[ミアンダ]ダイアログ ボックスには、既定値が自動的に入力されます。

- ターゲットの長さ: 選択したトレースの希望の長さを入力します。

- 許容差: 長さ調整の許容範囲を入力します。

- ギャップ: ミアンダ波とミアンダ波の間隔。既定のギャップは、2.5 mil の係数で計算されます。ギャップを減らすことはできますが、銅箔クリアランスより小さくすることはできません。[デザイン ルール]エディタでギャップ係数を調整します。

- 対称:

非対称: 1 つの信号にミアンダを適用します(差動ペアにおいては自動的に最小になります)。

- 対称: 両方の信号にミアンダを適用します。

**注**: 差動ペアの場合、ターゲットの長さは長い方の信号の長さです。単一トレースの場合は、現在の長さです。
