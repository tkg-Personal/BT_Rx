# デザイン ルール チェックを実行する

- PCB ルールのツールバーで、 **[ルール] > [デザイン ルールをチェック]** をクリックします。

- (省略可能)コマンド ライン インタフェース(CLI)に DRC と入力します。

- エラーは、 **[エラー]** ![エラー アイコン](../images/icon/ecd/errors.png) パネルに表示されます。[エラー]パネルを手動で開くには、CLI に ERRORS と入力します。

- **[エラー]** ![エラー アイコン](../images/icon/ecd/errors.png)パネルを使用して、エラーを確認、修正、承認します。ライブ チェックインの基本設定をコントロールし、[エラー]パネルから手動で更新します。
