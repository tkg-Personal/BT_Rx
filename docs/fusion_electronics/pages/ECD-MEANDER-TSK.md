# 差動ペアにミアンダ形状を追加する

- **[PCB デザイン]**ツールバーで、**[配線] > [ミアンダ]**または**[やり直し] > [ミアンダ]** ![ミアンダのアイコン](../images/icon/meander-icon.png) をクリックします。

- 長さを調整する単一のトレースまたは差動ペアを選択します。

- **[ミアンダ]**ダイアログで、既定値を調整します。ミアンダはトレース上で選択した点から開始します。

- トレースに沿ってクリックして、ミアンダ パターンを開始します。ターゲットの長さと選択したトレースの長さがヘッドアップ ディスプレイ(HUD)に表示されます。カーソルを動かすと値が更新されます。

- ミアンダ曲線の形状と長さを調整するには、トレース上でカーソルを移動します。HUD の情報が緑色に変わり、チェック マーク アイコンが表示されます。

 ![ミアンダ形状の HUD](../images/how-to/meander-do.png)

- ハイライト表示されたセクションの先をクリックして、ミアンダを保存します。

- 次のトレースに進むか、**[完了]**をクリックします。

**注**: ミアンダを削除してやり直すには、**[元に戻す]**をクリックします。
