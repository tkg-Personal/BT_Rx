# ビアを配置する

- **[2D PCB]** 作業スペースで、 **[配線] > [ビア]** ![ ビアのアイコン](../images/icon/ecd/via.png) をクリックします。

- **[VIA]**ダイアログで、各ビアのプロパティを設定します。

ビアの長さ: リストからビアの長さを選択します。

- パッド形状: 正方形、円形、または八角形から選択します。

- 寸法: 既定値を設定するには、[自動]を使用します。カスタム寸法を設定するか、リストから寸法を選択します。

- はんだマスクを適用します。

自動: [デザイン基本設定]で指定された既定のはんだマスク値を使用します。

- オフ: はんだマスク開口部を無効にします。はんだマスクはビア全体を覆います。

- オフセット: カスタム サイズを設定するか、リストからサイズを選択します。カスタム値が[デザイン基本設定]の制限を超える場合、許可されている制限に自動的に調整されます。

- カーソルを使用してビアを配置します。選択した場所がデザイン ルールに違反している場合、ビアは最も近い許容可能な場所に自動的に移動されます。

- **[完了]**をクリックして、ビアの配置を終了します。
