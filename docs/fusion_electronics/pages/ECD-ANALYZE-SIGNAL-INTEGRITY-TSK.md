# 信号の整合性を解析する

Fusion Signal Integrity Extension

 この機能は拡張機能の一部です。拡張機能は Fusion の追加機能にアクセスするための柔軟な方法です。[詳細情報](https://help.autodesk.com/view/fusion360/JPN/?contextId=EXT-OVERVIEW)。

![信号 ui イメージを分析する](../images/how-to/ecd-analyze-signal-ui.png)

2D PCB ドキュメントをアクティブにしてコンポーネントを配置し、次の操作を実行します。

- PCB の**[シミュレーション]**ツールバーで、**[信号を解析]** ![信号を解析のアイコン](../images/icon/ecd-pcb-impedance.png)をクリックします。

- ボード上の信号のリストから信号を選択します。

- 任意の周波数を指定します。

- 目標インピーダンス (Z0) を指定します。

- 目標インピーダンスのパーセンテージとして、許容インピーダンス範囲を構成する許容値を指定します。

- **[信号を解析]**をクリックします。入力フィールドの下の領域に結果が表示されます。

- [結果]セクションで、信号パスかセグメントを選択します。これらの選択はキャンバス内でクロスハイライト表示されます。解析結果を確認し、適切な調整を行います。パスまたはセグメントをダブルクリックすると、そのオブジェクトの範囲にキャンバスがズームされます。

- キャンバス オーバーレイの表示を切り替えるには、**[インピーダンス オーバーレイ]** ![インピーダンス オーバーレイのアイコン](../images/icon/ecd-pcb-impedance.png) または**[オーバーレイなし]** ![オーバーレイなしのアイコン](../images/icon/ecd-si-no-overlay.png) をクリックします。
