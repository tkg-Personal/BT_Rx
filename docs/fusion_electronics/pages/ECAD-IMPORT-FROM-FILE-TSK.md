# ローカルの .lbr ファイルから電子コンポーネントをインポートする

## 前提条件

- [既存のライブラリを開く](ECD-EDIT-CSTM-LIB-TSK.htm)か、[新しいライブラリを作成](ECD-CREATE-CSTM-LIB-TSK.htm)します。

## 手順

- ツールバーの **[インポート]** グループで、 **[コンポーネントをインポート]** をクリックします。[コンポーネントをインポート]ダイアログが開きます。

- ダイアログの左下で、 **[ローカル ファイルからインポート]** をクリックします。

- ファイル ブラウザを使用して、インポートする .lbr ファイルを選択し、 **[開く]** をクリックします。

- **[インポートするコンポーネントを選択]** ダイアログで、インポートするコンポーネントを選択します。各ライブラリを展開して、そのコンポーネントを表示できます。

- **[OK]** をクリックして、選択したコンポーネントをインポートします。

- 重複するコンポーネント名が見つかった場合は、重複するコンポーネントに新しい名前を入力するように求められます。

## 次のステップ

コンポーネントをインポートした後、次の操作を実行できます。

- 必要に応じて、インポートしたコンポーネントを編集および修正します。

- 新しいパッケージを作成して、インポートしたフットプリントに 3D モデルをアタッチします。
