# 概要 - 寸法

引出線、オフセット、および属性にさまざまな寸法タイプを使用する場合、信頼性を高めるために実験が必要です。属性に関する注意事項:

- **[表示]**設定は、数値ラベルに単位を含めるかどうかをコントロールします。

- **[比率]**は、テキスト サイズに対するテキストの太字の度合いを設定します。

- **[精度]**は、小数点以下の最大桁数を設定します。
