# QuickRoute 複数のエアワイヤ

- [PCB 設計]ツールバーで、 **[QuickRoute Multiple]** ![QuickRoute Multiple のアイコン](../images/icon/ecd/quick-route-multiple.png) をクリックします。

- [QuickRoute Multiple]ダイアログで、配線するレイヤー、およびトレースとビアの属性を確認または選択します。

- クリックして、配線するパッド間を横切る線分を描画します。選択したパッドがハイライト表示されます。

- クリックして選択を終了します。

- 右クリックしてエアワイヤをトレースに変換します。

- ルートが許容可能であることを確認します。
