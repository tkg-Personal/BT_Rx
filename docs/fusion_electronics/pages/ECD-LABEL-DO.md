# ネット ラベルまたはバス ラベルを追加する

- [回路図デザイン]ツールバーで、**[ラベル]** ![ラベルのアイコン](../images/icon/ecd/label.png) をクリックします。

- ネットが別の回路図シートに移動し、ラベルで接続点の位置を記録する場合は、[ラベル]ダイアログで[外部参照]をオンにします。

- ラベルを配置するおおよその位置でネットまたはバスをクリックします。ラベルがカーソルの横に表示されます。

- 必要に応じてラベルの位置を移動し、ラベルを右クリックすると、クリックするたびに反時計回りに 90 度回転します。

- クリックしてラベルを配置します。

- ラベルを付ける別のネットの場所をクリックするか、[Esc]を押してラベル付けを終了します。
