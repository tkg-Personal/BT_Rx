# スプラインを使用して 2D PCB アウトラインを作成する

スプラインを使用して、2D プリント基板(PCB)の物理的な境界を定義します。

- ボードのアウトラインが存在する場合は、削除します。選択用長方形を使用してボード全体を囲み、 **[削除]** をクリックすることができます。

> **Note:** 注: 回路コンポーネントとエアワイヤは削除されません。

- PCB の **[設計]** ツールバーで、 **[アウトライン スプライン]** ![アウトライン スプラインのアイコン](../images/icon/ecd/outline-spline.png) をクリックします。

- 各制御点をクリックし、アウトラインを閉じるときにダブルクリックして、ボードのアウトラインを描画します。ボードは閉じた図形に拘束されます。

- アウトラインの形状を変更するには、アウトラインをクリックして制御点を表示します。これらの点を移動して、スプライン曲線を調整します。
