# 2D PCB でボード アウトラインを編集する

**[ボード アウトライン]** レイヤーは、PCB の外形寸法を定義するために使用されます。**2D PCB** で **[ボード アウトライン レイヤー]** に描画するには、いくつかの方法があります。このレイヤーは、3D PCB アウトライン スケッチと同期します。

- ボード形状ツールを使用する

- BoardOutline 画層と描画線に切り替える

- 線分の画層をボードのアウトラインに変更する

- BoardOutline レイヤーに関する情報を含むコンポーネントを追加する

2D ドキュメント キャンバスでは、ボードのアウトラインがオレンジ色で表示されます。ボードのアウトラインは、次の方法で変更できます。

- **アウトライン ポリライン**

- **アウトライン スプライン**

- **アウトライン円弧**

- **アウトライン円**

 ![アウトライン ポリライン](../images/how-to/board-outline.png)

 *図 1: 線幅やコーナーの形状などの属性を指定できる[アウトライン ポリライン]パネル*

## 2D PCB での読み取り専用

ボードのアウトラインが 3D PCB で拘束または投影を使用して定義されている場合、2D PCB では読み取り専用になります。スケッチ データを保持するには、3D PCB でのみ編集することをお勧めします。
