# ピンを追加する

- ライブラリが開いた状態で、**[新しい記号]**をクリックし、最初から描画するか、または既存の記号をベースとしてインポートするかを選択します。

- [ライブラリ記号]ツールバーで、 **[ピン]** ![ピンのアイコン](../images/icon/ecd/pin.png) をクリックします。

- [ピン]ダイアログで、ピンを配置する前に、各ピンについて次のプロパティを確認または設定します。

[方向を回転]: 右クリックして、4 つのオプションを循環することができます。

- [機能]: [なし]、[Dot (反転)]、[Clk (時計)]、[DotClk (反時計)]

- [長さ]: [点]、[短い]、[中央]、[長い]

- [表示]: [方向]、[方向とピン名]

- [方向]: nc (未接続)、in、out、io、oc (オープン コレクタ)、pwr、pas (パッシブ)、hiz (高インピーダンス)、sup (一般供給ピン)

- [SwapLevel]: 0 以上の整数

- クリックして各ピンを配置し、必要に応じてプロパティを変更します。
