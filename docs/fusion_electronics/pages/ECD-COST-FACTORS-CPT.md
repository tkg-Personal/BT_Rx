# コスト係数が配線プロセスにどのように影響するか

各コスト係数(cfxxx)の値は 0 から 99 までの値が可能ですが、全範囲がすべてのパラメータに対して有益というわけではありません。したがって、各パラメータには合理的な値を指定します。

コントロール パラメータ(mnxxx)は、0 ～ 9999 の範囲で指定できます。各パラメータに対して妥当な桁数を指定します。

パラメータは、オートルータのセットアップ メニューを使用して設定します。[ルート]パスと[最適化]パスの設定は、個別に設定できます。メニューは、[画層のコスト]、[コスト]、[最大]の 3 つのセクションに分かれています。
