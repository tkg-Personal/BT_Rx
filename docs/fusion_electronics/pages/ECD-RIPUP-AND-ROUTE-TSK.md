# コンポーネントまたは信号を再配線する

- [PCB 設計]ツールバーで、 **[やり直し] > [再配線]** ![再配線のアイコン](../images/icon/ecd/ripup-&-route.png) をクリックします。

- 影響を受けるコンポーネント、信号、またはパッドをクリックします。必要に応じて、同じレイヤー上の接続されているすべての信号が再配線されます。

- 再配線された信号で、手動での修正が必要かどうかを確認します。
