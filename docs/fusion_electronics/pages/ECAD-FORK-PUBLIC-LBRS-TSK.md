# パブリック ライブラリを分岐する

ハブにパブリック ライブラリのコピーを作成し、プロジェクトに合わせてコンポーネントをカスタマイズします。

パブリック ライブラリを分岐すると、元のライブラリに影響を与えることなく、コンポーネントに変更を加えることができます。これは、既存のコンポーネントの変更や、カスタム プロパティの追加が必要な場合に役立ちます。

## 前提条件

- [電子デザイン ライブラリ マネージャを開く](ECD-OPEN-LIBMGR-TSK.htm)

## 手順

- **電子デザイン ライブラリ マネージャ**で、 **[パブリック]** タブをクリックして使用可能なパブリック ライブラリを表示します。

- 分岐するライブラリを見つけて右クリックします。

- メニューから **[新規ハブ ライブラリとしてコピー]** ![ライブラリの分岐アイコン](../images/icon/ecd/lbr-fork.png) を選択します。

- わかりやすい名前を付けて、新しいハブ ライブラリを保存します。

- ハブへのライブラリのアップロードが完了するまで待ちます。

## 次のステップ

ライブラリの分岐後、次の操作を行うことができます。

- **コンポーネントを使用する:** 配置パネルで新しいハブ ライブラリからコンポーネントにアクセスします。

- **コンポーネントを編集する:** プロジェクトの必要に応じてコンポーネントをカスタマイズします。

- **コンポーネントを追加する:** 分岐されたライブラリで新しいコンポーネントを作成します。
