# CAM プロセッサを実行する

- キャンバスで回路基板を開き、[製造]ツールバーで**[CAM プロセッサ]** ![CAM プロセッサのアイコン](../images/icon/ecd/cam-processor.png) をクリックします。

- 使用するジョブ テンプレートがある場合は、**[ジョブ ファイルをロード]** ![テンプレートをロード アイコン](../images/icon/ecd-2dpcb-load-template.png) をクリックし、ジョブ テンプレートにナビゲートして選択します。

- **[アクティブな CAM ジョブとして設定]**をクリックします。

- 製造出力ファイルの構造と出力タイプを調べます。リスト内のセクション名(OBD++ や Gerber など)をクリックしてセクションを折りたたみます。

- ファイルに問題がなければ、ダイアログの下部にある**[ジョブを処理]**をクリックして、製造ファイルの ZIP アーカイブを作成します。

- アーカイブを保存するフォルダを選択します。既定のパスは、**[基本設定] > [電子部品] > [フォルダ]**で指定します。

ODB++ 出力の場合、既定の接頭辞は CAMOutputs/ODBFiles です。フォルダが作成され、そこに出力が配置されます。Gerber 出力の場合、既定の接頭辞は CAMOutputs/GerberFiles です。
