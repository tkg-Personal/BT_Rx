# モジュールを追加する

- [回路図デザイン]ツールバーで、**[接続] > [モジュール]** ![モジュールのアイコン](../images/icon/ecd/module.png) をクリックします。

- [選択]ダイアログで、[新規]フィールドにわかりやすいモジュール名を入力し、モジュールのラベル付けに使用する短い接頭辞を入力します。モジュールがカーソルにアタッチされます。

- クリックしてモジュールを配置します。<Prefix>-1 というラベルが付けられ、<Name>: 1 というタイトルの新しい回路図シートが生成されます。別のモジュールを配置すると、ラベルとシートのタイトルは同じになりますが、番号は 2 に増分されます。

- モジュールのサイズを変更するには、[Ctrl]を押しながらクリックして選択し、[インスペクタ]でサイズを変更します。
