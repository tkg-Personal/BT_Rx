# レイアウト エディタ

既存の基板ファイルを開くか、新しい基板を作成すると、[レイアウト エディタ]ウィンドウが開きます。

コマンド ツールバーからアクセスできないコマンドの説明は、[回路図エディタ]ウィンドウに関するセクションにも記載されています。メニュー バーのプルダウン メニューからすべてのコマンドにアクセスすることもできます。言うまでもなく、これは[回路図エディタ]ウィンドウと[レイアウト エディタ]ウィンドウにも適用されます。

## [レイアウト コマンド]ツールバーのコマンド

#### INFO

選択したオブジェクトのプロパティを表示します。コマンド ラインに INFO IC1 と入力すると、IC1 という名前のオブジェクトのプロパティ ダイアログが表示されます。選択したオブジェクトに応じて、ここで一部のプロパティを変更することができます。

#### SHOW

マウスで選択するオブジェクトをハイライト表示します。

コマンド ラインにオブジェクト名を入力することもできます(複数同時入力も可能)。*　と　?　もワイルドカードとして使用できます。また、[Ctrl]+[Show]を押すと、選択したオブジェクトの表示状態が切り替わります。

#### DISPLAY

表示する画層を選択および選択解除します。基板の上側のコンポーネントを選択できるのは、画層 23 (OriginsTop)が表示されている場合のみです。基板の下側のコンポーネントと画層 24 (OriginsBottom)についても同様です。

画層の意味については、「付録」を参照してください。

DISPLAY コマンドは、いわゆる「画層プリセット」(エイリアス)をサポートします。これにより、画層の特定の組み合わせに名前を付け、LAYER コマンドでパラメータとして使用することができます。このコマンドを使用すると、1 つのビューから別の画層の組み合わせへの変更を簡単に行うことができます。

色または塗り潰しスタイルを変更するには、いずれかの画層エントリをダブルクリックします。

DISPLAY LAST は、最後に表示された画層の組み合わせに切り替えます。

> **Note:** 
注:

 [DISPLAY]メニューには、デザイン ルールの画層設定で定義された画層のみが表示されます。

DISPLAY の詳細については、ヘルプ機能を参照してください。

#### MARK

次のマウス クリックによって、座標表示の新しい原点が定義されます。座標表示ボックスには、絶対座標に加えて、相対座標(R X 値 Y 値)と極値(P 半径 角度)も表示されます。

最初に[MARK]アイコンをクリックしてから交通信号アイコンをクリックすると、絶対座標値のみが再表示されます。

#### GROUP

グループを定義します。そのグループを、移動、回転、または COPY と PASTE を使用して別の図面にコピーできます。または、そのグループのプロパティを変更します。

既定では、GROUP コマンドは常にアクティブです。図面の何もない領域をクリックしてマウス ボタンを押したままにすると、長方形をドラッグしたり、グループ内のオブジェクトの周囲に銅箔ベタを描画することができます。グループを定義した後、コマンド アイコンを選択しなくても、グループのオブジェクト上でマウス ボタンをクリックしたままにすることで、グループを即座に移動することができます。

オプションの GROUP コマンド(既定でオン)がアクティブになっていない場合、またはグループに対して別のコマンドを実行する場合は、グループを定義し、コマンド アイコン(回転、コピーなど)を選択し、そのグループを[Ctrl] + 右クリックしてコマンドを実行します。

コマンド ラインで GROUP ALL コマンドを使用すると、すべてのオブジェクトが選択されます。

確実にすべてのオブジェクトが選択されるようにするには、前もってすべての画層を表示します。一方、特定の画層を選択解除すると、特定のオブジェクトを選択から除外できます。

GROUP の詳細については、回路図エディタに関するセクションおよびヘルプ機能を参照してください。

#### MOVE

表示されている任意のオブジェクトを移動します。マウスの右ボタンを使用すると、オブジェクトが回転します。

> **Note:** 
注:

 MOVE コマンドで信号を接続することはできません。これは、ワイヤ(トレース)を別のワイヤまたはパッドを越えて移動する場合も該当します。信号を配線するには、ROUTE を使用します。

[Ctrl]キーを押したままオブジェクトを選択すると、特定の方法でオブジェクトが選択されます。詳細については、ヘルプ機能を参照してください(CRICLE、ARC、LINE、MOVE、ROUTE など)。

グループを移動する方法については、「回路図エディタ」セクションで「MOVE」を参照してください。

#### MIRROR

オブジェクトをミラー化します。MIRROR コマンドを使用して、コンポーネントを基板の反対側に配置できます。

#### ROTATE

オブジェクトを回転します(MOVE でも可能)。マウスの左ボタンを押したままにすると、マウスを動かして選択したオブジェクトを回転できます。パラメータ ツールバーに現在の角度が表示されます。グループ(GROUP とマウス右ボタン)を使用して実行することもできます。ROTATE をグループと一緒に使用することもできます。ROTATE コマンドをアクティブにし、[Ctrl]キーを押しながら図面をマウスの右ボタンでクリックして、回転の中心を設定します。グループは、指定した角度だけ反時計回りに回転します。または、[角度]ボックスまたはコマンド ラインに角度を入力します。構文の詳細については、ヘルプ機能を参照してください。

#### ALIGN

ALIGN コマンドを使用すると、選択したオブジェクトを互いに関連付けて位置合わせしたり、最も近いグリッド点に原点の位置を移動することができます。

以下のモードがサポートされます。

- [エッジを位置合わせ] [上] | [下] | [左] | [右]

- [中央揃え] [垂直] | [水平]

- [均等割り付け] [垂直] | [水平]

- [原点をグリッドに位置合わせ] 

#### COPY

パーツやその他のオブジェクトをコピーします。

オブジェクトをコピーする場合、新しい名前が割り当てられますが、値は保持されます。単一ワイヤをコピーすると、コピーの名前は同じになります。

[Ctrl]キーを押したままオブジェクトをクリックすると、オブジェクトがその原点で選択されます。したがって、オブジェクトは現在選択しているグリッドに配置されます。

COPY はグループと一緒に使用できます。グループはオペレーティング システムのクリップボードに入れられます。たとえば、Electronics の別のプログラムにコピーすることが可能です。

#### PASTE

貼り付けバッファからオブジェクトを挿入します。

レイアウト全体(および該当する場合は回路図)を現在の図面に貼り付けるためには、メニュー[編集]/[貼り付け...]を使用します。詳細については、ヘルプを参照してください。

> **Note:** 注: ビアを貼り付ける場合は、**[発側呼び合いを使用]**オプションでソース ネットに貼り付けるか、**[発側呼び合いを使用]**オプションを選択解除して新しいネットに貼り付けることができます。

#### DELETE

表示されているオブジェクトを削除します。

グループが定義されている場合、[Ctrl]キーを押しながらマウスの右ボタンで削除できます。

コマンド ラインに DELETE SIGNALS と入力すると、一致する回路図がロードされていない場合は、レイアウト内のすべてのトラックと信号が削除されます。[Shift]キーを押しながら銅箔ベタのワイヤをクリックすると、DELETE コマンドによって銅箔ベタ全体が削除されます。ワイヤの曲げの上でマウス左ボタンをクリックしながら[Ctrl]キーを押したままにすると、曲げが削除されます。次の曲げとの間に新しい直接接続が描画されます。

オブジェクトを削除できない場合、DRC コマンドに関連したエラー銅箔ベタにその理由がある可能性があります。これらは、ERRORS コマンド(ERRORS CLEAR)を使用して削除することができます。画層 23 (OriginsTop)、または 24 (OriginsBottom)が表示されていない場合、コンポーネントを削除できません。CHANGE オブジェクトのプロパティ(ワイヤの幅や文字のサイズなど)を変更します。プロパティを変更した後に[Esc]キーを押すと、以前に使用した値メニューが再び表示されます。これにより、新しい値を簡単に選択できます。ヘルプ機能も参照してください。

または、オブジェクトのプロパティを表示し、コンテキスト メニューの[プロパティ]エントリを使用して変更することもできます。オブジェクトを右クリックすると、コンテキスト メニューが開きます。

#### PASTE DBL

図面にデザイン ブロックを追加します。デザイン ブロックが基板と回路図で構成されている場合、レイアウト パーツをマウス カーソルで配置できます。それに従って、新しいシートの回路図に自動的に回路図パーツが追加されます。

#### ADD

図面にライブラリ要素を追加します。ここでは、パッケージ用の便利な検索機能が用意されています。USE は、使用可能なライブラリを指定します。

[ADD]アイコンを右クリックすると、最近配置したデバイスのリストが含まれているポップアップ メニューが開きます。

#### PINSWAP

ピンが同じ Swaplevel で定義されている場合、コンポーネントの同等のパッドに接続されている 2 つの信号を交換します。複数のパッドに接続されているピンは交換できません。

#### REPLACE

コンポーネント(回路図がない場合はパッケージ)を、任意のライブラリのコンポーネントで置き換えます。

パッケージ バリアントのみを変更し、デバイス全体は変更しない場合は、CHANGE PACKAGE コマンドまたは PACKAGE コマンドを使用します。

[REPLACE]アイコンを右クリックすると、最近置換したデバイスのリストが表示されているポップアップ メニューが開きます。

#### LOCK

基板上のコンポーネントの位置と方向をロックします。コンポーネントがロックされている場合、CUT および PASTE コマンドで移動または複製することはできません。[Shift] + LOCK でコンポーネントのロックを解除します。これは、コンテキスト メニューの[ロック解除]エントリでも可能です。ロックされたコンポーネントとロック解除されたコンポーネントを区別するために、ロックされたコンポーネントの原点の十字は + ではなく x のように表示されます。

ただし、プロパティ ダイアログで新しい座標値を入力することにより、ロックされたコンポーネントの位置を変更することができます。

#### NAME

コンポーネント、信号、ビア、銅箔ベタに名前を付けます。NAME を使用すると、銅箔ベタを 1 つの信号から別の信号に移動することができます。

#### VALUE

コンポーネントの値を指定します。たとえば、ある抵抗器は値として 100 k を取得します。このアイコンを右クリックすると、既に使用されている値のリストが開きます。エントリを選択し、連続してコンポーネントをクリックすることで 1 つまたは複数のコンポーネントに適用します。

#### SMASH

名前、値、および属性(存在する場合)のテキストをデバイスから分離し、個別に配置できるようにします。アタッチ解除された(分解された)テキストのサイズも個別に変更できます。また、GROUP と組み合わせて使用することもできます。グループが定義されている場合、[Ctrl]キーを押しながら右クリックで分解できます。

分解されたテキストを非表示にするには、DELETE コマンドを使用します。

テキストを分解解除するには、[Shift]キーを押したまま SMASH コマンドを使用します。テキストは編集できなくなり、ウィンドウが更新された後、元の位置に表示されます(コンテキスト メニューの[分解解除]でも可能)。

または、コンテキスト メニューの[プロパティ]エントリの[分解済み]オプションのオン/オフを切り替えます。

#### MITER

ワイヤ ジョイントを丸めるか、ベベルを適用します(銅箔ベタの輪郭でも可能)。マイタの勾配は、マイタ半径によって決まります。正の符号を指定すると丸いジョイントになり、負の符号を指定するとベベルになります。マイタ半径は、一部のワイヤ曲げモードにも影響します(ヘルプ機能で SET、Wire_Bend を参照)。

#### SPLIT

ワイヤに曲げを挿入します。

たとえば、既に配線されているトラックのセクションの画層を変更する場合は、SPLIT コマンドを使用して 2 つのワイヤの曲げを挿入し、新しく作成されたセグメントの画層を CHANGE LAYER コマンドを使用して変更することができます。Electronics は、自動的にワイヤの曲げの位置にビアを設定します。

SPLIT コマンドを使用すると、既存のトラックの再配線をすばやく行うことができます。トラックをクリックして、ワイヤの曲げを挿入します。次に、マウスを移動して新たに配線します。前のトラックを削除するには、RIPUP コマンドを使用するか、[Ctrl]キーと組み合わせて DELETE を使用します。

#### OPTIMIZE

1 本の直線上にある信号画層内のワイヤ セグメントを結合します。

#### MEANDER

信号の長さのバランスを保つためにミアンダ形状を描画します(特に差動ペアの場合)。[Ctrl]キーを押したときに、信号の長さを計測するために使用できます。

#### SLICE

SLICE コマンドは線分を 2 つの部分に切断します。配線済みトレースの場合、ギャップには信号の 2 つの部分を接続するエアワイヤが含まれます。したがって、信号は実際に 2 つの異なる部分に切断されるのではなく、指定されたギャップ幅に従ってトレースがリップアップされます。左クリックで切断ワイヤを開始し、2 回目のクリックで切断ワイヤを終了します。切断ワイヤと交差するすべてのオブジェクトがスライスされます。例外は

銅箔ベタの輪郭です。SLICE コマンドはコマンド ラインで使用できます。切断線は、SLICE (0.2 3) (0.5 4); のように、開始座標と終了座標で定義されます。

#### ROUTE

手動で信号を配線します。エアワイヤがワイヤに変換されます。

既定では、ROUTE コマンドは「障害物を回避」モードで動作します。そのため、自動的にデザイン ルールが考慮され、トレースのパスに沿った障害物が回避されます。パラメータ ツールバーのアイコンをクリックすることで配線モードが「障害物を無視」モードに設定されている場合、ユーザがすべてのデザイン ルールに関して責任を負います。

ROUTE には、異なるマウス ボタンを使用する複数のオプションがあります。また、[Ctrl]キーおよび[Shift]キーと組み合わせることもできます。

#### RIPUP

配線済みのワイヤ(トラック)を未配線の信号(エアワイヤ)に変換し、銅箔ベタの塗り潰しを非表示にします。コマンド ラインで信号名を使用すると、特定の信号のみをリップアップしたり、特定の信号を除外したり、銅箔ベタ専用にコマンドを実行することができます。詳細については、ヘルプ機能を参照してください。コンポーネントに接続されていないワイヤは、DELETE コマンドで削除する必要があります。

#### LINE

線分と円弧を描画します。画層 1 から 16 までで使用した場合、LINE コマンドは電気的接続を作成します。Style パラメータ(CHANGE)により、線種が決定されます。DRC とオートルータは、使用されるスタイルに関係なく、常に線分を連続線として扱います。マウスの右ボタンをクリックすると、ワイヤの曲げが変更されます(SET WIRE_BEND)。

ヘルプ機能での[Ctrl]キーと[Shift]キーとの組み合わせの特徴に注意してください。たとえば、[Ctrl]キーを押しながらワイヤの描画を開始すると、ワイヤは近くにある既存のワイヤの終端で開始されます。このワイヤが現在設定されているグリッド内に存在しない場合でも同様です。ワイヤの幅、スタイル、および画層は、既存のワイヤから採用されます。

#### TEXT

テキストを配置します。テキストの高さを変更するには、CHANGE SIZE を使用します。テキストにベクトル フォントを使用している場合、CHANGE RATIO によって太さが変更されます。CHANGE TEXT は、テキスト自体を変更するために使用します。CHANGE FONT は書体を変更します。CHANGE ALIGN は、テキストの位置合わせ(原点の位置)を定義します。オプション[常にベクトル フォント] ([オプション]/[ユーザ インタフェース])は、実際に特定のテキストに対して設定されているフォントには関係なく、すべてのテキストをベクトル フォントで表示および印刷します。

反転したテキストを銅箔レイヤーに表示する場合は、画層 41(RestrictTop)または画層 42 (RestrictBottom)にテキストを入力し、PPOUR コマンドを使用してテキスト周囲の Top 画層または Bottom 画層に銅箔面を描画する必要があります。銅箔ベタは、制限領域(テキスト)に銅がないように維持します。

複数行のテキストの改行を挿入するには、[Shift]+[Enter]を押します。行間隔は、テキストがまだ配置されておらず、まだマウス カーソルにアタッチされている場合に限り、[プロパティ]ウィンドウまたはパラメータ ツールバーで設定できます。

> **Note:** 
注:

 銅箔レイヤーのテキストはベクトル フォントとして書き込むことを強くお勧めします。そうすると、CAM プロセッサの出力がレイアウト エディタで表示されるテキストと同じであることを確認することができます。ヘルプ機能も参照してください。

#### CIRCLE

円を描画します。画層 41 (RestrictTop)、42 (RestrictBottom)、または 43 (RestrictVias) で使用する場合、このコマンドによってオートルータ用の制限された領域が作成されます。ワイヤ幅が 0 の円は塗り潰しで描画されます。

#### ARC

円弧を描画します(LINE コマンドでも可能)。

CHANGE CAP FLAT | ROUND は、直線または丸みのある円弧の終点を定義します。円弧がトレースの一部であり、両端がワイヤに接続されている場合、キャップは丸くなります。

CAM プロセッサで Gerber 形式で製造データを生成すると、平坦なキャップを持つ円弧がエミュレートされます。つまり、複数の小さくて短い直線で描画されます。丸いキャップを持つ円弧はエミュレートされません。

#### RECT

長方形を描画します。画層 41 (RestrictTop)、42 (RestrictBottom)、または 43 (RestrictVias) で使用する場合、このコマンドによってオートルータ用の制限された領域が作成されます。

#### PPOUR

ソリッドまたはハッチングされた銅箔ベタを描画します。

銅箔ベタは、ルーティング レイヤー上の信号メンバーの銅箔ベタです。DRC ルールに従って、異なる信号のオブジェクトから分離し、同じ信号のオブジェクトに接続することによって、銅箔領域を自動的に塗り潰します。

銅箔ベタの定義輪郭は常に点線で表示されます。ハッチングされた銅箔ベタの場合、その幅と間隔が線のメッシュを定義し、指定した線幅を使用して塗り潰すことができる領域を暗黙的にコントロールします。ソリッド銅箔ベタの場合、その幅は同様に最小ウェブ幅としてソリッド塗り潰しプロセスで使用されます。

#### PCUTOUT

銅箔ベタが領域を銅で塗り潰すことができない銅箔ベタのカットアウト空間を描画します。

銅箔ベタのカットアウトは、同じレイヤーでオーバーラップするすべての銅箔ベタ オブジェクトの塗り潰し空間を定義するために、ルーティング レイヤー上に配置されます。

#### PSHAPE

単純なソリッド銅箔ベタ形状を描画します。

銅箔ベタ形状は、信号のないソリッド形状です。最も一般的な用途は、オートルータが特定の領域(RestrictTop 画層、RestrictBottom 画層、または RestrictVias 画層)を使用しないようにすること、およびフットプリント エディタでデバイスのパッドと SMD の併合形状を定義することです。

#### POLYGON

現在のエディタ コンテキスト、パラメータ設定、およびレイヤーに従って、適切な銅箔ベタ、銅箔ベタ形状、または銅箔ベタのカットアウトを描画します。

#### VIA

めっき済み貫通穴を配置します。ROUTE コマンドの実行中に画層を変更すると、ビアが自動的に配置されます。NAME コマンドで名前を信号の名前に変更することで、ビアを信号に割り当てることができます。ビアは、外側の画層ではさまざまな形状にすることができますが(丸い、正方形、八角形)、内側の画層では常に丸い形状です。

#### SIGNAL

信号の定義。Forward&Back Annotation がアクティブである場合は、この操作は実行できません。この場合、回路図エディタで NET コマンドを使用して接続を定義する必要があります。

#### HOLE

取り付け穴(めっき貫通穴ではない)を定義します。

#### ATTRIBUTE

コンポーネントの属性を定義します。[編集]/[グローバル属性]メニューを使用して、レイアウト全体に対して有効な属性を定義できます。

#### DIMENSION

基板に寸法を追加するために使用できます。オブジェクトに適用することも、任意の寸法を描画することもできます。オブジェクトを選択すると、適切な寸法タイプ(Dtype)が選択されます。必要なタイプではない場合は、マウスの右ボタンをクリックして変更します。図面内の任意の位置から開始する場合は、[Ctrl]キーを押しながらマウスの左ボタンをクリックします。さまざまな寸法タイプがあります(平行、水平、垂直、半径、直径、角度、引出線)。

寸法線、文字サイズの単位などの設定は、オブジェクトのプロパティ ダイアログまたは CHANGE コマンドで行うことができます。このコマンドは、オブジェクトのグループに対して実行することもできます。

#### RATSNEST

最も短いエアワイヤを計算し、銅箔ベタの塗り潰しを再計算して表示します。

特定のエアワイヤを計算して表示または非表示にするには、信号の名前を指定して RATSNEST コマンドを使用します。前に感嘆符を付けると、指定した信号名のエアワイヤが非表示になります。詳細については、ヘルプ機能を参照してください。

銅箔ベタの計算は、SET コマンドを使用して非アクティブにすることができます。[オプション]/[設定]/[その他]を使用するか、コマンド ラインに SET POLYGON_RATSNEST ON | OFF または短縮して SET POLY ON | OFF と入力します。

ROUTE を使用してワイヤを描画している間、選択した信号に対して RATSNEST が自動的に実行されます。

RATSNEST がアクティブである間、レイアウト エディタのステータス バーに、現在計算されている信号の名前が表示されます。

#### AUTO

オートルータを起動します。

コマンド ラインに AUTO FOLLOWME と入力すると、[オートルータのセットアップ]ウィンドウがフォローミー モードで開き、フォローミー ルータのみのパラメータを設定できます。

#### AUTO BGA

BGA オートルータを起動します。

AUTO BGA と���力するか、このアイコンをクリックすると、EAGLE は、BGA コンポーネントに接続された信号を BGA 領域外で配線するために特殊なオートルータを起動します。最初の手順では、レイアウト内の BGA コンポーネントを選択します。次に、配線する信号を選択します。また、信号に対する画層の割り当てを決定することもできます。このオプションが有効になっている場合、マイクロ ビアがサポートされます。BGA ルータを起動する前に、デザイン ルールを確認してください。

#### ERC

回路図と基板の整合性チェックを実行します。

#### DRC

デザイン ルールを定義し、デザイン ルール チェックを実行します。

コマンド ラインに DRC * と入力すると、[デザイン ルール]ウィンドウが開きます。そこで設定を確認および調整し、デザイン ルール チェックを開始せずにダイアログ ウィンドウを閉じることができます。

#### ERRORS

DRC によって検出されたエラーを表示します。基板に対してデザイン ルール チェックをまだ処理していない場合は、エラー リストを表示する(エラーが見つかった場合)前に自動的に処理されます。
