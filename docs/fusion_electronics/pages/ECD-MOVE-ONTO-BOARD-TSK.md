# コンポーネントを 2D PCB に移動する

コンポーネントをボードにドラッグして、ボード上の配置を開始します。

- **[電子デザイン]** 作業スペースの 2D PCB ドキュメントにある **[デザイン]** ツールバーで、 **[修正] > [移動]** ![移動のアイコン](../images/icon/common/move.png) をクリックして **[移動]** パネルを開きます。

- キャンバスで、コンポーネントの中心にあるアンカー(+ 記号)をクリックし、ボードにドラッグします。

- 同じ手順を繰り返して、他のコンポーネントをボードに配置します。

- すべてのコンポーネントを配置したら、 **[完了]** をクリックして **[移動]** パネルを閉じます。
