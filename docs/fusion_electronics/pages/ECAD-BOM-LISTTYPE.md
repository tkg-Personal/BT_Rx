# 電子デザイン部品表のリスト タイプを変更する

異なるリスト タイプに切り替えることで、電子デザイン部品表でのコンポーネントの表示方法を変更できます。

- [電子デザイン]作業スペースで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) を開きます。

- 部品表の左上にある **[リスト タイプ スイッチャ]** をクリックします。

- ドロップダウン メニューから、使用するリスト タイプを選択します。

**値でグループ化**: コンポーネントをデバイス、テクノロジ、パッケージ、および値でグループ化して表示します。

- **コンポーネント**: すべてのコンポーネントを、グループ化せずに平坦なリストで表示します。

部品表が更新され、選択した形式でコンポーネントが表示されます。
