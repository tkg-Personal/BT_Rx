# 線分を描画する

- [回路図ドキュメント]ツールバーで、 **[線分]** ![線分のアイコン](../images/icon/ecd/line.png) をクリックします。

- [線分]ダイアログで、曲げ形状、線幅、スタイル(連続または点線)、半径、またはマイタ形状の設定を確認および調整します。

- クリックして線分を開始し、曲げスタイルに応じて、クリックしてコーナーまたは曲線を作成します。

- ダブルクリックして線分を終了します。
