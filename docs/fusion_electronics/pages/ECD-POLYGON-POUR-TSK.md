# 銅箔ベタ

回路基板のサイズと形状を指定した後、トレースを配線する前に、銅箔ベタを注入することをお勧めします。 

- キャンバスの上部で、ポリゴンベタを注入するレイヤーを選択します。

- [PCB 設計]ツールバーで、**[ポリゴンベタ]** ![ポリゴンベタ アイコン](../images/icon/ecd/polygon-pour.png) をクリックします。

- [ポリゴンベタ]ダイアログで、コーナーと塗り潰しのスタイル、熱、孤立などの設定のオプションを確認または設定します。

- クリックして銅箔ベタの最初の頂点を配置し、カーソルを移動し、クリックして他の頂点を配置します。最後にダブルクリックして銅箔ベタを閉じます。[信号]ダイアログが開きます。

- [信号]ダイアログで、信号の名前を入力します。
