# 指定子を再番号付けする

**[指定子を再番号付け]** を使用して、自動的にデザイン内のコンポーネント参照指定子の再番号付けを行います。これは、コンポーネントの追加、削除、または並べ替え後に、論理的な番号付け順序を維持するのに役立ちます。

指定子の再番号付けは、回路図エディタと 2D PCB エディタの両方で行うことができます。**[指定子を再番号付け]** には、ビューに応じて、どのコンポーネントにどのような順序で再番号付けを行うかを制御する、次のような異なるオプションが示されます。

- 番号付けスキーム

- 範囲オプション

- グループ化設定

**[指定子を再番号付け]** は、コマンド ラインで RENUMBERREFDES と指定して使用することもできます。
