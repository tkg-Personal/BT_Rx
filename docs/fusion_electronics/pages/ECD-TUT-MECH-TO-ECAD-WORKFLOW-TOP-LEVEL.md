# チュートリアル: 機械から電子部品へのワークフロー

現在、多くの製品には回路基板が搭載されており、機械設計および電子デザイン チームが連携する必要があります。連携する領域の 1 つは、PCB 自体のサイズと形状にあります。 

このチュートリアルの内容:

- [デザイン]作業スペースで PCB のサイズを定義します。

- [電子デザイン]作業スペースで PCB をデザインします。

- [デザイン]作業スペースで、PCB のマルチコンポーネント デザイン、および筐体を作成します。
