# 2D PCB 内の指定子を再番号付けする

ボードの調整後に正しい順序でなくなった場合などに、2D PCB コンポーネントの参照指定子を自動的に再番号付けして、論理的で空間的に正しい順序になるようにします。

- 2D PCB デザインのツールバーで、 **[修正] > [指定子を再番号付け]** ![再番号付けのアイコン](../images/icon/renumber.png) をクリックします。

- [指定子を再番号付け]ダイアログで、再番号付けオプションを選択します。

**位置**: 再番号付けするボードの側面( **[上]** ![上のアイコン](../images/icon/renumber-board-top.png)、 **[下]** ![下のアイコン](../images/icon/renumber-board-bottom.png)、または **[両方]** ![両方のアイコン](../images/icon/renumber-board-both.png))を選択します。

- **側面ごとの順序**: [上から下へ]または[ボトムからトップへ]のいずれの順序で再番号付けするかを選択します。

- **開始順序**: どのコーナーからコンポーネントの再番号付けを開始するかを選択します。

**左上** ![左上のアイコン](../images/icon/renumber-upper-left.png): 左上のコーナーから番号付けを開始します。

- **右上** ![右上のアイコン](../images/icon/renumber-upper-right.png): 右上のコーナーから番号付けを開始します。

- **左下** ![左下のアイコン](../images/icon/renumber-lower-left.png): 左下のコーナーから番号付けを開始します。

- **右下** ![右下のアイコン](../images/icon/renumber-lower-right.png): 右下のコーナーから番号付けを開始します。

- **順序方向**: 方向のパターンは、[開始順序]の選択内容によって異なります。

**上**の開始順序([左上]または[右上])の場合は、次のいずれかを選択します。

**横方向から下方向へ** ![横方向から下方向へのアイコン](../images/icon/renumber-upper-left-across-then-down.png): 横方向に行に番号付けをしてから、下にある次の行に移動します。

- **下方向から横方向へ** ![下方向から横方向へのアイコン](../images/icon/renumber-upper-left-down-then-across.png): 下方向に列に番号付けをしてから、横にある次の列に移動します。

**下**の開始順序([左下]または[右下])の場合は、次のいずれかを選択します。

- **横方向から上方向へ** ![横方向から上方向へのアイコン](../images/icon/renumber-lower-left-across-then-up.png): 横方向に行に番号付けをしてから、上にある次の行に移動します。

- **上方向から横方向へ** ![上方向から横方向へのアイコン](../images/icon/renumber-lower-left-up-then-across.png): 上方向に列に番号付けをしてから、横にある次の列に移動します。

- **開始番号**: 順序の開始番号を入力します。

- **ロックされたコンポーネントを除外**: 再番号付けからロックされたコンポーネントを除外します。

- **側面ごとにグループ化**: ボードの側面ごとに番号付けをグループ化します。

- **グループ番号付け**: 100 ごとまたは 1000 ごとにグループ化する場合に選択します。

再番号付けオプションを変更すると、下の表に参照指定子の名前がどのように変更されるかが示されます。

- (省略可能)エラーが発生した場合は、エラー アイコン ![エラー アイコン](../images/icon/error-icon.png) の上にマウス ポインタを置くと、エラーの原因と解決方法が表示されます。

- (省略可能)並べ替え方法を変更するには、コンポーネント リストの列ヘッダをクリックします。既定では、コンポーネントは**現在の名前**でソートされ、PCB 側面でグループ化されます。

> **Note:** 注: 並べ替えに関係なく、**エラー** ![エラー アイコン](../images/icon/error-icon.png)または**警告** ![警告アイコン](../images/icon/warning-icon.png) が付いたコンポーネントは、リストの上部に表示されます。

- (省略可能)**チェック ボックス**を使用して、含めるコンポーネントまたは除外するコンポーネントを選択できます。より簡単にコンポーネントを検索するには、検索バーを使用します。

- **[OK]** をクリックして、再番号付けを適用します。

> **Note:** 注: ライブラリの接頭辞がオーバーライドされると、ライブラリの元の接頭辞に更新されます。元の接頭辞に更新される参照指定子は、 **[警告]** アイコン ![警告アイコン](../images/icon/warning-icon.png) でマークされます。
