# PCB レイアウト内のオブジェクトを反転する

**[反転]** ツールは、フットプリントなどの PCB レイアウト内のオブジェクトをボードの反対側に反転します。

- **[電子デザイン]** 作業スペースの 2D PCB ドキュメントにある **[デザイン]** ツールバーで、 **[修正] > [反転]** ![反転のアイコン](../images/icon/ecd/flip.png) をクリックして **[反転]** パネルを開きます。

- 必要に応じて、 **[反転]** パネルで **[ルート モード]** 設定を調整します。

- キャンバスで、反転するオブジェクトをクリックします。

> **Note:** 注: オブジェクトを再度クリックすると、元のレイヤーと方向に戻ります。

- 必要に応じて、キャンバス内で別のオブジェクトをクリックして反転します。

- 終了したら、 **[完了]** をクリックして変更を確定し、パネルを閉じます。
