# ピンを入れ替える

ピンの入れ替えは、一般に、回路図デザインのレイアウトを改善するために使用されます。呼び合いを入れ替えるには、両方のピンが同じ SwapLevel である必要があります。

ピンを入れ替えると、接続されているネットが移動するため、短絡が生じる可能性があります。
