# 電子デザイン選択フィルタ パネル

この **[選択フィルタ]** パネルはデザイン ペインの右側にあり、特定のオブジェクト タイプを選択することができます。

**[検査]** パネルは、次の作業時に使用できます。

- 回路図かいろず

- 回路基板

- ライブラリらいぶらり

回路基板の場合、選択する画層を指定します。

![選択フィルタ](../images/how-to/selection-filter.png)

上部のフィールドでオブジェクト タイプを選択し、下で画層を選択したら、図面内の選択長方形を使用して目的のオブジェクトを選択します。
