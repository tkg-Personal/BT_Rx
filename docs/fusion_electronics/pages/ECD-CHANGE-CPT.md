# オブジェクトのプロパティ

電子デザインには多くのデータが含まれます。たとえば、コンポーネントには値、参照指定子、複数のフットプリント、属性などがあります。

## ボードの製造元に相談する

PCB を専門家に製造してもらう場合は、以下のパラメータに特定の値を規定しているかどうかを基板の製造元に尋ねます。

- トラック幅

- はんだランドの形状

- はんだランドの直径

- SMD パッドの寸法

- 文字のサイズと太さ

- ドリル穴直径

- 信号画層の数

- 多層基板の場合: ブラインド ビアと埋め込みビアの製造方向と基板の構成

- 異なる電位間のクリアランス値

- はんだストップ マスクとクリーム フレームに関するパラメータ

事前にこの情報を知ることで、時間とコストを節約できます。

Fusion Electronics の **[変更]** ツールを使用すると、デザイン内のオブジェクトのプロパティを修正することができます。コンポーネントの値、ワイヤの幅、ビアのサイズ、文字オブジェクトのサイズ、寸法の単位、ラベルの比率、ネット、コンポーネント、ワイヤ、パッド、ビア、文字オブジェクト、寸法、ラベル、または属性セットのクラスを変更できます。たとえば、次のいずれかを入力します。

- **キャップ**は、ワイヤ円弧の終端の形状を設定します

- **クラス**はネット クラスを参照します

- **表示値**はライブラリに適用されます

- **Dline、Dtype**、および **Dunit** によって、回路基板に適用される寸法が設定されます

- **比率**と**サイズ**はラベル文字に適用されます

- **属性セット**はコンポーネントに適用されます

> **Note:** 注: 値は、オブジェクトの右クリック メニューまたは **[検査]** 固定パネルからも変更できます。
