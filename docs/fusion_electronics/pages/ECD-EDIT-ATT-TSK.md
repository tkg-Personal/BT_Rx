# 属性値を編集する

属性値を編集して、特定のスコープでコンポーネント プロパティを定義またはオーバーライドします。複数のコンポーネントまたはライブラリ全体に値を適用するには、上位のスコープを選択します。既存の既定値をオーバーライドするには、下位スコープを選択します。既定では、編集スコープは、値が最後に設定されたときに使用されたスコープと一致します。

## 前提条件

- [[属性]ダイアログを開きます](ECD-OPEN-ATT-DIALOG-TSK.htm)。

- 目的のコンポーネント バリアントの属性が表示されていることを確認します。

## 手順

- **[属性]ダイアログ**で、属性テーブルのセルをダブルクリックするか、セルを右クリックして **[値を編集]** を選択し、**[値を編集]ダイアログ**を表示します。

- 新しい値のスコープを選択します。

**ライブラリ**: ライブラリ内のすべてのコンポーネントとバリアントに値を適用します。

- **コンポーネント**: 現在のコンポーネントのすべてのバリアントに値を適用します。

- **このバリアント**: 現在のバリアントのすべての属性セットに値を適用します。

- **属性セット**: 現在のバリアントの選択した属性セットに値を適用します。

- 目的の値を入力します。

- [OK]をクリックすると、変更した内容が適用されます。

> **Note:** 注: 下位スコープで設定された値は、上位スコープで設定された値をオーバーライドします。
