# ドキュメントの同期について

同期プロセスは、元の回路図デザインと機能が減らされた PCB バリアントなどの、2 つの電子デザインの違いを詳細に特定するために使用されます。デザイン プロセス全体を通してこの 2 つのデザインが自動的に同期されるため、このプロセスを回路図デザインと関連する PCB を比較するために使用することはできません。

> **Note:** 注: 現時点では、シンクロナイザはモジュールではサポートされていません。
