# 一般ルールを編集する方法

- ルールのツールバーで、 **[ルール] > [デザイン ルール]** をクリックします。

- **[デザイン ルール]** ダイアログで、 **[一般ルール]** をクリックします。[一般ルール]ダイアログには 3 つのパネルがあります。

左側のパネルには、使用可能なすべてのルールが表示されます。

- 右側のパネルには、選択したルールとパラメータの情報が表示されます。

- 一覧からルール カテゴリを選択します。

- **[パラメータ]** パネルに **[最小値]** と **[推奨値]** を入力します。

- **[有効]** をチェックして、ルールをアクティブにします。

**注**:

- パラメータには、最小値よりも大きな優先値を指定する必要があります。

- 既定のルール スコープは、一般ルールでは変更できません。カスタム スコープの場合は、代わりにカスタム ルールを使用します。
