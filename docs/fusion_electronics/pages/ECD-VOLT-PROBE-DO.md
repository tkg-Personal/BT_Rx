# 電圧プローブを追加する

- [回路図デザイン]ツールバーで、 **[電圧プローブ]** ![電圧プローブのアイコン](../images/icon/ecd/voltage-probe.png) をクリックします。[電圧プローブ]ダイアログ ボックスが開き、プローブ ラベルの配置前または配置中にラベル プロパティを変更するためのオプションが表示されます。

- クリックして、回路内の正確な位置にプローブを配置します。

- カーソルを空白の領域にドラッグし、クリックしてラベルを配置します。

- 別のプローブを配置する場合は、その位置をクリックしてラベルを配置します。

- 電圧プローブを削除する場合は、プローブ ラベルの終端をクリックし、右クリックして[削除]を選択します。この削除モードでは、他の電圧プローブをクリックして削除することもできます。
