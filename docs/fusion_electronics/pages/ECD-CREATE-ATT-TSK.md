# 新しい属性を作成する

Fusion 電子デザインでは、ライブラリ、コンポーネント、またはコンポーネント バリアントのプロパティを表す属性を作成できます。属性を作成して値を変更すると、指定したスコープ全体にわたって値が変更されます。  

## 前提条件

- [[属性]ダイアログを開きます](ECD-OPEN-ATT-DIALOG-TSK.htm)。

- 目的のコンポーネント バリアントの属性が表示されていることを確認します。

## 手順

- [属性]ダイアログで、左下隅にある **[新規属性]** をクリックして、**[新規属性]ダイアログ**を表示します。

- 新規属性の名前を入力します。矢印をクリックして内部候補から選択するか、カスタム名を入力します。

- 属性タイプを選択します。

**変数**: 属性値を回路図で上書きできます。

- **定数**: 属性値を上書きできません。

- 属性値のスコープを選択します。

**ライブラリ**: ライブラリ内のすべてのコンポーネントとバリアントに属性を追加します。

- **コンポーネント**: 現在のコンポーネントのすべてのバリアントに属性を追加します。

- **このバリアント**: すべての属性セットに既定値を使用して、属性を現在のバリアントに追加します。

- **属性セット**: 選択した属性セットにのみ値を指定して、属性を現在のバリアントに追加します。

- (省略可能)スコープ全体にわたる属性の既定値を入力します。属性値を既定で空にする場合は、このフィールドを空白のままにします。

- **[OK]** をクリックして属性を作成します。
