# オブジェクトを削除する

- [デザイン]ツールバーで、 **[削除]** ![削除のアイコン](../images/icon/ecd/delete.png) をクリックして、削除操作をアクティブにします。

- オブジェクトを削除するには、一度に 1 つずつクリックします。削除したオブジェクトを復元する場合は、[元に戻す]をクリックします。

- 削除操作を終了するには、他のコマンドをクリックします。
