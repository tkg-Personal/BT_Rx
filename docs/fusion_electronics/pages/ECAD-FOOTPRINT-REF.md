# フットプリント

電子部品の 2D 表現で、回路基板で使用されるものと同じです。関連用語として「コンポーネント」、「パッケージ」、「記号」を参照してください。

- **フットプリント**: 各コンポーネントには、少なくとも 1 つのフットプリントがあります。バリアントが存在する場合、バリアントごとに 1 つのフットプリントがあります。バリアントには、別のピンを割り当てることができます。

**フットプリント**(必須)は、回路基板で使用されている電子部品の 2D リプレゼンテーションです。その寸法は、基板レイアウトを容易にするために空間的に正確です。
