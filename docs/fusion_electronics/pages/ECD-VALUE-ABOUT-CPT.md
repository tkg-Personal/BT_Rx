# コンポーネント値

コンポーネント値は、次のようなコンポーネントの特定の電気的特性を表します。

- 抵抗 (オーム)

- 静電容量(ファラッド)

- インダクタンス(ヘンリー)

他のコンポーネントの場合は、値がパーツ番号やその他の識別情報を表すこともあります。

コンポーネント値とその名前は、回路図上でコンポーネント記号と組み合わさって、コンポーネントの定義を完成させます。この値は、デザイン内のコンポーネントを識別するために、部品表(BOM)やその他のレポートで使用されます。
