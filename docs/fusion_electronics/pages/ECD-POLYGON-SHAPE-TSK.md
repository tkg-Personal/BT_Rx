# 銅線以外の銅箔ベタ形状を作成する

- キャンバスの上部で、ポリゴンベタを作成するレイヤーを選択します。

- [PCB 設計]ツールバーで、 **[ポリゴン] > [形状]** ![ポリゴン形状のアイコン](../images/icon/ecd/polygon.png) をクリックします。

- [ポリゴン形状]ダイアログで、コーナー スタイルなどの設定のオプションを確認または設定します。

- クリックして銅箔ベタの最初の頂点を配置し、カーソルを移動し、クリックして他の頂点を配置します。最後にダブルクリックして銅箔ベタを閉じます。
