# PTH パッド配列を追加する

一連のめっき済み貫通穴(PTH)パッドを集積回路(IC)またはチップ フットプリントに追加します。

- ライブラリが開いた状態で、**[新しいフットプリント]**をクリックし、最初から描画するか、または既存の記号をベースとしてインポートするかを選択します。

- [ライブラリ記号]ツールバーで、 **[配置] > [PTH パッド配列]** ![PTH 配列のアイコン](../images/icon/ecd/pth-array.png) をクリックします。

- **[PTH パッド配列]**ダイアログで、ベース名(通常は単一の文字)を指定します。ハイフンはあってもなくてもかまいません。

- チップのどちら側にピンを配置するかを選択します。

- 使用するパッド形状を選択します。

- ピン数と開始インデックス(通常は 1)を指定します。

- チップの長方形のアウトラインを描画するかどうか、および >NAME ラベルと >VALUE ラベルを含めるかどうかを選択します。

- [パッド間隔]フィールドで、隣接するパッド間の X 距離と Y 距離を指定します。

- **[OK]**をクリックして、キャンバスにチップを配置します。
