# ポートを追加する

各ポートは、サブデザイン モジュールから上位デザインにネットをエクスポートし、ネットの接続点を定義します。モジュールは、個別のシートに詳細なサブデザインを作成することで階層回路図を整理する場合に役立ちます。モジュールにポートが追加されると、自動的にそのポートにラベルが付けられ、モジュールの各インスタンスに追加されます。
