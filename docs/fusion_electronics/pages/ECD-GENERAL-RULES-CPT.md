# 一般的なデザイン ルール

Fusion では、業界共通標準に基づいた一般的なデザイン ルールを提供します。各ルールの**[最小値]**を**[デザイン ルール]**ダイアログで直接変更し、右側の詳細ビュー パネルで推奨値を調整することができます。

Fusion ではこのデザイン ルールの推奨値を、ユーザー固有の銅箔クリアランス、トレース幅、および VIA ドリル サイズを対象とする次のインタラクティブなワークフローに使用します:

- 銅箔ベタ

- 手動配線

- マルチ配線

- 差動ペアの配線

> **Note:** 注:

- 推奨値は最小値より小さくできません。

- 一般ルールの優先度は最も低く、Fusion では最後に評価されます。一般ルールの**推奨値**は、カテゴリで最小にする必要があります。銅箔クリアランス、トレース幅、または VIA ドリル サイズの場合、Fusion では、適用可能な最適ルールの推奨値と一般ルールの値のうち、大きい方の値が使用されます。
