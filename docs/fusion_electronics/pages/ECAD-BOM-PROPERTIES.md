# 電子デザイン部品表の属性を編集する

一部の属性は、部品表で直接変更することができます。これにより、BOM ビューを離れることなくコンポーネント情報を更新できます。

- **編集可能な属性:** 製造元、製造元のパーツ番号、ベンダー、ベンダーのパーツ番号

- **要件:** 属性はロックすることも、読み取り専用にすることもできません。

## 列マッピング

編集可能な属性は、コンポーネント属性に対して読み取りと書き込みを行います。

- **製造元:** MANUFACTURER

- **製造元のパーツ番号:** MPN

- **ベンダー:** VENDOR

- **ベンダーのパーツ番号:** VPN

## 部品表のセルを編集する

- [電子デザイン]作業スペースで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) を開きます。

- 部品表で、変更するセルをダブルクリックします。

- 変更内容を入力し、 **[Enter]** を押します。

> **Note:** **ヒント:** 部品表または開いているドキュメントから属性を編集できます。Fusion は、両方のビューを同時に更新します。
