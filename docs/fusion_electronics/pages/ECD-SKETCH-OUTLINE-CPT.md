# アウトライン スケッチ

すべての 3D PCB には、PCB のボード アウトラインを編集するために使用される **[アウトライン]** という名前のスケッチが含まれています。これは、 **[ブラウザ]** の **[スケッチ]** フォルダと、 **[タイムライン]** の先頭に位置しています。編集は、これらの場所、またはツールバーから開始できます。

- 2D PCB で**ボード アウトライン レイヤー**を変更すると、**アウトライン スケッチ**がこれらの変更に合わせて更新されます。

- **アウトライン スケッチ**で変更を加えて 2D PCB に同期すると、2D PCB の**ボード アウトライン レイヤー**がこれらの変更に合わせて更新されます。**アウトライン スケッチ**に拘束または投影を追加した場合、2D PCB の**ボード アウトライン レイヤー**は読み取り専用になります。  

## ヒント

- 新しくリンクされた PCB でアウトライン スケッチを初めて編集する場合、スケッチ平面を選択するよう求められます。

- 3D PCB では**アウトライン スケッチ**のみを使用してボード アウトラインを描画し、拘束や投影などの優れた機能を活用することができます。
