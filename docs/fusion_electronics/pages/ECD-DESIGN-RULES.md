# デザイン ルール

Fusion では、**デザイン ルール**を使用してデザインの要件を定義します。これらのルールにより、Fusion が従うべき手順セットが作成され、PCB は設計基準を満たします。PCB ボードはこれらのデザイン ルールに照らしてチェックされますが、長方形、円、テキストなどの電気的に無関係なオブジェクトは、**信号層**上にない限り、クリアランス エラーに関して無視されます。ボードがデザイン ルールに準拠していることを確認するには、ボードのデザイン後に **デザイン ルール チェック**(DRC)ルーチンを実行します。これにより、エラーが特定されます。

Fusion ではデザイン ルールをその優先度に基づいて適用し、手動配線などまず最も優先されるタスクをチェックします。これにはクリアランス、銅箔幅、または VIA ドリル サイズのチェックが含まれます。ルールの優先度を設定して、まず最も厳格なスコープをチェックし、次にそれほど厳格ではないスコープをチェックします。

定義された一連のデザイン ルールを使用して、厳しい製造要件を満たす PCB 設計を作成します。設計プロセスの開始時にこれらのルールを設定すれば、製造可能性を気にすることなく設計に集中できます。必要に応じて、**デザイン ルール**のルールを調整します。カスタマイズしたルールを新しい名前で新しい Fusion **電子デザイン ルール(EDRU)**ファイルとして保存できます。

![[デザイン ルール]ダイアログ](../images/how-to/design-rules-dialog.png)

**[元に戻す/やり直し]** 機能は、デザイン ルールに対する変更を取り込みます。

## インタラクティブ ワークフローでルール スコープを設計する

次のインタラクティブ ワークフローでは、カスタム ルールを含むすべてのデザイン ルールが使用されます。

**銅の注入**:

- クリアランス計算: 適用可能な最も適したクリアランス ルールの推奨値を使用します。

**手動配線**:

- クリアランス計算: 適用可能な最も適したクリアランス ルールの推奨値を使用します。

- 銅箔幅: 適用可能な最も適した銅箔幅ルールの推奨値を使用します。

- VIA ドリル サイズ: 適用可能な最も適切なドリル サイズ ルールの推奨値を使用します。

**マルチ配線**:

- クリアランス計算: 適用可能な最も適したクリアランス ルールの推奨値を使用します。

**差動ペアの配線**:

- クリアランス計算: 適用可能な最も適したクリアランス ルールの推奨値を使用します。

**注**: 適用可能な最も適したルールとは、オブジェクトが**オブジェクト 1** (すべてのルール)のスコープと**オブジェクト 2** のスコープ(バイナリ ルールのみ)に一致する最初のルールです。同じ信号の銅箔クリアランス ルールの場合、**両方のオブジェクト**のスコープも一致する必要があります。

オートコンプリートを使用する場合、**クイック配線**、**マルチ配線**、**差動ペア配線**および**ミアンダ**は、次の信号ベースのルール スコープのみを使用できます。

- In Net Classes

- In Signal

- Is Signal

- In Diff Pair

- In Named Group
