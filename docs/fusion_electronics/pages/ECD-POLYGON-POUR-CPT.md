# ポリゴンベタ

ポリゴンを使用して信号の銅箔シートまたは銅箔レイヤーを作成します。

**[塗り潰しスタイル]**は通常は[ソリッド]ですが、低容量の塗り潰しには[ハッチング]スタイルを使用できます。

熱伝導率を低減しながらパッドとポリゴンベタ間の電気接続を提供するために、既定では**[熱]**がオンになっています。その結果、ポリゴンベタに対する熱を過度に失わずに、確実に接続のはんだ付けを行うことができます。

**[孤立]**は、デザイン ルールやネット クラスで指定された他の信号からの特定の距離がポリゴンベタによって自動的に維持される場合に作成できる小さいポリゴンベタです。これらの信号は、その信号との接続がない場合は既定でドロップ(削除)されますが、そのままにすることもできます。

**[選択表示]**値は、ポリゴンベタ領域と他の信号またはオブジェクトとの間の距離です。既定値 0 をお勧めします。
