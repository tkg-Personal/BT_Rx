# 電子デザイン部品表のバリアントを変更する

電子デザイン部品表に取り込むアセンブリ バリアントを変更するには、 **[バリアント スイッチャ]** から別のバリアントを選択します。

- [電子デザイン]作業スペースで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) を開きます。

- 部品表の左上にある **[バリアント スイッチャ]** をクリックします。

- ドロップダウン メニューから、表示するアセンブリ バリアントを選択します。

部品表が更新され、選択したアセンブリ バリアントのコンポーネントが表示されます。

> **Note:** 
注:

**** 部品表でアセンブリ バリアントを変更しても、キャンバスでアクティブなアセンブリ バリアントは変更されません。
