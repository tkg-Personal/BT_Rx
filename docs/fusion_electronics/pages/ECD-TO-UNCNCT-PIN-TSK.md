# 電子デザイン回路図でピンを未接続として示す

- [デザイン]ツールバーで、 **[接続] > [ネット]** ![ネットのアイコン](../images/icon/ecd/net.png) をクリックして、 **[ネット]** パネルを開きます。

> **Note:** 注: 既定では、 **[接続]** が選択されています。

- [ネット]パネルで、 **[接続なし]** をクリックします。

配置するための十字マークがカーソルに表示されます。

- 接続しないピンの上にマークを配置するには、目的のピンをクリックします。

ピンの上に **[接続なし]** マークが表示されます。

- ピンのマーキングが完了したら、 **[完了]** をクリックして **[ネット]** パネルを閉じます。

## ピンを一括して未接続としてマークする

- **[デザイン マネージャ]** 固定パネルの **[項目]** テーブルで、1 つまたは複数のピンを選択します。  

- 右クリックして、 **[接続を配置しない]** マークを選択します。
