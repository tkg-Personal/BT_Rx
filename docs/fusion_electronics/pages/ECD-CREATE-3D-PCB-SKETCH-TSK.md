# 3D PCB でスケッチを作成する

3D PCB ドキュメントに機械設計から 3D ジオメトリを参照するには、この機能を使用します。

- **[3D PCB]** ツールバーで、 **[PCB スケッチを作成]** ![3D スケッチを作成のアイコン](../images/icon/ecd/sketch-3d-pcb.png) をクリックします。

- キャンバスで、上部レイヤーまたは下部レイヤーをクリックします。

スケッチがブラウザのボード ノードに追加されます。

![3D PCB スケッチ](../images/how-to/ecd-create-sketch-3d-pcb.png)

- スケッチ ブラウザ ノードを右クリックして、次のようなタスクを実行します。

スケッチの編集(ダブルクリックして編集することもできます)

- スケッチの削除

> **Note:** 注: 3D PCB スケッチでは、構築ジオメトリのみが使用されます。

- スケッチ ツールを使用して、ジオメトリを作成したり、コンテンツを挿入したりします。

![3D PCB スケッチ円](../images/how-to/ecd-3d-pcb-sketch-circle.png)

- 作成または編集が終了したら、 **[スケッチを終了]** をクリックします。
