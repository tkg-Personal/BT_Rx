# モジュールにポートを追加する

- [回路図デザイン]ツールバーで、**[ポート]** ![ポートのアイコン](../images/icon/ecd/port.png) をクリックします。

- [ポート]ダイアログで、ポートの方向を、io (既定)、nc、in、out、oc、pwr、pas、または hiz から選択します。

- モジュール上の位置をクリックしてポートを配置します。

- [選択]ダイアログの[新規]フィールドで、ポートに短い名前を割り当てます。ポートがモジュールに追加されます。
