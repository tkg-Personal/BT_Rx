# テキストを追加する

![テキストを追加する](../images/how-to/ecd-sch-add-text-access.png)

- (1) [回路図ドキュメント] > (2) [ドキュメント]タブ > (3) [描画]パネルで、 **[文字]** ![文字のアイコン](../images/icon/ecd/text.png) をクリックします。

- ダイアログ ボックスで、文字サイズとフォントの設定を確認し、調整します。

- ボックスにテキスト文字列を入力します。値を変更したら、[**Enter**]を押します。

- デザイン内で、クリックしてテキスト文字列のインスタンスを配置します。右クリックしてテキストを 90 度回転したり、ダイアログ ボックスで[ミラー]をクリックしてテキスト ハンドルを文字列の一方の端からもう一方の端に移動することができることに注意してください。

- ダイアログで、[OK]をクリックしてコマンドを終了します。
