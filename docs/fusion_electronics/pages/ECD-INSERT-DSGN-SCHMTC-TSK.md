# 回路図に電子デザインを挿入する

以下は、デザインを挿入する場合に推奨される手順です。

- 回路図デザイン ツールバーで、 **[配置] > [電子デザインを挿入]** ![回路図を挿入のアイコン](../images/icon/ecd/insert-elec-design.png) を選択します。

- [参照]ダイアログで、挿入するデザインに移動して選択します。回路図デザインがカーソルにアタッチされます。  

- キャンバスで回路図を位置決めし、クリックして配置します。

インポートした回路図デザイン シートが、現在の回路図ドキュメントに追加されます。
