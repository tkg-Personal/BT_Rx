# 回路図シート

大規模な回路図デザインで作業する場合、プロセスを簡素化するためにシートを追加することができます。異なるシートを使用して、回路図の異なる部分を設計することができます。

![[シート]ウィンドウ](../images/concepts/sheets-window.png)

**重要**: 自動外部参照ラベルを作成して、ネット上の信号が別のシートのネットにリンクする場所を表示する場合、両方のシートには、ラベル位置をマッピングするためのブロックまたは境界システムを含むフレームが必要になります。
