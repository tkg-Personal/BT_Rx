# その他のポリゴン コマンド

**[ポリゴンのカットアウト]:** コンポーネントを配置するために、注入されたポリゴンベタの内側に空き領域を作成します。

**[ポリゴン形状]:** ネット名を持たないポリゴンを作成します。ロゴやドキュメントなどの銅線以外のグラフィック項目に使用します。

**[アウトラインからポリゴンベタ]:** ボードのアウトラインなどの閉じた図形を選択し、それをポリゴンベタに変換します。

**[すべてのポリゴンベタを非表示]/[すべてのポリゴンベタの塗り潰しを表示]:** ポリゴンベタ内部の塗り潰しをオフにしてアウトラインのみを表示するか、塗り潰しをオンに戻します。これらのコマンドは、注入された電気ポリゴンベタにのみ影響するため、ボード上のどのポリゴンベタが注入されていて、どのポリゴンベタがグラフィック形状のみであるかをすばやく確認できます。

**[ポリゴンベタを再塗り潰し]:** 1 つまたは複数のポリゴンベタを再実行して、移動するコンポーネントからの損傷を修復します。
