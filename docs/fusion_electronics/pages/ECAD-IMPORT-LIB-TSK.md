# 完全な電子デザイン ライブラリをインポートする

Fusion Electronics のカスタム ライブラリと EAGLE ライブラリをプロジェクトにインポートすることができます。

- Fusion ウィンドウの左上隅で、 **[データ パネルを表示]** ![データ パネルのアイコン](../images/how-to/data-panel-icon.png) をクリックしてデータ パネルを開きます。

- ライブラリをインポートするプロジェクトを選択し、ダブルクリックして開きます。

- データ パネルの上部付近にある **[アップロード]** をクリックして、 **[アップロード]** ダイアログを開きます。

- **[アップロード]** ダイアログで、 **[ファイルを選択]** をクリックし、インポートするライブラリを保存した場所を参照します。

> **Note:** 注: ファイルを **[アップロード]** ダイアログにドラッグ アンド ドロップすることもできます。

- 適切なライブラリを選択し、 **[開く]** をクリックして **[アップロード]** ダイアログに戻ります。

- **[アップロード]** ダイアログで、インポート先の場所を確認し、別のプロジェクトまたはフォルダを選択する場合は、 **[場所を変更]** をクリックします。

- **[アップロード]** をクリックしてライブラリをインポートし、ダイアログを閉じます。
