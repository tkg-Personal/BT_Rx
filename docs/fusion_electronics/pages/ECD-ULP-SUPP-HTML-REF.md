# サポートされている HTML タグ

EAGLE では、HTML ページの形式設定に使用されるタグのサブセットがサポートされています。このサブセットを使用して、ユーザ言語ダイアログのいくつかのオブジェクトのテキストを形式設定できます(#usage ディレクティブ、またはライブラリ オブジェクトの説明で)。

最初の行にタグが含まれている場合、テキストは HTML とみなされます。これに該当しない場合にテキストを形式設定する場合は、<html>...</html> タグでテキスト全体を囲む必要があります。

次の表に、サポートされているすべての HTML タグ、およびそれらで使用可能な属性を示します。

| 
タグ | 
説明 | 
 |

| 
<html>...</html> | 
HTML ドキュメント。 | 
 |

| 
<body>...</body> | 
HTML ドキュメントの本文。ここでは、属性 bgcolor (背景色)が認識されます(例: bgcolor="yellow"、bgcolor="#0000FF")。この属性は、dlgTextView 内でのみ機能します。 | 
 |

| 
<h1>...</h1> | 
最上位の見出し。 | 
 |

| 
<h2>...</h2> | 
次のレベルの見出し。 | 
 |

| 
<h3>...</h3> | 
次の次のレベルの見出し。 | 
 |

| 
<p>...</p> | 
左揃えの段落。align 属性を使用して位置合わせを調整します。使用可能な値は、left、right、および center です。 | 
 |

| 
<center>...</center> | 
中心揃えの段落。 | 
 |

| 
<blockquote>...</blockquote> | 
インデントされた段落で、引用に便利です。 | 
 |

| 
<ul>...</ul> | 
順序付けされていないリスト。type 引数を渡して、箇条書きスタイルを定義することもできます。既定は type=disc で、その他のタイプは circle と square です。 | 
 |

| 
<ol>...</ol> | 
順序付きリスト。type 引数を渡して、列挙ラベル スタイルを定義することもできます。既定は type="1" で、その他のタイプは "a" と "A" です。 | 
 |

| 
<li>...</li> | 
リスト項目。このタグは、ol または ul のコンテキスト内でのみ使用できます。 | 
 |

| 
<pre>...</pre> | 
大きいコード チャンクの場合。コンテンツ内の空白は保持されます。小さいコードの場合は、インラインスタイルのコードを使用します。 | 
 |

| 
<a>...</a> | 
アンカーまたはリンク。次の属性が認識されます。 - **href** - <a href="target.html">...</a> のようにターゲットを参照します。指定したターゲット ドキュメント内の追加のアンカーを指定することもできます(例: <a href="target.html#123">...</a>)。名前に空白が含まれるローカル ファイルにリンクする場合は、<a href="file:/path with blanks/target.html">...</a> のように、ファイル名の先頭に file: を付加する必要があります。- **name** - アンカー名。<a name="123">...</a> のように使用します。 | 
 |

| 
<em>...</em> | 
強調(<i>...</i> と同じ)。 | 
 |

| 
<strong>...</strong> | 
強い強調(<b>...</b> と同じ)。 | 
 |

| 
<i>...</i> | 
斜体のフォント スタイル。 | 
 |

| 
<b>...</b> | 
太字のフォント スタイル。 | 
 |

| 
<u>...</u> | 
下線付きのフォント スタイル。 | 
 |

| 
<big>...</big> | 
大きいフォント サイズ。 | 
 |

| 
<small>...</small> | 
小さいフォント サイズ。 | 
 |

| 
<code>...</code> | 
コードを示します(<tt>...</tt> と同じ)。大きいコード チャンクの場合、ブロックタグ pre を使用します。 | 
 |

| 
<tt>...</tt> | 
タイプライタのフォント スタイル。 | 
 |

| 
<font>...</font> | 
フォント サイズ、ファミリ、文字色をカスタマイズします。このタグは次の属性を認識します。 - **color** - 文字色(例: color="red"、color="#FF0000")。- **size** - フォントの論理サイズ。論理サイズ 1 ～ 7 がサポートされています。値は、絶対値(例: size=3)、または相対値(例: size=-2)のいずれかです。後者の場合、サイズは単に加算されます。 - **face** - フォントのファミリ。たとえば、face=times などです。 | 
 |

| 
<img...> | 
イメージ。このタグは次の属性を認識します。 - **src** - イメージ名(例: <img src="image.png">)。イメージの URL は外部の場合もあります(例: <img src="http://web.cadsoft.de/cslogo.gif">)。- **width** - イメージの幅。イメージが指定したサイズに収まらない場合は、自動的に尺度変更されます。 - **height** - イメージの高さ。 - **align** - イメージの配置場所を決定します。既定では、通常の文字と同様に、画像は一列に配置されます。画像をどちらかの側に配置するには、left または right を指定します。 | 
 |

| 
<hr> | 
水平線。 | 
 |

| 
<br> | 
改行。 | 
 |

| 
<nobr>...</nobr> | 
改行なし。ワード ラップを禁止します。 | 
 |

| 
<table>...</table> | 
表定義。既定の表はフレームなしです。フレームを取得するには、ブール属性 border を指定します。その他の属性は次のとおりです: - **bgcolor** - 背景色。 - **width** - 表の幅。これは、ピクセル単位の絶対値または列の幅に対するパーセント単位の値です(例: width=80%)。 - **border** - 表の罫線の幅です。既定値は 0 (罫線なし)です。 - **cellspacing** - 表のセルの周囲にスペースを追加します。既定値は 2 です。- **cellpadding** - 表のセルの内容の周囲にスペースを追加します。既定値は 1 です。 | 
 |

| 
<tr>...</tr> | 
表の行。table 内でのみ使用できます。属性 bgcolor (背景色)が認識されます。 | 
 |

| 
<td>...</td> | 
表のデータ セル。tr 内でのみ使用できます。次の属性が認識されます - **bgcolor** (背景色)。 - **width** - セルの幅。これは、ピクセル単位の絶対値、または表全体の幅に対する相対値(例: width=50%)です。 - **colspan** - このセルが連結する列数を定義します。既定値は 1 です。**rowspan** - このセルが連結する行数を定義します。既定値は 1 です。- **align** - 位置合わせ。使用可能な値は、left、right、および center です。既定は左揃えです。 | 
 |

| 
<th>...</th> | 
表のヘッダ セル。td と似ていますが、既定では中心揃えで太字です。 | 
 |

| 
<author>...</author> | 
このテキストの作成者を示します。 | 
 |

| 
<dl>...</dl> | 
定義リスト。 | 
 |

| 
<dt>...</dt> | 
定義タグ。dl 内でのみ使用できます。 | 
 |

| 
<dd>...</dd> | 
定義データ。dl 内でのみ使用できます。 | 
 |
