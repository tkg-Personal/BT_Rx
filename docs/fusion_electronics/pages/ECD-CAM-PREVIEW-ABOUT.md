# CAM プレビューについて

CAM プレビューには、アクティブなボード デザインの現在またはライブ状態を表す 3 つのタブがあります。

![CAM プレビュー UI](../images/concepts/ecd-cam-preview.gif)

- **[プレビュー]**タブを使用してボードの外観を確認し、必要に応じてボードのイメージをエクスポートします。

タブのすぐ下にある[表示]ツールには、ボードのさまざまな部分の色を設定するオプションとともに、**[ズームしてフィット]**、**[ズームして拡大]**、**[ズームして縮小]**、**[プレビュー設定]**などのオプションがあります。

![プレビュー設定](../images/concepts/ecd-cam-prvw-color-stgs.png)

- ビューのドロップダウン リストを使用して、プレビューをファイルします。

**上面ビュー**

- **下側**

- **ドリル**は、NPTH、PTH、およびビアホールのボード上のすべての穴の位置を表示します。

- **DXF をエクスポートすると、現在のボードが DXF 形式で保存されます。DXF ファイルを開くと、レイヤ構造が保持されていることが分かります。

![DXF エクスポート レイヤ](../images/concepts/ecd-cam-prvw-dxf-exp.gif)

- **イメージをエクスポートすると、現在のプレビューが PNG または JPG 形式で保存されます。

- **CAM...** [CAM プレビュー]ダイアログを閉じ、CAM プロセッサを開きます。

- **[ボード]**タブを使用して、ボード サイズ、レイヤー スタックアップ、コンポーネント、パッド、ドリル、およびルーティングの詳細を確認します。

![CAM プレビュー ボード タブ](../images/concepts/ecd-cam-prvw-brd-tab.png)

- **[ドリル]**タブを使用して、ドリルのタイプ、数、およびサイズの詳細を確認します。

各行を選択すると、プレビューで穴がハイライト表示されます。

![CAM プレビュー ドリル タブ](../images/concepts/ecd-cam-prvw-drill-tab.png)

- **[CSV]** と **[JSON]** は、ドリル テーブルをそれぞれのファイル形式にエクスポートします。

- **[類似ドリル サイズの結合]** は、許容差の値を使用して、その許容差の範囲と単位タイプ内のすべての穴を結合します。

- **[グループ]**は、タイプ ボックスが選択された場合に、プレビューに穴タイプを表示します。既定では、すべてのタイプが選択されています。
