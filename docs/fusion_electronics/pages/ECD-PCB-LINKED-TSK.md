# リンクされた PCB を作成する

**[リンクされた PCB を作成]** ツールを使用して、外部コンポーネントとして PCB を作成し、アクティブなデザインに挿入します。

- **[デザイン]** 作業スペースで、 **[作成] > [PCB を作成] > [リンクされた PCB を作成]** を選択します。

 **[リンクされた PCB を作成]** ダイアログが表示されます。

- **[リンクされた PCB を作成]** ダイアログで、PCB に一意の**名前**を入力します。

- 場所: 新しい PCB がここに保存されます。現在のデザインが保存されているプロジェクト フォルダが自動的に選択されます。Fusion は、新しいデザインを自動的に親と同じ場所に保存します。場所を変更するには:

 a.フォルダ名または[X]をクリックします。

 **[フォルダを選択]** ダイアログが表示されます。

 b.**[フォルダを選択]** ダイアログで、デザインを保存する新しい場所にナビゲートします。

 c.**[選択]** をクリックします。  

- 省略可能: 既定と異なる場合は、**親**を選択します。

- 省略可能: 新しいコンポーネントを作成した後にアクティブにしない場合は、 **[アクティブ化]** ボックスをオフにします。

- [OK]をクリックします。

> **Note:** **注:** リンクされた PCB が新しく作成され、インプレイス編集で開きます。ボード ボディを作成するには、 **[アウトライン スケッチ]** で閉じたプロファイルを用意し、それを **[ボードを定義]** ツールで選択する必要があります。
