# PCB に電子デザインを挿入する

- [2D PCB 設計]ツールバーで、 **[配置] > [電子デザインを挿入]** ![回路図を挿入のアイコン](../images/icon/ecd/insert-elec-design.png) を選択します。

- [参照]ダイアログで、挿入するデザインに移動して選択します。PCB デザインがカーソルにアタッチされます。  

- キャンバスでデザインを位置決めし、クリックして配置します。インポートした基板デザインが現在の PCB ドキュメントに追加されます。

2D PCB に挿入するデザインに関する注意事項

- 関連付けられた回路図をアクティブなデザインの新しいシートとして追加します。

- 既存のボード内に配置すると、ポリゴン カットになる場合があります。不要な場合は、適切なボードのプロファイルを修正または削除します。
