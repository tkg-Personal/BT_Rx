# ボードのアウトライン

ボードのアウトラインは、ボード形状を拘束する閉じた図形です。これにより、コンポーネントの配置と配線接続に使用できる領域が決まります。新しい 2D PCB を作成する場合、既定では長方形のアウトラインが使用されます。ボードのアウトラインは、PCB の外形寸法を定義します。PCB を初めて作成すると、既定のボード アウトラインが作成されます。必要に応じて、2D PCB スペースまたは 3D PCB スペースに異なるボードのアウトラインを描画することができます。

## 2D PCB アウトライン

Fusion Electronics は、適切に描画された輪郭(単一の閉じたアウトライン、自己交差なし)を検出し、残りの背景とは異なる背景色でボード領域を表示します。このジオメトリは、[20 - ボード アウトライン]レイヤーに描画されます。また、ボードのアウトラインには、[46 - フライス加工レイヤー]に描画された取り付け穴とフライス加工輪郭も含まれます。

### ヒント

- 既存のボード アウトラインを削除しても、コンポーネントとエアワイヤは元の位置に残りますが、灰色のボード サーフェスが作図キャンバス全体に表示されます。新しいボード形状に閉じた図形を作成すると、灰色のボード サーフェスはボード形状の内側に拘束されます。

- 古いボードよりもかなり大きい新しいボード形状を作成する場合は、古いボードを完全に囲む新しい形状を作成してから、古いボードを削除することができます。

## 3D PCB アウトライン

3D PCB のボード アウトラインは、アウトライン スケッチ内に含まれています。

### ヒント

- **[プロジェクト]** ツールを使用してアセンブリ コンテキストからジオメトリを参照する場合、ボード アウトラインは機械的な筐体からの更新を反映することができます。

- **[ボードを定義]** ツールを使用してアウトライン スケッチにあるもの以外のプロファイルを選択した場合、アウトライン スケッチは 2D PCB ボードのアウトライン定義に使用されません。

- 3D 基板の形状を修正し、その変更を 2D PCB にプッシュし直した場合、それ以降、基板の形状は 3D 基板のアウトライン スケッチを介してのみ修正できます。
