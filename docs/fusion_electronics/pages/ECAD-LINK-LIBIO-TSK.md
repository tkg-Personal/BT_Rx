# 既存の Library.io ライブラリにリンクする

- チーム ライブラリを開いた状態で、**[管理] > [library.io の管理ライブラリにリンク]** ![リンクのアイコン](../images/how-to/link-io-icon-16.png) をクリックします。[管理ライブラリを選択]ダイアログが開き、使用可能な管理ライブラリが表示されます。

- 適切なライブラリを選択して、**[OK]**をクリックします。

- チーム ライブラリを管理ライブラリ内のすべてと置き換える場合を除き、[更新]ダイアログで**[最新バージョンを統合]**をクリックします。

![[更新]ダイアログ](../images/how-to/library-update-db.png)

- 管理ライブラリ コンポーネントがローカル ライブラリに追加されると、重複項目の処理方法を決定するようにプロンプトが表示されることがあります。

![記号オプション](../images/how-to/library-update-symbol-options.png)

**コンテンツ マネージャ**のライブラリのタイトルの横に、2 つのアイコンが表示されます。これは、ライブラリがチーム ライブラリ ![チームのアイコン](../images/how-to/team-icon-16.png) とリンク ライブラリ ![リンクのアイコン](../images/how-to/link-icon-16.png) の両方であることを示します。
