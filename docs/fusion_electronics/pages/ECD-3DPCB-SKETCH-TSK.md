# スケッチ プロファイルを使用して 3D PCB ボディを定義する

**[3D PCB]** のボディの定義に使用するスケッチ プロファイルを選択し、Fusion の**2D PCB キャンバス**で使用する座標系を定義します。

- **[3D PCB]** 作業スペースで、 **[作成] > [ボードを定義]** を選択します。

 **[ボードを定義]** ダイアログが表示されます。

- プロファイルを選択します。これは通常、**アウトライン スケッチ**のプロファイルですが、アセンブリ コンテキストからスケッチ プロファイルを参照することもできます。

- スケッチ プロファイルから点をクリックして、原点を選択します。

- スケッチ プロファイルからスケッチ線分またはエッジをクリックして、**X** 軸を選択します。

- **[X 軸を反転]** をクリックして、**X** 軸の方向を反転します。

- **[Z 軸を反転]** をクリックして、**Z** 軸の方向を反転します。

> **Note:** 注: 

- 新しく作成した **[リンクされた PCB]** でボード ボディを作成するには、 **[ボードを定義]** ツールを使用して閉じたプロファイルを選択する必要があります。

- **Y** 軸は直接選択されず、**X** 軸および **Z** 軸から推測されます。
