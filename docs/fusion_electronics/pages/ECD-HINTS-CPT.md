# 実用的なヒント

このセクションには、オートルータでの作業に役立つ情報が記載されています。

## 全般

レイアウト コスト(cfLayer)は、外側の画層から内側の画層の順に大きくするか、すべての画層で同じにする必要があります。外側の画層より小さい値を内側の画層で使用しないでください。これにより、必要な配線メモリが大幅に増加する可能性があります。

> **Note:** 
注:

 オートルータは、円弧としてワイヤをレイアウトすることはできず、マイクロ ビアを設定することもできません。

## 片面基板

最も単純なケースでは、レイヤー 16 (*Bottom*)のみがアクティブになります。優先方向は定義されていません。適切なグリッドを選択して、オートルータを実行します。

オートルータでは、Top 画層も使用できます。そこに配置されたトラックは、基板上のワイヤ ブリッジとして認識されます。レイヤー 41 (*RestrictTop*)では、コンポーネントの周囲、およびワイヤ ブリッジが許可されない領域に制限領域を定義できます。

## 供給レイヤーがある SMD 基板

最初に、供給信号を配線してください。一般的に、SMD コンポーネントと内側の画層に接続するビアの間には、短いトラックを使用することをお勧めします。

パラメータを変更する前に、オートルータで[名前を付けて保存]をクリックして、現在の(既定の)値をオートルータ コントロール ファイル(CTL ファイル)に保存します。

オートルータのセットアップで、バス ルータとすべての最適化パスをオフにします。配線のみがアクティブなままです。次のコスト係数を変更します。

| 
 | 
 | 
 |

| 
cfVia = 0 | 
ビアを許容 | 
 |

| 
mnVia = 1 | 
接続ごとに最大 1 つのビア | 
 |

| 
cfBase.1/16 = 30..99 | 
Top/Bottom のトラック数を減らす | 
 |

| 
mnSegments = 2..8 | 
短いトラック | 
 |

オートルータを開き、[選択]ボタンを使用して、配線する信号を選択します。配線パスの後で、必要に応じて手動で結果を最適化することができます。

残りの接続は、この後で配線されます。[ロード]をクリックして以前に保存したコントロール パラメータをロードし、必要に応じて値を調整します。

## 一部の信号が配線されていない場合に実行できる処理

この場合は、次の設定を確認してください。

- 十分に細かいルーティング グリッドを選択しましたか?

- トラック幅は適切ですか?

- ビアの直径はより小さいですか?

- 最適な最小クリアランスが選択されていますか?

これらの値をさらに最適化するのが合理的でない場合は、リップアップ レベルを上げてみてください。
