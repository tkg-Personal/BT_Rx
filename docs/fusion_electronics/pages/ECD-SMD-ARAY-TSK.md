# SMD パッド配列を追加する

一連の表面実装型デバイス(SMD)パッドを集積回路(IC)またはチップ フットプリントに追加します。

- ライブラリが開いた状態で、**[新しいフットプリント]**をクリックし、最初から描画するか、または既存の記号をベースとしてインポートするかを選択します。

- [ライブラリ記号]ツールバーで、 **[配置] > [SMD パッド配列]** ![SMD 配列のアイコン](../images/icon/ecd/smd-array.png) をクリックします。

- **[SMD パッド配列]**ダイアログで、ベース名(通常は単一の文字)を指定します。ハイフンはあってもなくてもかまいません。

- チップのどちら側にピンを配置するかを選択します。

- ピン数と開始インデックス(通常は 1)を指定します。

- チップの長方形のアウトラインを描画するかどうか、および >NAME ラベルと >VALUE ラベルを含めるかどうかを選択します。

- [SMD 間隔]フィールドで、隣接するパッド間の X および Y 距離を指定します。

- [OK]をクリックして、キャンバスにチップを配置します。
