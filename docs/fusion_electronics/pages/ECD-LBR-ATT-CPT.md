# 電子デザイン: コンポーネント属性

電子デザイン ライブラリ属性は、Fusion 電子デザインのコンポーネント、コンポーネント バリアント、またはライブラリに関する主要なデータを格納するプロパティです。属性は、データシート、部品表データ、物理プロパティ、シミュレーション詳細などの重要な情報を定義するのに役立ちます。この情報は、回路図や下流のワークフローで使用されるときに、コンポーネントとともに移動します。

## 既定の属性

新しいコンポーネントを作成すると、Fusion 電子デザインによって次の属性が既定で割り当てられます。

- **CATEGORY_L1**: プライマリ カテゴリ(抵抗器、センサーなど)。

- **CATEGORY_L2**: セカンダリ カテゴリ(固定とトリムポット、磁気と加速度計など)

- **DATASHEET**: コンポーネント データシートへのリンク。

- **MANUFACTURER**: コンポーネント メーカー名。

- **MPN**: 製造元のパーツ番号。

- **VALUE**: コンポーネントの値(抵抗、静電容量など)。

- **VENDOR**: サプライヤーまたはベンダー名。

- **VPN**: ベンダーのパーツ番号。

コンポーネントを回路図に配置して部品表にリンクしたときにデータの準備が完了するように、これらの属性を入力します。

## オプションの属性

必要に応じて、[属性を作成]ダイアログを使用して、次のオプションの属性を追加できます。

- **DESCRIPTION**: コンポーネントの注記。

- **MAXTEMP**: 最大動作温度。

- **OPERATING_TEMPERATURE**: 標準動作温度範囲。

- **PACKAGE**: 物理パッケージ タイプ(表面実装または貫通穴)。

- **PACKAGE_SIZE**: 物理パッケージ サイズ。

- **PART_STATUS**: コンポーネントのライフサイクル ステータス(アクティブ、廃番など)。

- **POPULARITY**: 使用頻度または人気度指標。

- **ROHS**: RoHS コンプライアンス ステータス(準拠または非準拠)。

- **SERIES**: コンポーネント シリーズまたはコンポーネント ファミリ。

- **THERMALLOSS**: 熱損失データ。

## カスタム属性

[属性]ダイアログを使用して、ライブラリ、コンポーネント、またはコンポーネント バリアントのカスタム属性を定義することもできます。カスタム属性を使用すると、既定の属性またはオプションの属性ではカバーされない特定のデータを追跡できます。詳細については、「[**新しい属性を作成する**](ECD-CREATE-ATT-TSK.htm)」のドキュメントを参照してください。
