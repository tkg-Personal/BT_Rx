# フィーチャ

## 全般

- 最大図面領域 150 x 150 インチ

- 分解能 0.003125 μm

- mm または inch のグリッド

- 最大 255 個の図面画層

- コマンド(スクリプト)ファイル

- データのエクスポートとインポート、および自己定義コマンドの実現のための、C 言語に似たユーザ言語

- 完全にドキュメント化され、読みやすい XML データ構造

- 簡単なライブラリ編集

- コラボレーションのために共有できる管理されたライブラリ

- ドラッグアンドドロップによる、既存の要素を使用した自己定義ライブラリの構成

- ドラッグアンドドロップによる、他のライブラリからの新しいパッケージ バリアントの簡単な生成

- パッケージ バリアントの自由回転(0.1 度のステップ)

- パッケージ エディタ内の任意のパッド形状

- ライブラリ ブラウザと強力なコンポーネント検索機能

- 3D パッケージ クリエータ

- テクノロジ サポート(例: 74L00、74LS00)

- ラベル付きの図面フレームを定義する FRAME コマンド

- 自由定義可能な属性(ライブラリ内のデバイス、回路図、レイアウトに適用可能)

- アセンブリ バリアントのサポート

- 使いやすい寸法記入ツール

- 一貫性を維持しながら異なるプロジェクトを統合(デザインの再利用)

- 回路図と基板(またはその一部)を共通のデザイン ブロックに保存し、他のプロジェクトでデザインを再利用

- 回路図とレイアウトのテンプレートとしてのデザイン ブロック

- 統合された PDF データ エクスポート機能

- グラフィックス ファイルのエクスポート機能(BMP、TIF、PNG...)

- オペレーティング システムのドライバの印刷出力と印刷プレビュー

- データベース サポートを使用したパーツ一覧の生成(bom.ulp)

- Gerber X2 サポートを使用したワンクリック製造データ

- すべてのオブジェクトのコマンドが含まれているユーザ定義可能なコンテキスト メニュー

- コンテキスト メニューを使用した、オブジェクトのプロパティへのアクセスと編集が可能

- 自動バックアップ機能

## 回路図エディタ

- 階層構造で回路図をデザイン可能: モジュールはモジュール インスタンスで表され、回路図の最上位レベルのポートで接続されます。

- 階層は任意の深さに到達可能

- 回路図あたり最大 999 個のシート

- 回路図とモジュールのシートのアイコン プレビュー

- ドラッグアンドドロップを使用した、モジュールおよび回路図のシートの並べ替え

- ネットの相互参照

- 接触の相互参照の自動生成

- ピン/バス/ネットのブレークアウト機能

- バス メンバ ネットへの自動ラベル配置

- パーツの簡単なコピー

- 回路図とレイアウトの間の一貫性を失わないパーツの置換機能

- 回路図と基板の間のオンライン Forward&Back Annotation

- 自動基板生成

- 供給信号の自動生成

- 電気ルール チェック(回路図でのエラー チェックおよび回路図とレイアウトの間の整合性チェック)

## レイアウト エディタ

- SMD を完全にサポート

- ブラインド ビアと埋め込みビアのサポート

- 任意の角度でのオブジェクトの回転(0.1 度のステップ)

- 基板ビューの反転

- コンポーネント、テキスト、長方形、円、寸法の移動をロック可能

- テキストを任意の方向に配置可能

- レイアウト配線中の信号線の動的計算

- 吸着パッド機能

- 任意の半径の丸いコーナーでトラックをレイアウト可能

- ワイヤのジョイントをスムーズにするマイタ

- 前の冗長トレースの自動削除により 2 つのパッド間のパス部分を再配線する、ループ削除機能

- 基板レイアウトのデザイン ルール チェック(オーバーラップ、パッドまたはトラックの計測など)

- ライブ DRC

- 銅注入(グラウンド面)

- パッケージ バリアントのサポート

- 手動配線のための障害物回避モードと障害物プッシュ モード

- 簡易半自動配線のためのクイック配線モード

- 既に配線されているトレースをスムーズにするスムーズ配線

- 差動ペアの配線

- 信号の長さ補正のためのミアンダ形状の自動作成

- SMD コンポーネントと信号のファンアウト ツール

- マシン、テスト機器、ミル マシンを設置するためのデータやその他のデータ形式のデータを生成するための、ユーザ定義可能で自由にプログラムできるユーザ言語

- CAM プロセッサを使用した、Gerber プロッタおよびドリル マシンの製造データの出力

- メカニカルな CAD システム用の 3D データ(STEP や STL など)の作成

## オートルータ

- 基本プログラムへの完全な統合

- グリッドレス配線アルゴリズムを使用した TopRouter (事前にオートルータを使用することが可能)

- 信号画層でのルーティング グリッドおよび優先方向の自動選択(任意)

- BGA エスケープ配線のための特殊な BGA オートルータ

- 複数の配線ジョブを同時に処理するためのマルチコア プロセッサのサポート

- レイアウトに定義した一連のデザイン ルールの使用

- 任意の時点での手動配線と自動配線の切り替え

- リップアップと再試行のアルゴリズム

- ユーザ定義可能な方法(コスト係数による)

- 0.8 mil (0.02 mm)までのルーティング グリッド

- 配置制限なし

- 最大 16 個の信号画層(ユーザ定義可能な優先方向を使用)

- ブラインド ビアと埋め込みビアの完全なサポート

- さまざまなネット クラスを考慮
