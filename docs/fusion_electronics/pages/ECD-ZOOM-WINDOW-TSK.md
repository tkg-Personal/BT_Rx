# 指定したウィンドウに拡大ズームする

デザインの対象領域を拡大ズームするには、このオプションを使用します。

- [回路図デザイン]または[PCB 設計]ツールバーで、**[ビュー] > [ウィンドウにズーム]** ![ウィンドウにズームのアイコン](../images/icon/ecd/zoom-window.png) を選択します。

- クリックしてドラッグし、デザインの対象領域の周囲に長方形を描画します。ウィンドウが拡大ズームされ、対象領域のみが表示されます。
