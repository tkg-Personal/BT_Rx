# 円形の 2D PCB アウトラインを作成する

プリント基板(PCB)の物理的な境界を定義するには、円形の基板アウトラインを作成します。

- ボード形状が存在する場合は、削除します。選択用長方形を使用してボード全体を囲み、 **[削除]** をクリックすることができます。

> **Note:** 注: 回路コンポーネントとエアワイヤは削除されません。

- **[PCB 設計]** ツールバーで、 **[アウトライン円]** ![アウトライン円のアイコン](../images/icon/ecd/outline-circle.png) をクリックします。

- 表示されるダイアログで、線幅の調整を検討します。

- キャンバスの下に表示されるプロンプトに従って、クリックして円の中心を定義します。

- カーソルをドラッグして円を拡大し、クリックして半径を定義します。

- ボードのアウトラインを選択する場合、 **[検査]** パネルでボードの半径を確認または調整できます。
