# チュートリアル: プロジェクト データを確認、編集する

複雑な回路図と回路基板を使用することにより、情報パネルでプロジェクト データを効率的に確認し、一括変更を行う方法について説明します。情報パネルは、コンポーネントを表示して確認するのが困難な、密集したデザインを使用する場合に特に効果的です。

このチュートリアルの内容:

- フィルタを使用してネットを検索し、回路図デザインでプロパティを変更します。

- 回路図デザインのコンポーネントを置き換えます。 

- 回路図デザインのコンポーネントのグループを移動します。

- 回路図デザインのコンポーネントのグループを作成します。

- 信号を検出し、PCB デザインで自動配線します。

- PCB デザインの特定の画層のコンポーネントを検索します。
