# PCB レイアウト内のオブジェクトを反転する

**[反転]** は、フットプリントなどの PCB レイアウト内のオブジェクトをボードの反対側に反転します。

![2D 結果を反転](../images/concepts/ecd-flip-2d.png)

図 1: 2D PCB 上で表示される IC1 の反転結果

![3D 結果を反転](../images/concepts/ecd-flip-3d.png)

図 2: 3D PCB 上で表示される IC1 の反転結果

[反転]をクリックすると、[反転]ダイアログが開き、次の便利なオプションが表示されます。

| 
ルート モード | 
説明 | 
 |

| 
再配線 | 
接続されているワイヤが再配線されます | 
 |

| 
切断 | 
接続されているワイヤが切断され、所定の位置に残ります | 
 |

| 
ルート解除 | 
接続されているワイヤが削除されます | 
 |

オプションを使用する場合は、反転するオブジェクトをクリックする前に設定します。

## 文字を反転する

プリント基板の下側の文字は、ボードの下側を見たときに読めるように自動的にミラー化されます。

回路図内の反転された文字は、原点の反対側に印刷されますが、通常通り読むことができます。

> **Note:** 注: 内側レイヤー(2...15)にある穴やビアなどのオブジェクトは、反転してもレイヤーは変更されません。
