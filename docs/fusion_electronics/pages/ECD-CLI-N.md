## NAME

**関数**

 名前を表示および変更します。

**構文**

 NAME ..
 NAME new_name
  NAME old_name new_name

> **Note:** **関連項目:** [LABEL](ECD-CLI-L.html#label)、[SHOW](ECD-CLI-S.html#show)、[REPOSITION](ECD-CLI-R.html#reposition)、[VALUE](ECD-CLI-V.html#value)、[RENUMBERREFDES](ECD-CLI-R.html#renumberrefdes)

NAME コマンドは、選択したオブジェクトの名前を表示または編集するために使用されます。

パーツ、要素、パッド、SMD、ピン、およびゲートを名前で選択することもできます。これは、特に、オブジェクトが現在表示されているウィンドウ領域の外にある場合に便利です。その他のオブジェクト タイプ(ネット、バス、信号など)は、最初にクリックする必要があります。

### ライブラリ

ライブラリ編集モードでは、NAME コマンドは、選択したパッド、SMD、ピン、またはゲートの名前を表示または編集するために使用されます。

### 自動命名

EAGLE では自動的に名前が生成されます(ネットの場合は N$..、要素の場合は E$..、信号の場合は S$..、パッド、ピン、SMD の場合は P$..)。一般に、これらの自動生成された名前をよく使用される名前で置き換えると便利です(たとえば 14 ピンのデュアル インライン パッケージの場合は 1...14)。パーツの自動命名は、[PREFIX](ECD-CLI-P.html#prefix) を使用してコントロールできます。

### 命名規則

EAGLE のオブジェクト名の各文字は自動的に大文字に変換されます。通常の文字に加えて、EAGLE ではブランクとセミコロン(「;」)を除く他のほとんどの特殊文字を使用できます。ネット、パーツ、モジュール インスタンス、信号、および要素の場合、コロン(「:」)もサポートされません。これは、階層デザイン内のオブジェクトを自動的に一意に命名するためにコロンが必要なためです。
たとえば、要素 AMP1:IC1 は、一貫性を想定すると、モジュール インスタンス AMP1 で使用されているモジュール内のパーツ IC1 にのみ関連します。メイン回路図以外に、各モジュールも独自の_名前空間_を持っています。したがって、メイン回路図と一部のモジュールにパーツ R1-R9 があっても問題はありません。[MODULE](ECD-CLI-M.html#module) コマンドの命名に関するセクションも参照してください。

### 回路図

ネットまたはバスの名前を変更する場合、異なるシート上に配置されている複数のセグメントで構成されている可能性があるため、3 つのケースをプログラムで区別する必要があります。したがって、メニューで次の選択肢がユーザに提示されます: [このセグメント]
[このシートのすべてのセグメント]
[すべてのシートのすべてのセグメント]

これらの質問は必要に応じてポップアップ メニューに表示され、マウスで適切な項目を選択するか、適切なホット キー(T、E、A)を押して回答することができます。

ネットおよびバスを変更する際に、名前の変更後に同じネットまたはバスにラベルを配置するかどうかをユーザに確認するチェックボックスがあります。このチェックボックスは、ラベルがまだ存在しない場合は既定でオンになります。このチェックボックスをオンにして名前の変更を確認すると、LABEL コマンドが開始され、完了するとコントロールが NAME コマンドに戻ります。

### ポリゴンベタ

ボード内の信号ポリゴンベタの名前を変更する際に、このポリゴンベタのみの名前を変更する(つまり、1 つの信号から別の信号に移動する)か、信号全体に別の名前を付けるかを選択できます。

## NET

 **関数**

 回路図にネットを描画します。

**構文**

 NET [net_name] [curve | @radius] ..
 NET BREAKOUT [PIN | UNIQUE] [LENGTH=VALUE] ..
 NET BREAKOUT [AUTOINCR] [LENGTH=VALUE] net_name ..  

**マウス キー** 

 *右クリック*すると、ワイヤ曲げスタイルが変更されます(「[SET Wire_Bend](ECD-CLI-S.html#x129)」を参照)。
 *[Shift]+右*クリックは、曲げスタイルの切り替え方向を反転します。
 *[Ctrl]+右*クリックは、対応する曲げスタイルを切り替えます。

> **Note:** **関連項目**: [BUS](ECD-CLI-B.html#bus)、[BREAKOUTBUS](ECD-CLI-B.html#breakoutbus)、[CLASS](ECD-CLI-C.html#class)、[NAME](#name)、[PINBREAKOUT](ECD-CLI-P.html#pinbreakout)、[SET](ECD-CLI-S.html#set)

NET コマンドは、回路図図面のネット レイヤに個々の接続(ネット)を描画するために使用されます。最初のマウス クリックでネットの始点をマークし、2 番目のマウス クリックでセグメントの終点をマークします。同じ点で 2 回マウス クリックすると、ネットは終了します。

ネット ワイヤはピンの先端にスナップされます。マウス カーソルがピンの任意の部分から SNAP_LENGTH 内にある場合(スナップするグリッドの前または後)、ピンの先端に小さい円が表示されます。これは、マウスをクリックすると、結果として得られるネット ワイヤ点がピンの先端に配置され、ピンへの接続が作成されることを示します。

既に別のネット ワイヤ、バス ワイヤ、またはピンが存在する点にバス ワイヤを配置すると、現在のバス ワイヤはその点で終了します。この機能を無効にするには、「SET AUTO_END_NET OFF;」を使用するか、[オプション]/[設定]/[その他]/[ネットとバスを自動終了]をオフにします。

他のネット ワイヤやピンが 2 つ以上ある点にネット ワイヤが配置される場合、接合が自動的に配置されます。この機能を無効にするには、「SET AUTO_JUNCTION OFF;」を使用するか、[オプション]/[設定]/[その他]/[接合を自動設定]を選択解除します。

パラメータ *curve* または *@radius* が指定されている場合、ネットの一部として円弧を描画できます(詳細は [LINE](ECD-CLI-L.html#line) コマンドを参照)。

### バス信号を選択する

ネットをバス上で開始すると、いずれかのバス信号を選択できるポップアップ メニューが開きます。次に、対応する名前がネットに付けられ、同じ信号の一部になります。バスに複数のパーツ バスが含まれている場合、関連するパーツ バスを選択できるポップアップ メニューが開きます。

### ネット名

ネット名を指定して NET コマンドを使用すると、ネットにはその名前が付けられます。コマンド ラインにネット名が含まれておらず、ネットがバスで開始されていない場合、N$1 という形式の名前が自動的にネットに割り当てられます。

ネットまたはネット セグメントが、回路図のさまざまなシートにわたっていて、同じネット名が使用されている場合、それらのネットまたはネット セグメントは接続されます。

ネット名にカンマ(「,」)を含めることはできません。これは、カンマが[バス](ECD-CLI-B.html#x43)の区切り文字であるためです。また、ネット名に「[」と「]」を含めることもできません。これは、これらの文字が[バス](ECD-CLI-B.html#x43)の範囲指定に使用されるためです。このような範囲のネットには、括弧がないインデックスを使用します。(たとえば バス D[0..15] に対応するネット: D0、D1、...)

### ネット ブレークアウト

NET コマンドには、(任意の数のパーツから)回路図内の任意のパーツ ピンを選択して、選択したピンからブレークアウトする新しいネットを、指定したラベル形式で作成できる、ブレークアウト オプションが含まれています。UI バージョンのコマンドは、NET パラメータ ツールバーの[ピンをブレークアウト]ボタンをクリックするか、CLI から NET BREAKOUT を使用することで利用できます。パラメータを指定せずに(またはNETパラメータツールバー ボタンを使用して)コマンドを開始し、ユーザ インタフェースでブレークアウトのタイプを指定できます。次に、回路図内の任意のピンをクリックして、選択したオプションに一致するラベルを持つ新しいネットをブレークアウトします。

その他のコマンド ライン オプションを使用すると、ラベルのタイプを指定できます(PIN、UNIQUE、または net_name)。net_name を指定した場合、接続されているピンには関係なく、これがネット名に使用されます。

キーワード PIN、UNIQUE、および AUTOINCR には特別な意味があり、net_name 値として使用することはできません。キーワード PIN を使用すると、新しいネットには接続先のピンの名前が付きます。UNIQUE を選択すると、part_gate_pin という形式のラベル文字列が作成されます。パーツ、ゲートおよびピンの名前が、選択したパーツおよびピンの値に基づいて、そのネットで一意の文字列に連結されます。

オプションのキーワード AUTOINCR はカスタム ネット名の前に指定され、ユーザがピンの追加を繰り返した場合、net_name の最後の文字が数字である場合は自動増分されることを示します。たとえば、net_name の前に ADDR1 と AUTOINCR を使用すると、ネットがピンに配置されるにつれ、ADDR2、ADDR3、(以下同様)となります。

オプションのキーワード LENGTH を使用すると、作成される新しいネット セグメントに対して、最小の長さをグリッド単位の数で設定できます。新しいネットの長さは、この最小の長さ、またはラベル テキストの幅に対して計算された長さのうち、大きい方になります。

バスは、BREAKOUTBUS コマンドを使用して、新しいラベル付きネットで自動的にブレークアウトすることができます。このコマンドは、コマンド ライン、またはバス オブジェクトの右クリック コンテキスト メニューで使用できます。詳細については、[BREAKOUTBUS](ECD-CLI-B.html#x44) を参照してください。

パーツは、PINBREAKOUT コマンドを使用して、新しいラベル付きネットで自動的にブレークアウトすることができます。このコマンドは、コマンド ライン、またはパーツ インスタンスの右クリック コンテキスト メニューで使用できます。詳細については、[PINBREAKOUT](ECD-CLI-P.html#x101) を参照してください。

### 線幅

次のコマンドを使用して、NET コマンドで描画される線の幅を変更できます。

SET NET_WIRE_WIDTH width;

(既定: 6 mil)。

### 反転信号

反転信号(「active low」)の名前は、次のように前に感嘆符(「!」)が付いている場合に上線付きで表示できます。

  !RESET

これは、次のような結果になります。

  _____
  RESET

詳細については、[TEXT](ECD-CLI-T.html#text) コマンドの説明を参照してください。
