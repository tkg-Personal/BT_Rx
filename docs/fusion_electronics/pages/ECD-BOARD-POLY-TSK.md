# ポリラインを使用して 2D PCB アウトラインを作成する

プリント基板(PCB)の物理的な境界を定義するには、ポリゴンの基板のアウトラインを作成します。

**2D PCB** 作業スペースで、次の操作を行います。

- ボードのアウトラインが存在する場合は、削除します。選択用長方形を使用してボード全体を囲み、 **[削除]** をクリックすることができます。

> **Note:** 注: 回路コンポーネントとエアワイヤは削除されません。

- PCB の **[設計]** ツールバーで、 **[アウトライン ポリライン]** ![アウトライン ポリラインのアイコン](../images/icon/ecd/outline-polyline.png) をクリックします。

- **[アウトライン ポリライン]** ダイアログで、曲げタイプ、線幅、およびその他のオプションの設定を確認および編集します。

**推奨**: 正確なボード寸法を使用する場合は、曲げタイプを使用せずにポリラインを描画することを検討してください。

- 各コーナーをクリックし、ポリゴンを閉じるときにダブルクリックして、ボードのアウトラインを描画します。ボードは閉じたポリゴンに拘束されます。
