# カスタム電子デザイン ライブラリ

カスタム電子デザイン ライブラリは、特定のデザイン ニーズに合わせて作成および管理する電子コンポーネント、記号、フットプリント、3D パッケージのコレクションです。カスタム ライブラリを使用すると、プロジェクトに合わせて再利用可能なコンポーネントのコレクションを作成し、チームのデザインすべてにおいて一貫性を確保できます。

## ライブラリ オーサリング

[ライブラリを作成](ECD-CREATE-CSTM-LIB-TSK.htm)すると、ライブラリ オーサリング作業スペースが表示され、ここで新しい**コンポーネント**、**記号**、**フットプリント**、**3D パッケージ**の作成を開始できます。また、他のライブラリから現在のライブラリにコンポーネントを読み込むこともできます。

## ライブラリのストレージと共有

保存したライブラリは Fusion ハブにアップロードされ、チーム メンバー全員が**データ パネル**を使用してそのライブラリにアクセスできるようになります。アップロードが完了すると、ライブラリが**ライブラリ マネージャ**に追加されて自動的にアクティブになり、そのコンポーネントが**配置パネル**で使用できるようになります。

> **Note:** 注: 他のチーム メンバーには、カスタム ライブラリがライブラリ マネージャに自動的に表示されない場合があります。そのような場合は[手動で追加](ECAD-ADD-MISSING-HUB-LBRS-TSK.htm)できます。
