# 未接続のピン

**[接続なし]** マークを使用して、ピンを未接続のままにしておく必要があることを明示的に示します。これにより、不必要な ERC/DRC 警告を防ぎながら、クリーンなデザインを確保できます。これは、マルチゲート IC の未接続ピンや、グラウンドを必要としないコンポーネントの未接続ピンに有効です。
