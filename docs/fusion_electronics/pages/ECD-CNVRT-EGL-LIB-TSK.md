# Eagle ライブラリを Fusion ライブラリに変換します。

前提条件

- **ターゲット ライブラリ**は**[使用中]**に設定する必要があります。

- **ターゲット ライブラリ**には、**使用ライブラリ**と同じコンポーネントが含まれている必要があります。コンポーネントにバリアントがない場合、それはリストに含まれています。

アクセス

- キャンバスで、コンポーネントを右クリックし、**[ライブラリを入れ替える]**をクリックします。

- [コンポーネントを配置]パネルで、コンポーネントを右クリックし、 **[デザインでライブラリを入れ替える]** を選択します。

## EAGLE ライブラリをマイグレーションする

- 電子デザインを開きます。

- [回路図]ドキュメントをクリックします。

- キャンバスで、ライブラリを入れ替えるコンポーネントを右クリックし、**[コンポーネントを開く]**をクリックします。

- 続くダイアログで、**[はい]** をクリックしてライブラリを Fusion 管理ライブラリとして保存します。

- ライブラリ エディタで、**[保存]**をクリックしてライブラリを閉じます。

- 回路図で、キャンバス内のコンポーネントを右クリックして**[ライブラリを入れ替える]**を選択するか、**[コンポーネントを配置]**パネルでコンポーネントを右クリックして**[デザインでライブラリを入れ替える]**を選択します。

- **[ライブラリを入れ替える]**ダイアログで、新しいライブラリを **[ターゲット ライブラリ]**として選択します。一致するコンポーネントを持つライブラリがコマンドによって特定された場合、一致するライブラリのターゲット ライブラリ フィールドがあらかじめ選択されます。

- すべてのコンポーネントがターゲット ライブラリに存在することを確認します。見つからないコンポーネントのリストが表示されます。ライブラリを使用するには、これらを調整する必要があります。

- ターゲット ライブラリに同じコンポーネントが含まれている場合は、[入れ替え]をクリックします。デザイン内のすべてのコンポーネントがターゲット ライブラリのコンポーネントと入れ替わります。
