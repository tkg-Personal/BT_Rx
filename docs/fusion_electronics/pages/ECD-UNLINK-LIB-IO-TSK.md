# ハブ ライブラリを Library.io からリンク解除する

リンクを解除すると、ハブ ライブラリが Library.io から切断されます。これにより、変更や更新に関する通知が停止します。リンクを解除すると、ライブラリを新しい Library.io ライブラリとしてパブリッシュすることや、別の既存のライブラリにリンクすることができるようになります。

## 前提条件

- **ハブ ライブラリへのアクセス権:** 現在 Library.io ライブラリにリンクされている ハブ ライブラリが開いている必要があります。

## 手順

- Fusion で、**ライブラリ オーサリング作業スペース**を開きます。

- ツールバーの **[管理]** セクションで、 **[Library.io]** セクションを見つけます。

- **[Library.io の管理ライブラリからリンク解除]** ![管理ライブラリからリンク解除のアイコン](../images/icon/ecd/unlink-io.png) をクリックします。

## 次のステップ

- **新しいライブラリとしてパブリッシュする:** リンクを解除すると、ライブラリを新しい Library.io ライブラリとして[パブリッシュ](ECD-PUB-LIO-LIB-TSK.htm)できるようになります。

- **既存のライブラリにリンクする:** リンク解除したライブラリを別の既存の Library.io ライブラリに[リンク](ECAD-LINK-LIBIO-TSK.htm)できます。

> **Note:** **注:** Library.io にリンクされている個々の 3D パッケージは引き続き、新しいバージョンが利用可能になったときに更新を受け取ることができます。
