# 2D PCB アウトラインを修正する

デザインの目的に合わせて 2D PCB のアウトラインを修正します。基板の 1 つのコーナーが原点(0,0)上にあることに注目してください。

- [電子デザイン]作業スペースの[2D PCB]ドキュメント キャンバスで、変更する基板の線を特定します。

- エッジまたはコーナーをクリック アンド ドラッグします。

> **Note:** 注: ウィンドウの右側にある **[検査]** 固定パネルで、正確なサイズを確認したり、数値を変更したりできます。
