# 電子デザイン ライブラリ内のアセットの説明を編集する

ライブラリ内の任意のタイプのアセットに説明を付けることができます。説明は、[配置]パネルでアセットを参照する際に表示したり、デザインにコンポーネントを配置する際に検索機能で使用することができます。

- **コンテンツ マネージャ**のコンポーネント行で右クリックし、 **[説明を編集]** を選択します。

- ダイアログのテキスト ボックスに説明を入力します。

- **[OK]** をクリックして、説明を保存します。

- [コンテンツ マネージャの詳細]セクションに説明が表示されます。

> **Note:** 注: 以前に作成した説明に HTML が含まれている場合がありますが、新しい説明には HTML タグを含めないようにする必要があります。**[HTML を削除]** ボタンを使用すると、既存の説明から HTML をすばやく削除できます。ライブラリまたはコンポーネントのデータシート リンクを追加するには、代わりに **DATASHEET** 属性を使用します。
