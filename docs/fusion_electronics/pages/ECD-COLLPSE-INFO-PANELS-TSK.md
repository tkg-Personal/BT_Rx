# 固定パネルを折りたたむまたは展開する

必要のないときは、[電子デザイン]作業スペースの固定パネルを折りたたんだり展開したりして、キャンバスをより広く表示することができます。

- スタック パネルの場合は、二重矢印コントロールをクリックして展開または折りたたみます。

 ![展開/折りたたみのアイコン](../images/how-to/ecd-panel-stack-collapse-01.png)

- タブ パネルの場合は、タブを右クリックして **[ダイアログを折りたたむ]** ![パネルを折りたたむのアイコン](../images/icon/info-panel-collapse.png) を選択します。

折りたたまれたパネルにカーソルを合わせると、パネルが表示され、パネル上で操作を行うことができます。パネルのタイトル バーをクリックして、一時的にパネルを表示することもできます。終了すると、パネルは自動的に折りたたまれます。たとえば、回路図図面シート間で切り替えるには、ウィンドウの下部にあるシート パネルの上にカーソルを合わせ、パネルが展開されたら、表示するシートを選択します。シートが表示され、パネルが所定の位置に折りたたまれます。
