# 信号を手動で配線する

配線をより詳細にコントロールする場合は、エアワイヤを手動で物理的な配線に変換します。

- [PCB 設計]ツールバーで、**[手動配線]** ![手動配線のアイコン](../images/icon/ecd/route-manual.png) をクリックします。

- [手動配線]ダイアログで、選択されているオプションを確認および修正します。

**[ループ削除]**は、アクティブな場合、新しいトレースが同じ点間の既存のトレースと置き換えられているかどうかを検出し、古いトレースを取り除きます。

- **[トレースの曲げ]**は、トレースの新しい各セグメントの角度または形状をコントロールします。[Ctrl]キーを押しながら右クリックすると、オプションを切り替えることができます。

- ダイアログの下半分にあるコントロールを使用して、トレースおよびビアの物理プロパティを指定します。トレース幅は、コマンド ライン フィールドに数値を入力することによっても変更できます。

- 固定された配線済み信号と置換するエアワイヤをクリックします。エアワイヤの端点がハイライト表示されます。

- カーソルをドラッグして最初のルーティング セグメントを開始し、クリックしてコーナーまたは曲げを配置し、作業を続行します。

- 最後のセグメントを削除する場合は、[Backspace]を押します。

- [Esc]を押して配線を終了します。

- **[ルート] > [手動]** ![手動ルートのアイコン](../images/icon/ecd/route-manual.png) をクリックし、コンポーネント上の端子をクリックし、エアワイヤに沿って接続先コンポーネントにドラッグし、クリックして[Esc]を押して接続を終了します。

 ![配線のイメージ](../images/how-to/board-route.png)

- 配線の一部を元に戻す場合は、セグメントをクリックしてハイライト表示し、[Delete]キーを押します。

- 接続が基板を通過して反対側に移動するようにする場合は、「ビア」と呼ばれる穴を作成します。接続に沿った点を選択し、マウスの中央ボタンをクリックして、緑色のリングで示されるビアを作成します。

![配線の GIF](../images/how-to/route-via.gif)

マウスを使用してビアを移動してからクリックすることでビアを配置し、配線を続行できます。このとき、色は基板の上側の赤色から下側の青色に変わります。

- 残りのエアワイヤの配線を続行し、必要に応じてビアを配置します。
