# 記号

回路図ドキュメントで使用される、Fusion ライブラリ コンポーネントの回路図表現です。関連用語として「コンポーネント」、「フットプリント」、「パッケージ」を参照してください。

**記号**は、回路図内のコンポーネントのロジックと、その機能的接続を表します。グラフィック用語では、記号が最も単純な要素です。すべての記号には、1 つまたは複数の端子ピン、またはピンがあります。
