# デザイン ドキュメント

**電子デザイン**には、**回路図**および **2D PCB** の両方にデザインをドキュメント化するためのツールが含まれています。これらのツールは、ツールバーの **[ドキュメント]** セクションにあります。キャンバスに直接ジオメトリを描画したり、文字を入力したり、PCB 内のオブジェクトに寸法を追加したり、デザインからビューを出力したりすることができます。
