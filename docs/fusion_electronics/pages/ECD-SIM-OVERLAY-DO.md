# シミュレーション オーバーレイを表示/非表示する

この機能は、動作点シミュレーションの表示に適用されます。

[回路図デザイン]ツールバーで、 **[シミュレーション オーバーレイを表示]** ![シミュレーション オーバーレイのアイコン](../images/icon/ecd/show-simulation-overlay.png) をクリックして、回路内のさまざまな場所の動作点値を表示または非表示にします。
