# オブジェクトのコピーと貼り付け

- オブジェクトを選択するか、複数のオブジェクトの周囲に選択用長方形を描きます。

- [デザイン]ツールバーで、 **[コピー]** ![コピーのアイコン](../images/icon/ecd/copy.png) をクリックします。

- [コピー]ダイアログで、[OK]をクリックして操作を終了します。

- [デザイン]ツールバーで、 **[貼り付け]** ![貼り付けのアイコン](../images/icon/ecd/paste.png) をクリックします。クリップボードからコピーしたオブジェクトがカーソル上に表示されます。

- 必要に応じて、[貼り付け]ダイアログで次の操作を行います。

オブジェクトを貼り付けるレイヤを選択します。**元のレイヤ**、**アクティブなレイヤ**、またはリストから別のレイヤを選択します。

- **[回転]**オプション(90 度、180 度、270 度でオブジェクトを回転)をクリックするか、[ミラー]をクリックしてオブジェクトを左から右に反転します。カーソル上でオブジェクトをプレビューしているときに右クリックすると、オブジェクトが段階的に回転します。注: ビアを貼り付ける場合、**[発側呼び合いを使用]**を選択して、同じネット上のビアを貼り付けることができます。選択解除されている場合は、貼り付け先のネットに新規ネットが作成されます。

- クリックして、目的の位置にオブジェクトを配置します。間違えた場合は、[元に戻す]をクリックしてから、再度[貼り付け]をクリックします。オブジェクトがカーソルに再び表示されます。
