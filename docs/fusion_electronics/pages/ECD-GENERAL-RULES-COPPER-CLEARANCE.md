# 銅箔クリアランス

銅箔クリアランスは、異なる信号のトラック、銅箔ベタ、パッド、SMD、ビア間の最小距離、および同じ信号の SMD、パッド、ビア間の最小距離です。「同じ信号」チェックの値を 0 に設定すると、それぞれのチェックが無効になります。

これは、次のルールに適用されます。

- ワイヤからワイヤ

- ワイヤからパッド

- ワイヤからビア

- パッドからパッド

- パッドからビア

- ビアからビア

- SMD からパッド

- SMD からビア

- SMD から SMD

ネットが特殊なネット クラスに属している場合、Fusion では[デザイン ルール]** > **[カスタム ルール]** で設定した **[クリアランス]** の直径値をチェックします。これらの値が **[デザイン ルール]** > **[一般ルール]** の値を超える場合、Fusion では高い方の値を適用します。
