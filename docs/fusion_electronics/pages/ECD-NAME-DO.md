# コンポーネント名を変更する

- [回路図デザイン]ツールバーで、**[名前]** ![名前のアイコン](../images/icon/ecd/name.png) をクリックします。

- 名前を変更するコンポーネントのオブジェクト ハンドル(+)をクリックします。[名前]ダイアログが開き、現在の名前が表示されます。

- 新しい名前を入力して、[OK]をクリックします。

- 別のコンポーネントをクリックして、名前を変更するか、[Esc]を押して名前の設定セッションを終了します。
