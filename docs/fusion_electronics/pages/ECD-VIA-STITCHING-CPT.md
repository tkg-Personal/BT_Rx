# ステッチ ビア

ビア ステッチでは、PCB の異なるレイヤーにまたがるより大きな銅領域を接続するために、複数のビア(銅で満たされた穴)を使用します。この技法により、PCB 内のリターン パスが強化され、熱散逸が低減され、バランスのとれた銅抵抗が維持されます。

また、ビア ステッチは、その基本的な役割にとどまらず、EMI 干渉からの保護という高周波動作における極めて重要な役割も果たしています。このため、Wi-Fi、Bluetooth、およびその他の広帯域の高周波要素において非常に貴重なツールとなっています。

## ビア ステッチのタイプ

- **一定グラウンド ビア ステッチ**: この一般的なタイプでは、健全なリターン パスとグラウンド面内の低抵抗が維持され、負荷デバイスから電源までのリターン パスが短くなります。

- **サーマル ビア ステッチ**: ボード上のヒートシンクを強化するために使用されるサーマル ビア ステッチでは、大電流電源トレース付近の大きな平面に多数のビアを使用します。表面積が増えることで、より効果的に熱を吸収し、ボードを低温に保ちます。

- **シールド ビア ステッチ**: 高周波 RF 回路または混合信号回路における EMI 関連の理由から、シールド ビア ステッチでは PCB ビア シールドが作成されます。これは、高周波トラック付近の大きな銅箔面の周囲に、1 列または複数列のビアをステッチするものです。

> **Note:** **注: 現在、Fusion ではシールド ビア ステッチはサポートされていません。**
