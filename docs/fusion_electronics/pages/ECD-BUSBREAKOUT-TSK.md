# バスを部分切断

- [回路図デザイン]ツールバーで、**[接続] > [バスを部分切断]** ![バスの部分断面を追加アイコン](../images/icon/ecd/bus-breakout.png) を選択します。

- バスが選択されている場合、**[バスを部分切断]**ダイアログにネットの一覧が表示されます。

- バスが選択されていない場合は、ネットを部分切断するバスを選択します。使用可能なネットが[ネット]一覧に表示されます。

- バスの**[エイリアス]**(短い名前)を確認し、必要に応じて変更します。

- [ネット]セクションで、部分切断するネットを選択します。部分切断しない項目は選択解除します。

- **[完了]**をクリックします。
