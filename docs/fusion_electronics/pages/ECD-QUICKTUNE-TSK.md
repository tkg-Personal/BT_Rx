# トレースを QuickTune する

- **[PCB デザイン]** ツールバーで、 **[QuickRoute] > [QuickTune]** ![QuickTune のアイコン](../images/icon/quick-tune-icon.png) をクリックします。

- 調整する基板上の単一のトレース、差動ペア、または複数のトレースを選択します。

- **[QuickTune]**ダイアログで、既定値を調整します。

- **[選択済みを調整]**をクリックして調整します。

- **[完了]**をクリックして、[QuickTune]ダイアログを閉じます。

**注**:

- エラーが発生した場合は、[QuickTune]ダイアログで設定を調整し、再試行してください。

- QuickTune は、丸キャップと結合を使用するようにトレースを変更します。
