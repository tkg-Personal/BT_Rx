# 電子デザイン ドキュメント

**電子デザイン** ドキュメントは、回路図、2D PCB、3D PCB ドキュメント間の同期を管理するプライマリ(包括的な)ドキュメントです。回路図ドキュメントや基板ドキュメントを作成すると、既定では、それらは単一の Fusion ウィンドウの個別のタブに配置されます。回路図、基板設計、およびライブラリを個別のウィンドウで平行して作業する場合は、これらのドキュメント タブをドッキング解除することができます。

デザイン ドキュメント内で、回路図、2D PCB および 3D PCB ドキュメントを作成すると、左側のパネルに一覧表示され、右側にサムネイルとして表示されます。

![電子デザイン ドキュメント](../images/overview/ecd-elec-design-doc.png)

図 1: 左側に(2)回路図、(3)2D PCB、(4)3D PCB ドキュメントの一覧が、右側にそのサムネイルが表示された(1)電子デザイン ドキュメント

- **デザイン ドキュメント**は、他の 3 つのドキュメント間の同期を管理するプライマリ(包括的な)ドキュメントです。新しいデザインは、常にデザイン ドキュメントと回路図ドキュメントから開始されます。

- **回路図ドキュメント**では、さまざまな基板コンポーネントとその接続を表す記号を追加して、デザインのロジックと機能をドキュメント化します。回路図ドキュメントには、必要に応じて最大 999 のシートを含めて、デザイン ロジックをドキュメント化することができます。回路図と 2D PCB ドキュメントは、[電子デザイン]作業スペースによって自動的に一貫性が保たれます。

- **2D PCB ドキュメント**では、コンポーネントを互いに関連させてレイアウトします。関連付けられている回路基板は、マウスを 1 回クリックするだけでいつでも生成することができます。2D PCB ドキュメントが開き、すべての回路図コンポーネントが空のボードの横に配置されます。コンポーネントには、コンポーネント間の未配線の点間の接続を表す細い黄色の線(エアワイヤとも呼ばれる)があります。ボード形状を定義またはインポートし、2D PCB ツールを使用してボード上のコンポーネントを特定することができます。

- **3D PCB ドキュメント**は、2D PCB データから生成された 3D モデルです。2D および 3D PCB ドキュメントは回路図から生成され、参照するライブラリのコンポーネントが含まれています。コンポーネント 3D モデルは、論理シンボルおよびコンポーネント フットプリントとともにライブラリに含めることができます。このコレクションは、3D パッケージと呼ばれます。コンポーネントに関連付けられた 3D モデルがない場合、3D ドキュメントを生成するときにプレースホルダ形状(長方形)で置き換えられます。[2D PCB]作業スペース内のコンポーネントに 3D モデルをマッピングすることができます。
