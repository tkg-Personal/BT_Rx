# ファンアウトの使用法とオプション

ファンアウトは、複数のコンポーネントをグラウンドにすばやく接続する場合に特に便利です。 

デバイス ファンアウトは、デバイスの各パッドから短い距離をトレースし、次にビアをドロップします。 

信号ファンアウトは、デザイン ルール違反に従って、信号パスに沿って各パッドからのビアを追加しようとします。

ボール グリッド アレイ(BGA)オートルータには、要素と信号に関する追加の設定があります。 

[スタイル]設定は、デバイス パッドまたは信号パスの外側、内側、または交互に両側の、どちらにビアを配置するかをコントロールします。
