# PCB データを Ansys にエクスポートする

- キャンバスで回路基板が開いた状態で、[シミュレーション]ツールバーの **[Ansys 用 PCB データを書き出し]** ![Ansys に書き出しのアイコン](../images/icon/ansys.png)をクリックします。

- [Ansys 用 PCB データを書き出し]ダイアログで、次の操作を行います。

出力のファイル名を入力します。

- ファイル タイプとして、**Ansys Electronics Database (*.edb)**を選択します。

- ファイルの保存場所を参照します。

- **[保存]**をクリックします。
