# Library.io とリンクする

Library.io とリンクすると、Fusion ハブ ライブラリと Library.io Web サイトが接続され、両方のプラットフォーム間で変更を同期できるようになります。Library.io ライブラリをハブに読み込むか分岐すると、リンクが自動的に作成されます。また、ハブ ライブラリを Library.io に[パブリッシュ](ECD-PUB-LIO-LIB-TSK.htm)することや、既存のライブラリに手動で[リンク](ECAD-LINK-LIBIO-TSK.htm)することもできます。

## Library.io にリンクする理由

- **更新の同期**: パブリック ライブラリから分岐した後で、パブリック ライブラリから更新をプルします。

- **ハブ間での共同作業**: Library.io を使用して、複数のハブ間でライブラリを同期します。

- **ライブラリのパブリッシュ**: (許可されたパブリッシュ元の場合)一般公開されている Library.io ライブラリに変更をパブリッシュします。

## Library.io ライブラリにアクセスする

ライブラリ マネージャの **[プライベート ライブラリ]** タブには、アクセス権のある Library.io ライブラリの一覧が表示されます。ただし、これらのライブラリをデザインで直接使用することは推奨されません。Library.io には独自の権限システムがあり、現在のハブのチーム メンバーは、これらのプライベート ライブラリにアクセスできない場合があります。

## 権限を管理する

Library.io Web サイトで、Library.io ライブラリの[権限を管理](ECAD-LIBIO-PERM-TSK.htm)できます。管理フォルダへのアクセス権を持つすべてのユーザが Library.io のライブラリに変更をプッシュできるため、信頼できる作成者にのみ権限を付与してください。
