# 信号をすべて配線解除する

- [PCB 設計]ツールバーで、 **[配線解除] > [すべて配線解除]** ![配線解除のアイコン](../images/icon/ecd/unroute-all.png) をクリックします。このオプションは、[ルート解除]ダイアログからも使用できます。

- ボード上のすべての信号を配線解除するかどうかを確認します。[はい]をクリックして、すべての信号を未配線の信号に変換します。
