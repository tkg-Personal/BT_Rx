# デザイン基本設定を使用する

- PCB ルールのツールバーで、 **[基本設定]** をクリックします。

- ユーザまたは基板ハウスの要件に合わせて値を調整します。

- **[名前を付けて保存]**をクリックしてデザイン ルール ファイル(EDRU)に保存し、別の基板にこの一連のルールを使用できるようにします。

- 一連のデザイン ルールを基板に適用するには、任意の DRU ファイルをコントロール パネルのツリー ビューの[デザイン ルール]ブランチから[レイアウト エディタ]ウィンドウにドラッグするか、[デザイン ルール]ウィンドウの[ファイル]タブで**[ロード]**をクリックします。

- **[説明を編集]**をクリックして、現在のパラメータ セットの説明を入力します。説明は通常、[ファイル]タブに表示されます。HTML テキストを使用することができます。

[DRC]ダイアログには以下のタブが含まれています。

| 
 | 
 | 
 |

| 
**ファイル** | 
デザイン ルールを管理する | 
 |

| 
**レイヤー スタック** | 
銅箔層の数、多層基板の構造、ビアの種類と長さ、銅箔層と分離層の厚さ | 
 |

| 
**環** | 
パッドおよび(マイクロ)ビアの残留銅リングの幅(以前は Restring と呼ばれていました) | 
 |

| 
**形状** | 
パッドと SMD の形状 | 
 |

| 
**供給** | 
銅平面の熱記号 | 
 |

| 
**マスク** | 
はんだストップ マスクとはんだクリーム マスクの値 | 
 |

> **Note:** 注: ほとんどのパラメータの説明に小さなイメージを利用しています。パラメータ行をクリックすると、関連付けられた表示が即座に表示されます。
