# 3D コンポーネントを移動する

- [3D PCB]ツールバーで、**[移動]** ![3D 移動のアイコン](../images/icon/ecd/move-3d.png) をクリックします。

- ボード上で、移動するコンポーネントをクリックします。数値または手動ハンドルを使用して移動することができます。

[移動]ダイアログで、次の操作を行います。

コンポーネントを X 方向と Y 方向に移動する正確な距離を設定する。

- ボード上でコンポーネントを回転する角度を設定する。負の値は時計回りの回転、正の値は反時計回りの回転です。

- [PCB を更新]を選択して、位置変更を 2D 回路基板デザインに自動的に転送する。

- [経路を再設定]を選択してコンポーネントへの電気接続を調整し、接続を維持する。

- コンポーネントの近くにあるハンドルを使用して、次のように手動で移動します。

矢印をクリックすると、コンポーネントがその軸に沿って前後にスライドします。

- 中心の正方形をクリックすると、コンポーネントの向きを維持したまま、基板上でパーツを任意の方向にフリーハンドで移動できます。

- [移動]ダイアログの[Z 角度]コントロールと同様に、円をクリックするとボード上でコンポーネントをフリーハンドで回転できます。
