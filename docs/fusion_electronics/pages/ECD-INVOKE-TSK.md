# コンポーネントに追加のゲートを呼び出す

- [回路図デザイン]ツールバーで、**[呼び出し]** ![呼び出しのアイコン](../images/icon/ecd/invoke.png) をクリックします。

- 使用する集積回路(IC)内の既存のゲートをクリックします。ダイアログが開き、IC で利用可能なゲートが表示されます。

- ダイアログでゲート タイプを選択し、[OK]をクリックします。

- 適切な位置にゲートを配置します。
