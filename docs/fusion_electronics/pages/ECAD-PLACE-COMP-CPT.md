# [コンポーネントを配置]パネル

**[コンポーネントを配置]** パネルはキャンバスの右側にあり、コンポーネント ライブラリへのアクセスを提供します。ライブラリ内の検索、フィルタ設定の変更による検索の絞り込み、および属性、回路図、フットプリント、3D モデルなどのコンポーネントの詳細の確認を行うことができます。パネルから直接ライブラリ マネージャを開くこともできます。

**[コンポーネントを配置]** パネルは、次の作業中に開くことができます。

- 回路図かいろず

- 2D PCB ドキュメント

> **Note:** **重要:** 2D PCB ドキュメントにコンポーネントを配置することは可能ですが、パッドを含むフットプリントは、リンクされた回路図から配置する必要があります。  

![[コンポーネントを配置]パネル](../images/how-to/ecd-plc-comp-info-panel.png)

コンポーネントを選択すると、パネルの下部に **[詳細]** と **[属性]** の 2 つのタブが表示されます。

- **[詳細]** タブには、含まれているコンポーネントの回路図、フットプリント、および 3D モデルのサムネイル イメージが表示されます。

- **[属性]** タブには、選択したコンポーネント属性のリストが表示されます。属性セット(旧称: テクノロジ)が以前に定義されている場合は、それらの属性セット間で切り替えることができます。
