# 電子デザイン部品表をエクスポートする

Fusion から部品表をエクスポートして、デバイスにローカル ファイルとして保存できます。どの項目を選択したかに関係なく、部品表全体がエクスポートされます。

## 部品表をエクスポートする

- [電子デザイン]作業スペースで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) を開きます。

- 部品表の右上にある **[部品表をエクスポート]** ![エクスポートのアイコン](../images/icon/bom-export.png) をクリックします。

- ドロップダウン メニューから、ファイル形式(.csv など)を選択します。

- **[部品表をエクスポート]** ダイアログで、エクスポートするファイル名と場所を選択します。

- **[保存]** をクリックします。

部品表が指定したファイルに保存されます。Fusion 通知センター ![通知アイコン](../images/icon/bom-notification.png) に **[ファイルを開く]** リンクとともに通知が表示されます。
