# デザイン ブロックを挿入する

- [回路図デザイン]ツールバーで、 **[配置] > [デザイン ブロックを挿入]** ![回路図を挿入のアイコン](../images/icon/ecd/insert-design-block.png) を選択します。

- [ファイルを選択]ダイアログで、挿入するデザイン ブロック(*.dbl)に移動して選択します。[開く]をクリックします。  

- [ファイルから貼り付け]ダイアログでは、インポートしたデザイン ブロックのネットの名前を変更したり、既定値を受け入れることができます。[OK]をクリックします。

インポートしたデザイン ブロックが、現在の回路図ドキュメントに追加されます。
