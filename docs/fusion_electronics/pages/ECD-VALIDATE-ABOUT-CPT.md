# 回路図デザインを検証する

電気的ルール チェック(ERC)は、デザインを検証し、修正を考慮すべきエラーや警告を報告します。

既定では、回路図ネットは既定のクラス内にあります。差動ペア(diffpair)クラス内の回路図ネットは、特殊なデザイン要件に準拠するために、より厳格なルールに従います。高電圧または高電流の信号を伝達するネットなど、他のネットには追加のクラスが必要になることがあります。
