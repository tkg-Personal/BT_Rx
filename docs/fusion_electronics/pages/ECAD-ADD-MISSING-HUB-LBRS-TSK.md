# 欠落しているハブ ライブラリをライブラリ マネージャに追加する

**ライブラリ マネージャ**の **[ハブ ライブラリ]** タブで、ハブ ライブラリが見つからない場合があります。これは、そのハブ ライブラリが他のチーム メンバーによって作成された場合、または以前に削除されている場合に発生することがあります。Fusion Electronics では、このタブを使用してライブラリに関する情報を得るため、このタブの同期を維持することが重要です。

## 前提条件

- [電子デザイン ライブラリ マネージャを開く](ECD-OPEN-LIBMGR-TSK.htm)

## 手順

- ライブラリ マネージャで、右上隅にある **[ライブラリを同期]** ![ライブラリを同期アイコン](../images/icon/ecd/lbr-import.png) ボタンを見つけてクリックします。

- 次のいずれかの同期オプションを選択します。

**選択したハブ ライブラリを追加:** 現在のハブから特定のライブラリを追加します。

- **見つからないライブラリのハブをスキャン:** 現在のハブで見つからないライブラリをすべて検出し、自動的に追加します。

## 次のステップ

- **アクティブなライブラリ:** ハブ ライブラリは既定でアクティブになっており、配置パネルに表示されます。

- **ライブラリを非アクティブにする:** 配置パネルで非表示にする必要がある特定のライブラリがある場合は、それらを非アクティブにすることができます。
