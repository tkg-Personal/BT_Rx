# ビア ステッチを配置する

ステッチ ビアを追加するには、次の操作を実行します。

- 2D PCB ドキュメントがアクティブであることを確認します。

- **[PCB 設計]ツールバー**で、 **[デザイン]タブ > [配線セクション] > [ビア ステッチ]** をクリックします。

- ボード全体にステッチング ビアを追加するには、 **[ボード全体をカバー]** を選択します。

- 特定の領域をステッチング ビアで塗りつぶすには、 **[ポリゴン形状を塗り潰し]** を選択し、形状を描画します。

- 必要に応じて、**[ステッチ]**設定と**[ビア]**設定を調整します。

- **[OK]** をクリックします。

ビア ステッチが**[デザイン マネージャ]**の**[信号]**の下に表示されます。
