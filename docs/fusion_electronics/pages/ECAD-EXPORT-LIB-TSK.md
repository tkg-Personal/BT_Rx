# カスタム ライブラリをエクスポートする

電子デザイン ライブラリは、Fusion 形式、または 2 つの EAGLE 形式のいずれかでエクスポートすることができます。

- プロジェクト内のデータ パネルでライブラリをダブルクリックして開きます。

- Fusion ウィンドウ上部の **[アプリケーション]バー**で、 **[ファイル]** ![新規ファイルのアイコン](../images/icon/common/file.png) **> [エクスポート]** をクリックして **[エクスポート]** ダイアログを開きます。

- **[エクスポート]** ダイアログで、ファイル形式を選択し、保存するフォルダの場所を指定します。

- **[エクスポート]** をクリックしてライブラリをエクスポートし、ダイアログを閉じます。
