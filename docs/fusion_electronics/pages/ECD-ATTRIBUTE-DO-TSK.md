# コンポーネント属性を管理する

このコマンドは、ライブラリで定義されているコンポーネント属性に対して機能します。

- 回路図または PCB ドキュメントのツールバーで、 **[属性]** ![属性のアイコン](../images/icon/ecd/attribute.png) をクリックします。

- [属性]ダイアログ ボックスで、フォントと文字サイズの設定を確認し、調整します。

- デザイン内のコンポーネントのハンドルをクリックします。コンポーネント属性のテーブルが開き、名前、値、およびこれらの属性がデザインに表示されるかどうかが示されます。

- 詳細を表示するには、テーブル内の属性をクリックします。

- 必要に応じて、新しい属性を追加したり、既存の属性を変更します。

属性 THERMALLOSS と MAXTEMP は、電子部品の冷却スタディでのみ使用されます。
