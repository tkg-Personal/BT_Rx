# アナログおよびデジタル ソースについて

シミュレーションを設計する場合は、*ngspice-simulation* および *ngspice-digital* ライブラリのパーツを使用して、必要なアナログ電源とデジタル信号のソースを設定します。これらのパーツは、SPICE (Simulation Program with Integrated Circuit Emphasis)と互換性があります。
