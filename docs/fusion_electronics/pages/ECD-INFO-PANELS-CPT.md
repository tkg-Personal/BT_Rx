# 電子デザイン固定パネル

電子デザインで行う作業の多くは、電子デザイン フレームの左側、右側、または下部に表示されるパネルで行います。これらのパネルでは、コンポーネントや回路デザインの詳細の確認、オブジェクトのグループ化、コンポーネントの置換を行うことができます。これらの機能は、コンポーネントを表示して確認するのが困難な、密集したデザインを使用する場合に特に効果的です。

パネルは、ニーズに最も適した形で整理することができます。セクションごとに上下にスタックすることができ、必要に応じて展開したり折りたたんだりすることができます。

![スタック パネル](../images/overview/ecd-panels-stacked.png)

また、パネルを 1 つのタブとして整理し、タブをクリックして切り替えることもできます。

![タブ パネル](../images/overview/ecd-panel-tabs.png)
