# PCB 材料特性を編集する

材料特性は設定されるため、編集する必要はありません。したがって、既定では材料特性の編集はオフになっています。しかし、提供された材料がニーズを満たさない場合もあります。

**重要**: 材料特性を変更してもマテリアル ライブラリは更新されません。これらは、現在のレイヤー スタックでのみ使用できます。材料の変更を保持するには、このレイヤー スタックをプリセットとして保存します。

- 2D PCB ドキュメントの**[デザイン]**ツールバーで、**[レイヤー]** > **[レイヤー スタック管理者]**をクリックします。ダイアログが表示されます。

- [レイヤー スタック]タブで、**[材料特性を編集]**を選択して、ダイアログでマテリアルの編集を有効にします。

 ![材料特性を編集](../images/how-to/ecd-lsm-edit-mtl-props.png)

- [マテリアル]列で、編集するマテリアルを選択します。表のセルに 2 つのボタンが表示されます。

左ボタンをクリックし、リストからマテリアルを選択します。リスト内の列が切り捨てられます。

![[マテリアル]ドロップダウン リスト](../images/how-to/ecd-lsm-edit-mtl-dropdn.png)

- 右ボタンをクリックすると、マテリアルの詳細を表示できるダイアログが表示されます。目的のマテリアルを選択します。

編集したマテリアルは[マテリアル]列に**[ユーザ定義]**と表示され、[ID] (#)列にアスタリスク(*)が表示されます。

![ユーザ定義マテリアル](../images/how-to/ecd-lsm-usr-def-mtl.png)

- 選択したレイヤーとその補完レイヤーのマテリアルをリセットするには、行を右クリックして**[ユーザ マテリアルをリセット]**をクリックします。材料特性は既定値に設定され、編集フォーカスは**[厚さ]**セルに配置されます。
