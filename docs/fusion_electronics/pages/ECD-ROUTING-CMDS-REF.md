# 配線コマンド

このトピックの以下のセクションでは、その他のレイアウト方針について説明します。多くは、コマンド ラインでコマンドを使用する方法が関係しています。

## 回路図なし(推奨されません)

選択したオブジェクトに関する詳細情報が表示されます。クリックしたオブジェクトに応じて、ダイアログで一部のプロパティを変更することができます。

> **Note:** 
注:

 CAM プロセッサでは常にベクトル フォントを使用して製造データを生成することに留意してください。レイアウト内のテキストは常にベクトル フォントで記述することをお勧めします(少なくとも信号画層では)。これにより、表示されるテキストが製造結果にほぼ一致するようになります。

## フットプリントを交換する

レイアウトの開発時に、選択したフットプリント バリアントを別のフットプリント バリアントで置き換える場合は、状況に応じて PACKAGE コマンドまたは REPLACE コマンドを使用できます。

### PACKAGE コマンド

レイアウトと回路図が一致しており、デバイスは複数のフットプリント バリアントで定義されていると仮定します。

コマンド ラインに PACKAGE と入力し、置き換えるフットプリントをクリックするか、またはマウスの右ボタンでフットプリントをクリックし、コンテキスト メニューからフットプリント エントリを選択します。3 番目のバリアントは、[CHANGE]アイコンをクリックして、フットプリント オプションを選択することです。次に、目的のフットプリントを選択し、表示されるダイアログで[OK]をクリックして確認します。

[すべての属性セットを表示]オプションがアクティブの場合、このデバイスで使用できるすべての属性セットのフットプリント バージョンが表示されます。このオプションがアクティブでない場合、選択した属性セットで定義されているフットプリントのみが表示されます。

また、フットプリントを回路図内から交換することもできます。

代替フットプリント バリアントが定義されていないデバイスは、ライブラリ エディタで修正できます。必要に応じてさらにフットプリント バリアントを追加し、新しいライブラリ定義で図面を更新します。

VALUE コマンドを使用して新しい値を指定(Value Off を使用して定義)したデバイスのフットプリント バリアントを変更しても、値は変更されません。103 ページも参照してください。

複数の同一パーツのフットプリント バリアントを変更する場合は、コマンド ラインで変更できます。まず、新しいフットプリント バリアントを取得するすべてのパーツが含まれているグループを定義します。コマンド ラインに

CHANGE PACKAGE 'new-device-name'

と入力し、[Ctrl]を押しながらマウスの右ボタンで図面をクリックします。新しいフットプリント バリアントの名前は、引用符で囲む必要があります。

#### REPLACE コマンド

一貫した回路図とレイアウトのペアREPLACE コマンドを使用すると、コンポーネントを別のコンポーネントで置換することができます。[追加]ダイアログ ウィンドウが開き、新しいパーツを選択できます。次に、回路図またはレイアウトで置換するパーツをクリックします。新旧のデバイスには互換性が必要です。つまり、使用されているゲートおよび接続されているピン/パッドは、名前または座標で一致する必要があります。互換性がない場合、置換はできません。

回路図なしのレイアウト関連付けられた回路図がないレイアウトがある場合は、REPLACE コマンドを使用してフットプリントを交換します。REPLACE を使用すると、[追加]ダイアログに似たウィンドウが開き、デバイスを検索できます。フットプリントを選択したら、レイアウトで置換するパーツをクリックします。

REPLACE コマンドは、レイアウト エディタで 2 つの方法で動作します([パラメータ]ツールバーで選択、または SET コマンドを使用)。
最初のモードでは、パッドまたは SMD の名前が同一であるフットプリントを交換できます。接続領域は任意の位置にすることができます。2 番目のモード(replace_same 座標)では、新しいフットプリント内のパッドまたは SMD が同じ座標(原点を基準)に配置されている必要があります。名前が異なっていてもかまいません。

デバイスの名前と値のテキストは、SMASH を使用してデバイスから分離されていない場合にのみ交換されます。

新しいフットプリントは別のライブラリから取得でき、追加のパッドと SMD を含めることができます。信号に接続されていた古いフットプリントの接続は、新しいフットプリントにも対応して存在する必要があります。この条件が満たされている場合、新しいフットプリントの接続数が少なくなる場合もあります。

## 属性セットを変更する

コンポーネントにさまざまな属性セットが定義されている場合、レイアウト内のデバイスの属性セットをいつでも変更することができます。属性セット オプションを指定して CHANGE コマンドを使用するか、またはコンテキスト メニュー(マウスの右ボタンでフットプリントをクリックすると使用できます)で[属性]コマンドを使用します。この手順は、前述の PACKAGE を使用してフットプリント フォームを交換した手順と同じです。

## 禁止領域を定義する

レイヤー 41 (tRestrict)および 42 (RestrictBottom)に長方形、銅箔ベタ、および円の形の領域を描くのは、オートルータでは禁止されています。これらの領域の内部にある上部画層または下部画層に銅オブジェクトを描画することはできません。これらの領域はデザイン ルール チェックによってテストされ、オートルータによって考慮されます。

レイヤー 43 (RestrictVias)は、オートルータがビアを設定できない制限された領域の描画するためのものです。このような RestrictVias 領域内に手動で配置されたビアは DRC によって検査されないため、エラーとして報告されません。

## 配線 - トラックを手動で配置する

ROUTE コマンドを使用すると、エアワイヤをトラックに変換できます。

ROUTE には、[障害物を回避] (既定)と[障害物を無視]の 2 つの異なるモードがあります。

これらのモードは、ROUTE コマンドのパラメータ ツールバーで選択できます。

#### 障害物を回避

このモードでは、配線エンジンはデザイン ルールに基づいて処理します。配線パスに沿って障害物がある場合、Electronics はトレース用の新しいパスを計算します。

#### 障害物を無視

このモードは、従来の Electronics の配線モードです。この場合、ユーザ自身がデザイン ルールを考慮する必要があります。つまり、クリアランス、ネット クラス、銅、寸法距離、オーバーラップなどを考慮する必要があります。ただし、ユーザは配線パスを完全にコントロールできます。

#### 障害物をプッシュ

このモードでは、配線エンジンはデザイン ルールを考慮しますが、密集した領域でのスペースを確保するために、配線中にトレースを移動します。

#### 配線方法

ROUTE コマンドをアクティブにした後、パラメータ ツールバーで開始画層を選択し、エアワイヤ上でクリックします。これで、トレースの最初のセグメントがマウス カーソルに従うようになりました。ワイヤの幅を確認してください。フィットしていますか?左クリックでセグメントを固定します。

次のセグメントのルーティング レイヤーを変更する場合は、マウスの中央ボタンをクリックします。画層の設定に応じて、画層選択メニューがポップアップ表示されます。または、2 層基板では、代替画層が自動的に選択されます。トレースの終端にビアが表示されます。次のマウス クリックでビアが固定され、次のセグメントは選択した画層でマウス カーソルに従います。画層変更は、スペース バーを押して開始することもできます。このように、その後も使用可能なルーティング レイヤーを順番にたどることができます。また、スペース バーを使用して上から下へ切り替えたり、[Shift] + スペース バーを使用して下から上へ切り替えることもできます。

マウスの右ボタンをクリックすると、マウス カーソルにトラックをアタッチする方法と、その配置方法が変更されます(SET コマンド、Wire_Bend パラメータ)。これらの中には、ワイヤを 90 度円弧または自由定義可能な円弧として使用できるモードがあります。

信号の名前とネット クラスがステータス バーに表示されます。信号線が完全に配置されると、Electronics は信号線の配置と同時に短いビープ音を出して正しい接続が存在することを確認します。

信号名は、コマンド ラインで直接使用できます(例: ROUTE VCC)。Electronics は、現在のマウス カーソルの位置に最も近い信号の接続点からトレースを開始します。

既に配置されているトレース、ビア、パッド、SMD の任意の点で配線を開始できます。

既に配線されているトレースの一部を再配線する場合、使用されなくなったトレース パスは削除されます。[ループ削除]オプションは既定でオンになっています。ROUTE コマンドのパラメータ ツールバーで、オン/オフを切り替えることができます。

信号の配線に十分なスペースがない場合は、他のトラックを MOVE および SPLIT コマンドで再配置するか、トラックのプロパティ(幅、画層)をトレースの[プロパティ]ダイアログまたは CHANGE コマンドで変更できます。

SPLIT コマンドを使用して、トレースに曲げを挿入することができます。

特定の点にめっき済み貫通穴(ビア)を配置する場合、VIA コマンドを使用してこの操作を行うことができます。接続先の信号をビアに割り当てるには、NAME コマンドを使用します。

長さが 0 のエアワイヤ(たとえば、Top 画層から Bottom 画層)は、画層 19 (Unrouted)に十字として描画されます。

同じ信号の別の配線が別の画層に既に存在する場合に同じ位置でワイヤを終了し、同時に[Shift]キーを押すと、Electronics はビアを配置します。それ以外の場合はビアを配置しません。

多層基板をデザインし、ブラインド ビアと埋め込みビアまたはマイクロ ビアを使用する場合は、多層基板に関するセクションで詳細(および VIA コマンドの詳細)を確認してください。

配線のレイアウト中に、Electronics は現在の信号の最も近い点への最短の接続を自動的に計算します。この接続はエアワイヤで表されます。

現在配線されている信号に属しているパッドと SMD には、いわゆる吸着パッド機能があります。パッドの周囲の特定の半径の範囲内では、ワイヤは自動的にパッドの中心にスナップされます。つまり、自動的に計算されたエアワイヤの長さがスナップ長の指定値よりも短くなった場合、ワイヤはパッドまたは SMD の中心点にジャンプします。パッドまたは SMD が現在使用されているグリッドに正確に配置されているかどうかは関係ありません。スナップ点は常に中心点になります。マウス カーソルをこのパッドから離れるように移動して制限値を超えると、すぐにエアワイヤが表示され、配線ワイヤは再度マウス カーソルに従って配線されます。

スナップ長は、[オプション]/[設定]/[その他]メニューで定義します。既定値は 20 mil です。

配線が進むにつれ、すべてのエアワイヤを再計算するために RATSNEST コマンドを頻繁に実行すると便���です。

複雑な基板の場合、[オプション]/[設定]/[その他]メニューで[スナップ長]を調整すると便利な場合があります。

ルーティング レイヤー内のトレースをより見やすくするために、単一レイヤー モードを有効にすることができます。現在選択されているルーティング レイヤー以外の表示レイヤーはグレーで表示されます。このモードは、ROUTE コマンドのパラメータ ツールバーからアクセスできます。クリックして、モードをオフまたはオンにします。

#### トレースの配線を解除する

配置されているトラックの全体または一部、またはビアを元の信号線に変換する場合は、RIPUP を使用します。トラックをクリックすると、最も近い曲げの間で分解されます。信号全体を元に戻す場合は、[リップアップ]をクリックし、コマンド ラインに信号の名前を入力します。複数の名前を同時に入力できます。

RIPUP には複数のモードがあります。

- 既定モード

このモードは、個々のセグメントまたはビアをリップアップする場合に使用します。

- 接続されたすべての銅線

現在の接続上のすべての銅線をリップアップします。複数の画層を通っている場合も含みます。

- 同じ画層上で接続されたすべての銅線

現在の画層に限定して、現在のトレース上の銅線をリップアップします。

- 2 つのコンポーネント間のリップアップ

2 つのコンポーネントを選択すると、両方のコンポーネントで共有されているすべてのトレースがリップアップされます。

- すべての可視銅箔ベタ(RIPUP @; と同じ動作)

コマンド

RIPUP GND VCC +5V

は、3 つの信号 GND、VCC、および +5V をエアワイヤに変換します。

RIPUP ! GND VCC

は、GND と VCC 以外のすべての信号をエアワイヤに変換します。

RIPUP ;

は、エディタに表示されているすべての信号をエアワイヤに変換します。すべてのトラックを実際に含めるには、トラックが描画されているすべての画層を表示する必要があります(DISPLAY)。

#### 円弧を使用したトレース

ワイヤを円弧として使用する場合や、ワイヤの曲げをスムーズにする場合は、ヘルプ機能で MITER コマンドに関するヒントを参照してください。マイタ半径によって、ワイヤのジョイントをマイタ接続する方法が決まります。正の値を指定すると丸くなり、負の値を指定すると直線になります。マイタ半径はいくつかの曲げモード(0、1、3、4。「SET コマンド」を参照)に影響し、SPLIT、ROUTE、LINE、および POLYGON の各コマンドのパラメータ ツールバーに追加で表示されます。

LINE または ROUTE がアクティブな間に、前述のワイヤの曲げ(曲げモード)を[Shift] + 右クリックすることができます。Electronics では 10 種類のモード(0 ～ 9)が認識され、パラメータ ツールバーにアイコンとして表示されます。[Shift]キーを押しながらマウスの右ボタンをクリックすると、選択の方向が反転します。

右クリックすると、補完的なワイヤの曲げの間で切り替えることができます。

マウスの右ボタンで使用できるワイヤの曲げを数種類のみにする場合は、eagle.scr ファイルで定義できます。たとえば、ワイヤの曲げ番号 2、5、6、7 を使用する場合は、次の構文を使用します。

SET WIRE_BEND @ 2 5 6 7 ;

ただし、別の曲げモードを使用する場合は、常にパラメータ ツールバーから曲げモードを選択することができます。

注:

 トラックの配置をオートルータに任せることもできます。

## POLYGON を使用して銅箔面を定義する

Electronics には基板の領域を銅で塗り潰す機能があり、一般的な信号やアースを作成する場合に便利です。POLYGON コマンドを使用して領域の境界を描画するだけです。アウトライン モードでは、銅箔ベタは点線で表示されます。NAME を使用し、次に銅箔ベタの境界をクリックすることで、銅箔ベタに信号名を付けます。すると、この信号を伝達するすべてのオブジェクトが銅箔ベタに接続されます。デザイン ルールで指定されている両方のパッドとビア(任意)が、熱記号で銅箔面に結合されます。この信号を伝達しない要素は、指定された距離を置いて保持されます。

RATSNEST は、レイアウト内のすべての銅箔ベタのサーフェス領域を計算して表示します。次の例のように信号名を使用して RATSNEST を呼び出すと、

RATSNEST GND;

GND 銅箔ベタのみが計算されます。レイアウト内のその他のすべての銅箔ベタは、アウトライン モードでは変更されません。

RIPUP を使用し、次に銅箔ベタの境界をクリックすると、再度内容が非表示になります。レイアウト内に複数の銅箔ベタがあり、アウトライン モードで再度表示する場合は、コマンド ラインに次のように入力します。

RIPUP @;

特定の信号のすべての銅箔ベタをアウトライン モードに切り替えるには、次のように信号名を指定します。

RIPUP @ GND;

構文の詳細については、RIPUP に関するヘルプを参照してください。

銅箔ベタの内容は基板ファイルに保存されません。最初にファイルをロードすると、銅箔ベタの点線のアウトラインのみが表示されます。RATSNEST によってのみ、再度計算および表示されます。

後の段階で、銅箔ベタの描画中、または CHANGE を使用して、パラメータ ツールバーを介してさまざまなオプションを変更できます。

#### 幅

銅箔ベタを描画する線の太さ。可能な限り大きい幅を選択します。これにより、基板が製造用に送信される際に不要なデータ量を回避できます。ワイヤの幅が CAM プロセッサの出力ドライバの解像度よりも低い場合は、警告が表示されます。線幅を細くすると、より複雑な形状の銅箔ベタを使用できます。

#### 注入

塗り潰しのタイプを指定します(領域全体(ソリッド)またはグリッド(ハッチング))。特殊なタイプのカットアウトを使用して、同じ画層内の他のすべての信号銅箔ベタから差し引かれる銅箔ベタを定義できます。内部信号画層の銅箔ベタのカットアウト(制限領域)に適しています。

#### ランク

重なっている銅箔ベタで短絡が作成されてはなりません。したがって、ランクを使用して、どの銅箔ベタを他から差し引くかを決めることができます。ランクが 1 の銅箔ベタはレイアウト エディタで最も優先度が高く、レイアウト内に描画される他の銅箔ベタがそこから差し引かれることはありませんが、ランクが 6 の銅箔ベタは最も優先度が低くなります。上位ランクとのオーバーラップが発生すると、ランクが 6 の銅箔ベタから適切な領域が切り取られます。同じランクの銅箔ベタは DRC によって比較されます。ランク プロパティは、さまざまな信号を持つ銅箔ベタに対してのみ機能します。同じ信号名を持つ銅箔ベタ オーバーラップの場合は効果がありません。これらは互いに重なって描画されます。信号に割り当てられていない、パッケージ エディタで作成された銅箔ベタは、他のすべての銅箔ベタから差し引かれます。使用できるランク パラメータはありません。

#### [間隔]

[注入]でオプション[ハッチング]が選択されている場合、この値によってグリッド線の間隔が決まります。

#### 分離

Dimension、tRestrict、bRestrict レイヤーにあり、銅箔ベタの信号およびオブジェクトに含まれていない他のすべての銅オブジェクトに関してその銅箔ベタが維持する必要がある値を定義します。デザイン ルールまたはネット クラスで特殊な信号に対してより高い値が定義されている場合は、より高い値が適用されます。

異なるランクを持つ銅箔ベタの場合、[選択表示]は常に銅箔ベタのアウトライン モードで表示される描画輪郭を参照します。これは、計算された銅箔ベタにより別の輪郭が取得された場合(たとえば、銅箔ベタより優先されるワイヤが原因)も該当します。実際のクリアランスは、指定された[選択表示]値よりも大きくなる場合があります。

#### 熱

銅箔ベタ内のパッドが、熱記号を介して接続されるか、銅箔面に完全に接続されるかを決定します。これはビアにも適用されます(デザイン ルールでそのオプションがアクティブになっていることを想定)。

熱コネクタの幅は、パッドのドリル直径の半分として計算されます。幅は、最小値がワイヤ幅、最大値が銅箔ベタのワイヤ幅の 2 倍という制限内にある必要があります。

熱コネクタの長さは、[デザイン ルール]の[供給]タブの[熱分離]の値によって定義されます。

> **Note:** 
注:

 銅箔ベタの幅を細くしすぎないでください。細すぎると、熱コネクタが電流負荷を処理できません。これは、基板のボトルネックにもあてはまります。銅箔ベタのワイヤの幅によって、銅の領域の可能な限り小さい幅が決まります。

#### 孤立

銅箔ベタの信号に電気的に接続されていない領域(島)を銅箔ベタに含めることができるかどうかを決定します。[孤立]がオフの場合、このような接続されていない領域は描画されません。

> **Note:** 
注:

 銅箔ベタを描画する際は、どの位置でもアウトラインが複数回描画(オーバーラップ)されないように、また、銅箔ベタのアウトラインが自己交差しないように注意してください。この場合、電子設計は領域の内容を計算できません。エラー メッセージ「*'Signalname' に無効なポリゴンが含まれています!*」が表示され、RATSNEST コマンドが中止されます。このメッセージが表示された場合は、ポリゴンのアウトラインを修正する必要があります。修正しないと、CAM プロセッサで製造データを作成できません。CAM プロセッサは、出力を生成する前に、レイアウト内の銅箔ベタを自動的に計算します。RATSNEST で計算した後も銅箔ベタがアウトライン モードのままである場合は、幅、分離、孤立のパラメータ、および銅箔ベタの名前を確認する必要があります。銅箔ベタの塗り潰しが、信号と接続する必要のあるいずれかのオブジェクトに到達できない可能性があります。NAME コマンドで銅箔ベタの名前を変更すると、別の信号に接続されます。
