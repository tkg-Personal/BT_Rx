# 電子デザイン ライブラリ マネージャ

**電子デザイン ライブラリ マネージャ**は、Fusion 電子デザインで利用可能なすべてのコンポーネント ライブラリを参照、検索、および管理するための中心的な場所です。必要なライブラリだけをアクティブ化することで**配置パネル**を整理し、ワークフローに合わせたパーソナライズされた作業スペースを構築できます。

## メリット

ライブラリ マネージャを使用すると、次の方法で効率的に作業できます。

- **ワークフローの整理**: 現在のプロジェクトに必要なライブラリのみをアクティブ化します。

- **コンポーネントの迅速な検索**: 1 か所から利用可能なすべてのライブラリを検索できます。

- **ライブラリの最新化**: リモートで変更が利用可能な場合は、パブリックまたはプライベート ライブラリを更新します。

- **チーム リソースの管理**: チーム メンバーと共有されているハブ ライブラリにアクセスします。

## ライブラリ タイプ

ライブラリ マネージャは、ライブラリを次の 3 つの主なカテゴリに分類します。

- **ハブ ライブラリ**: Fusion ハブを介してチーム メンバーと共有されるライブラリです。これらのライブラリを使用すると、チーム全員が同じコンポーネントにアクセスでき、コラボレーションが可能になります。

- **パブリック ライブラリ**: オートデスクまたはサードパーティ プロバイダによって提供されるライブラリです。これらは、デザインで使用できる標準コンポーネントを含む読み取り専用ライブラリです。

- **プライベート ライブラリ**: 個人の Library.io 管理フォルダまたはローカル ディスクに保存されるライブラリです。Library.io のコンポーネントは、Library.io Web サイトを介して共有しない限り、チーム メンバーはアクセスできません。ローカル ディスクに保存されているコンポーネントは、自分だけがアクセスできます。

## 主な機能

- **検索とフィルタ**: ライブラリ名またはコンポーネント名でライブラリを検索できます。フィルタを適用して、タイプやステータスで結果を絞り込み、必要なものをすばやく見つけます。

- **ライブラリのアクティブ化**: ライブラリをアクティブ化して、配置パネルで使用できるようにします。

- **コンテンツ プレビュー**: 任意のライブラリを選択し、アクティブ化する前に詳細パネルですべてのコンポーネントとパッケージを含む内容を確認します。

- **更新とメンテナンス**: パブリックまたはプライベート ライブラリにリモート変更がある場合に、ライブラリ マネージャから直接更新できます。使用していないライブラリを表示から削除して、作業スペースを整理することもできます。

## 重要事項

- **ハブ ライブラリの欠落**: チーム メンバーが作成したハブ ライブラリは自動的に表示されない場合があります。[ハブをスキャン](ECAD-ADD-MISSING-HUB-LBRS-TSK.htm)して、表示されていないライブラリをすべて読み込みます。

- **個人用ビュー**: ライブラリのアクティブ化は自分の配置パネルにのみ影響します。他のチーム メンバーの設定には影響しません。
