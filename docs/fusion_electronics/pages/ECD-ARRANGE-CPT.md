# オブジェクトの配置

**[配置]** コマンドは、選択したオブジェクトを XY グリッドまたは円形状に移動します。[配置]は、回路図と 2D PCB の両方のコンテキスト作業スペースで使用できます。ダイアログでは、最後に使用された値が記憶されます(間隔を除く)。値を変更するたびに、[**Enter**]を押して新しい値の効果をプレビューしてください。
