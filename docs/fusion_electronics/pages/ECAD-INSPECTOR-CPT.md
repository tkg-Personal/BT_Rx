# 電子デザインの検査パネル

既定では、 **[検査]** 固定パネルはデザイン ペインの右側に配置され、選択したオブジェクトのプロパティと属性を表示します。複数のオブジェクトが選択されている場合は、それらのオブジェクトに共通するプロパティが表示されます。**[検査]** パネルの情報は、キャンバスで選択したオブジェクトに基づいて入力されます。

次のドキュメントで作業しているときに、 **[検査]** パネルにアクセスできます。

- 回路図

- 2D PCB

- ライブラリ

![検査](../images/overview/ecd-inspector-panel.png)

*図 1: 電子デザイン ドキュメントの **[検査]** パネルの既定の位置。それぞれ 1. プロパティ、2. 属性、3. クイック アクションが表示されています。*

## 使用例

**[検査]** 固定パネルは、次の用途に使用します。

- コンポーネントを移動、削除、および回転する

- コンポーネント グループの値を変更して、一貫性を持たせる

- コンポーネント グループを別のコンポーネントに置き換える

- モジュールのサイズを変更する

- 選択したオブジェクトのプロパティを確認する

- ネット クラスと信号の色をカスタマイズする

- 基板の形状を調整する

たとえば、複数のオブジェクトが同じ Y 値の位置にある場合、 Y 値を変更することでそれらをグループとして移動することができます。**[クイック アクション]** ボタンは選択内容によって異なりますが、コンポーネントの削除、属性の定義、ライブラリ内の別のコンポーネントへの置換などの一般的な操作が容易になります。
