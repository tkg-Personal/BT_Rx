# 部品表を確認する

- [回路図ドキュメント]ツールバーで、**[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) をクリックします。

- 左下で、**[リスト タイプ]**を変更してすべてのパーツを個別に表示するか、[値]に変更して同じ値を持つコンポーネントの数を表示できます。すべてのユーザ定義属性を表示するには、**[属性を一覧表示]**チェックボックスをオンにしておく必要があります。

- 左上では、最も完全なバリアント('')と、値が異なるコンポーネントや削除されたコンポーネントがある他のバリアントとの間で、表示を切り替えることができます。
