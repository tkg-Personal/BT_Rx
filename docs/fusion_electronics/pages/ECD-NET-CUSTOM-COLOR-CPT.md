# ネットと信号のカスタム カラー

カスタム カラー機能を使用すると、ネットや信号を簡単に追跡できます。この機能を使用すると、特定の色をネットと信号に割り当てることができます。これにより、特に密集した設計の場合や、重要なネットを扱う場合に、それらを簡単に追跡して区別できるようになります。

[検査]パネルで信号またはネット クラスごとに色を選択して、設計をより明確にすることができます。この色分けされたビューは、信号とネット クラスを区別するのに役立ちます。また、既定のビューに戻すこともできます。
