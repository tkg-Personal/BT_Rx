# 違反のコントロール

回路基板ドキュメントが開いている場合、違反のコントロールのメニューがナビゲーション バーに追加されます。

![違反メニュー](../images/concepts/violators-menu.png)

これらのコントロールは、トレースしているルートと、他の近接するルートやビアとの間の相互作用のためのオプションを提供します。

## 違反を無視

新しいルートが既存のルートと交差します。

## 違反をプッシュ

新しいルートが既存のルートを横にずらしますが、接続は維持されます。

![違反をプッシュ](../images/concepts/push-violators.gif)

## 違反を回避

既存のルートやコンポーネントの周囲に新しいルートを配置できますが、これらをずらすことはできません。

![違反を回避](../images/concepts/walkaround-violators.gif)

## ビアをプッシュ

新しいルートがビアを横にずらすことができます。
