# フェーズ プローブを追加する

- [回路図デザイン]ツールバーで、 **[フェーズ プローブ]** ![フェーズ プローブのアイコン](../images/icon/ecd/phase-probe.png) をクリックします。[フェーズ プローブ]ダイアログが開き、プローブ ラベルの配置前または配置中にラベル プロパティを変更するためのオプションが表示されます。

- クリックして、回路内の正確な位置にプローブを配置します。

- カーソルを空白の領域にドラッグし、クリックしてラベルを配置します。

- 別のプローブを配置する場合は、その位置をクリックしてラベルを配置します。

- フェーズ プローブを削除する場合は、プローブのラベルの終端をクリックし、右クリックして[削除]を選択します。この削除モードでは、他のフェーズ プローブもクリックして削除できます。
