# 固定パネルをドッキング、ドッキング解除、および移動する

[電子デザイン]作業スペースで固定パネルをドッキングまたはドッキング解除し、視認性を向上させるために他の場所に移動することができます。

- パネルのタイトル バーをクリック アンド ドラッグし、キャンバス、別のパネル、または別のモニタにドロップします。

- ドッキングされていないパネルを別のパネルまたはキャンバスのエッジにクリック アンド ドラッグしてドッキングします。

- パネルのタイトル バーを右クリックし、コンテキスト メニューからパネルをドッキングする位置を選択します。
