# 回路図デザインに空白のシートを追加する

大規模な回路図デザインで作業する場合、デザイン プロセスを簡素化するためにシートを追加することができます。

- [電子デザイン回路図]ドキュメントで、ウィンドウの下部にある **[シート] ** ![シートのアイコン](../images/icon/ecd/sheet.png) をクリックして、 **[シート]** 固定パネルを開きます。

- **[シート]** パネルで右クリックし、コンテキスト メニューから **[新規作成]** を選択します。

新しい空白のシートが **[シート]** パネルに追加されます。

## シートを削除する

- [電子デザイン回路図]ドキュメントで、ウィンドウの下部にある **[シート] ** ![シートのアイコン](../images/icon/ecd/sheet.png) をクリックして、 **[シート]** 固定パネルを開きます。

- **[シート]** パネルで、削除するシートをクリックして選択します。

 選択したシートは青色でハイライト表示されます。

- 選択したシートの上にマウス カーソルを置き、右クリックしてコンテキスト メニューから **[削除]** を選択します。

- **[はい]** をクリックして、シートの削除を確認します。

シートが **[シート]** パネルから削除されます。
