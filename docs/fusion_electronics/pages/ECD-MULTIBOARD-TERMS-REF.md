# 多層基板の用語と例

## コア

片側または両側が銅でコーティングされている非柔軟性のカーネル。レイヤー設定では * で表されます。たとえば、5*12 の場合、レイヤー 5 および 12 が基板のコアです。

## プリプレグ

内側レイヤーおよび外側レイヤーを互いに加圧するために、多層基板の製造工程で使用される柔軟な接着層または分離層。レイヤー設定では + で表されます。1+2 は、レイヤー 1 がプリプレグであり、レイヤー 2 と組み合わせられることを示します。

## レイヤー スタック

現在の製造手順でまとめて処理されるコアとプリプレグで構成される任意の数のレイヤーのパック。

## 埋め込みビア

このビアの製造工程は、貫通(標準)ビアと変わりません。現在のレイヤー スタックは、完全にドリルで貫通されます。次の製造手順では、現在のレイヤー スタック上のコアおよびプリプレグをさらに加圧して、既にドリル加工されたビアをおおう(埋め込む)ことができます。完了した基板にビアが表示されていない場合、そのビアを埋め込みビアと言います。これは括弧で表されます。たとえば、1+(2*15)+16 の場合、レイヤー 2 から 15 までが埋め込みビアです。

## ブラインド ビア

ブラインド ビアは、外側レイヤーを任意の内側レイヤーと接続しますが、すべての銅箔層を通るわけでありません。ブラインド ビアの特殊性は、製造工程にあります。現在のレイヤー スタックは、ドリルで貫通されません。ドリル穴には、互いに接続できるようにする必要があるレイヤーの数に応じて特定の深さがあります。ブラインド ビアは、ドリル直径に対する深さの特定の比率に従う必要があります。この情報については、ご使用の基板ハウスを確認してください。この比率は、[サイズ]タブで[最小ブラインド ビア比率]として定義する必要があります。これは、角括弧と、その角括弧の前または後にコロンでマークしたターゲット レイヤーで表されます。例 [3:1+2+3*14+15+16] では、レイヤー 1 から 3 へのブラインド ビアが可能です。ブラインド ビアは、定義よりも短くなる場合があります。この例では、レイヤー 1 から 2 へのビアを使用することができます。オートルータでは、さらに短いブラインド ビアを使用することもできます。

## マイクロ ビア

マイクロ ビアは、ブラインド ビアの特殊なケースです。最大深さが 1 つのレイヤーで、ドリル直径が非常に小さくなっています。

## ビアを表示する

長さと形状が異なるビアで作業している場合は、画層 18 (Vias)の画層色を背景色に設定するのが適切です([DISPLAY]メニューの[変更]の[色])。そうすることにより、レイヤー関係を認識することができます。

## レイヤー設定

コアとプリプレグを組み合わせることで、多くのバリアントを実現することができます。次のセクションでは、いくつか例を挙げてレイヤー設定の機能を示します。この段落全体をお読みください。たとえば、4 層基板をデザインする場合でも、他のすべての例も参照して理解を深めることをお勧めします。

レイヤー スタックを定義する場合は、常にサーフェス レイヤーから内側レイヤー方向に行います。たとえば、4 層基板の場合、正しいスタックアップではレイヤー 1、2、15、16 を使用します。1、2、3、16 は使用しません。常に外側レイヤーから内側レイヤーに向かって作業します。

## 4 層基板**

### 例 1

- レイヤー 1、2、15、および 16 を使用します。

- 基板構造: 内側にコア 1 つ、外側にプリプレグ。

- 接続: 1 ～ 2 (ブラインド ビア)、2 ～ 15 (埋め込みビア)、および 1 ～ 16 (貫通ビア)

**設定式**

[2:(1+(2*15)+16)]

**説明**

式 | 
説明 |

 |

|

ブラインド ビアでは、ドリル直径に対するビア深さの特定の比率が保持される必要があります。このためには、レイヤーの厚さの値を指定する必要があります。これらの値は、基板ハウスによって指定されます。いずれにしても、レイアウトを開始する場合は、事前に基板ハウスを確認してください。

次のイメージに示すように、[銅]フィールドに値(銅箔レイヤーの厚さ)を入力し、[分離]フィールドに値(分離層の厚さ)を入力します。[銅]フィールドと[分離]フィールドの下に、基板の厚さの合計が示されます。

## 6 層基板

### 例 3

- レイヤー 1、2、3、14、15、および 16 を使用します。

- 基板構造: コア 2 つ、外側にプリプレグ。

- 接続: 2 ～ 3、14 ～ 15 (埋め込みビア)、1 ～ 16 (貫通ビア)

**設定式**

(1+(2*3)+(14*15)+16)

**説明**

| 
 | 
 | 
 |

| 
(2*3)+(14*15) | 
埋め込みビアがある 2 つのコアをまとめて加圧します。 | 
 |

| 
1+(2*3)+(14*15)+16 | 
この画層スタックは、プリプレグで分離されている外側画層 1 および 16 でおおわれています。 | 
 |

| 
(1+(2*3)+(14*15)+16) | 
括弧内の式全体で 1 ～ 16 の貫通ビアを定義します。 | 
 |

![レイヤーの例](../images/reference/multi-layer-eg03.png)

これらの例で銅箔層と分離層の厚さに使用している値は架空のものです。許可される値については、基板ハウスを確認してください。

### 例 4

- レイヤー 1、2、3、14、15、および 16 を使用します。

- 基板構造: コア 1 つ、両側に 2 つのプリプレグ。

- 接続: 3 ～ 14 (埋め込みビア)、2 ～ 14 (内側レイヤー スタック内のブラインド ビア)、1 ～ 16 (貫通ビア)

**設定式**

(1+[14:2+(3*14)+15]+16)

**説明**

| 
 | 
 | 
 |

| 
2+(3*14)+15 | 
埋め込みビアがあるコア。両側に 1 つのプリプレグ。 | 
 |

| 
[14:2+(3*14)+15] | 
画層 2 から 4 までのブラインド ビア。 | 
 |

| 
1+[14:2+(3*14)+15]+16 | 
この画層スタック上で、両側のプリプレグに加圧します。 | 
 |

| 
(1+[14:2+(3*14)+15]+16) | 
括弧により、貫通ビア 1 ～ 16 が可能になります。 | 
 |

## 8 層基板

### 例 5

- レイヤー 1、2、3、4、13、14、15、および 16 を使用します。

- 基板構造: コア 3 つ、外側にプリプレグ。

- 接続: 1 ～ 3、14 ～ 16 (ブラインド ビア)、2 ～ 3、4 ～ 13、14 ～ 15 (埋め込みビア)、1 ～ 16 (貫通ビア)。

**設定式**

[3:(1+(2*3)+(4*13)+(14*15)+16):14]

**説明**

| 
 | 
 | 
 |

| 
(2*3)+(4*13)+(14*15) | 
3 つのコア(それぞれに埋め込みビアがあります)は、まとめて加圧され、プリプレグで分離されます。 | 
 |

| 
1+(2*3)+(4*13)+(14*15)+16 | 
プリプレグで分離されている外側の銅箔層 1 および 16 は、このレイヤー スタックに加圧されます。 | 
 |

| 
(1+(2*3)+(4*13)+(14*15)+16) | 
括弧により、貫通ビア 1 ～ 16 が可能になります。 | 
 |

| 
[3:(1+(2*3)+(4*13)+(14*15)+16):14] | 
ブラインド ビア 1 ～ 3 および 16 ～ 14。 | 
 |
