# SVG の挿入

# 電子デザイン ドキュメントに SVG ファイルを挿入する

**[SVG を挿入]** は画像をインポートする簡単な方法です。会社のロゴやその他の重要な記号などの SVG ファイルを、回路図ドキュメントや PCB ドキュメントに挿入することができます。SVG ファイルをデザイン エディタにインポートすることで、繊細なシルクスクリーン作業や正確なロゴ配置に適した正確なベクトル ベースのグラフィックスを実現できます。**[SVG を挿入]** は、ライブラリ ドキュメントの記号エディタおよびフットプリント エディタでも使用できます。

SVG イメージをインポートすると、曲線がターゲット画層上に配置されます。画像は、**グループ**としてキャンバスに配置され、 **[デザイン マネージャ]** から全体をドラッグして移動することができます。次の操作を実行できます。

- グループが配置されている画層を変更する

- グループを回転、ミラー化、または削除する

- グループをグループ解除する

## 既定の設定

- **ターゲット画層**:

21 PCB のシルクスクリーン上部

- 97 回路図の情報

- **ストロークのみ**: オフにすると、元のイメージ ファイルの閉じたプロファイルがインポートされた画像で塗り潰されます。

- **グループ名**: 拡張子を除いたファイル名です。この名前は編集できます。

- **注釈を表示**: 回路図内のこれらのグループではオフになります。

- **尺度設定**: 既定のサイズは 1 です。画像が PCB ボードのアウトラインよりも大きい場合は、ボードのアウトライン内に収まるように尺度設定されます。

## 既知の制限事項

**複数の色**: 複数の色を使用した SVG はサポートされておらず、単一色としてインポートされる場合があります。

**背景**: インポートする前に背景ボックスを削除します。

**CSS スタイル**: CSS で定義されたスタイルはサポートされていません。

**データの精度**: このアプリケーションには高い精度(通常 0.00001)が要求されるため、塗り潰された形状が一部塗り潰されていないか、まったく塗り潰されていない状態でインポートされる場合があります。

**テキスト**: インポートする前に、ロゴに使用されるテキストを形状に変換します。

**パフォーマンス**: サイズが大きい画像や複雑な画像は、移動中にパフォーマンスの問題が発生する可能性があります。
