# 移動

- [デザイン]ツールバーで、**[移動]** ![移動のアイコン](../images/icon/move-icon.png) をクリックします。

- オブジェクトの原点を左クリックすると、オブジェクトがカーソルにアタッチされます。別の位置に移動するか、矢印キーを使用してオブジェクトを移動します。注: 代替グリッド設定ごとに移動するには、[Alt]キーと矢印キーを使用します。

- 必要に応じて、右クリックして、オブジェクトを反時計回りに 90 度回転します。追加の右クリックでオブジェクトを回転し続けます。

- コンポーネントを配置する位置にカーソルを移動してクリックします。

- [移動]コマンドがアクティブな状態で、別のオブジェクトの原点をクリックして移動します。

- 終了したら、[Esc]を押します。
