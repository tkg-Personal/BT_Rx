# ピン配列を追加する

集積回路(IC)記号またはチップ記号に一連のピンを追加します。

- ライブラリが開いた状態で、**[新しい記号]**をクリックし、最初から描画するか、または既存の記号をベースとしてインポートするかを選択します。

- [ライブラリ記号]ツールバーで、 **[配置] > [ピン配列]** ![ピン配列のアイコン](../images/icon/ecd/pin-array.png) をクリックします。

- **[ピンを配置]**ダイアログで、ベース名(通常は単一の文字)を指定します。ハイフンはあってもなくてもかまいません。

- チップのどちら側にピンを配置するかを選択します。

- ピン数と開始インデックス(通常は 1)を指定します。

- チップの長方形のアウトラインを描画するかどうか、および >NAME ラベルと >VALUE ラベルを含めるかどうかを選択します。

- [OK]をクリックして、キャンバスにチップを配置します。
