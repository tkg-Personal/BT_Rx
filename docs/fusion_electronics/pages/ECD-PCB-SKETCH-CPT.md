# PCB スケッチ

**PCB スケッチ**は、 **[デザイン]** 作業スペースのスケッチに似ていますが、その機能は、機械アセンブリに PCB を統合するためのものに限定されています。

**3D PCB では、次のことができます。**

- アウトライン スケッチを編集します。

- PCB スケッチを作成し、下書き線を描画します。

- ブラウザ ノードからスケッチの表示設定をコントロールします。

- **[2D PCB にプッシュ]** ![3D スケッチを作成のアイコン](../images/icon/ecd/push-2d-pcb.png) を使用して、2D PCB 内の構築ジオメトリをレイアウトのために参照します。

**2D PCB では、次のことができます。**

- **[表示]** メニューからスケッチの表示設定をコントロールします。

![PCB スケッチの表示設定](../images/how-to/ecd-pcb-sketch-visibility.png)

- **デザイン マネージャ**を使用してスケッチを検索します。

![PCB スケッチ デザイン マネージャ](../images/how-to/ecd-dm-vw-pcbsketch.png)

- **[検査]** を使用して、スケッチ要素のプロパティを表示します。

![デザイン マネージャ PCB スケッチ](../images/how-to/ecd-dm-pcb-sketch.gif)

## ヒント

- 3D モデルのジオメトリをスケッチに参照するには、 **[プロジェクト]** (P)ツールを使用します。

 ![3D PCB スケッチ プロジェクト](../images/how-to/ecd-3d-pcb-sketch-tools.png)

- 機械設計で作業し、モデル参照を使用して PCB ボード アウトラインを作成する場合は、通常のスケッチ ジオメトリを使用します。
