# カスタム 3D モデルを使用する

**[カスタム 3D モデルを追加]** を使用して、カスタム 3D モデルを回路図またはボードのコンポーネントまたは要素に割り当てます。3D モデルへの既存の参照がある場合は、無視されてカスタム 3D モデルが優先されます。

## カスタム 3D モデルを追加する

- 回路図デザインまたは PCB 設計の[編集]ツールバーで、 **[カスタム 3D モデルを追加]** ![カスタム 3D モデルを追加のアイコン](../images/icon/add-custom-3d-icon-16.png) をクリックします。

- キャンバスで、コンポーネントのハンドルをクリックします。

カスタム 3D モデルがアクティブなドキュメントのみに追加されることを通知する警告メッセージが表示されます。ライブラリには追加されません。[OK]をクリックして、警告メッセージを閉じます。

- 目的の場所にコンポーネントを保存します。

Fusion で代替 3D モデルを作成したり、STEP ファイルをインポートしたり、3D パッケージ ジェネレータを使用してカスタム モデルを作成することができます。この例では、3D パッケージ ジェネレータを使用します。

- **[3D パッケージ ジェネレータ]**ツールをクリックし、リストをスクロールして使用するものを選択します。

- 要件に基づいて仕様やその他の詳細を調整し、[追加]をクリックします。

- **[完了]** をクリックしてモデルを保存し、回路図または PCB ドキュメントに戻ります。

## カスタム 3D モデルを編集する

- 回路図デザインまたは PCB 設計の[編集]ツールバーで、 **[カスタム 3D モデルを編集]** ![カスタム 3D モデルを編集のアイコン](../images/icon/edit-custom-3d-icon-16.png) をクリックします。

- キャンバスで、カスタム 3D モデルが割り当てられているコンポーネントのハンドルをクリックします。

- **[3D パッケージ ジェネレータ]**ツールをクリックし、リストをスクロールして使用するものを選択します。

- 要件に基づいて仕様やその他の詳細を調整し、[更新]をクリックします。

- [完了]をクリックしてモデルを保存し、回路図または PCB ドキュメントに戻ります。

## カスタム 3D モデルを削除する

カスタム 3D モデル定義を削除して、ライブラリ定義を復元します。

- 回路図デザインまたは PCB 設計の[編集]ツールバーで、 **[カスタム 3D モデルを削除]** ![カスタム 3D モデルを削除のアイコン](../images/icon/remove-custom-3d-icon-16.png) をクリックします。

- キャンバスで、カスタム 3D モデルが割り当てられているコンポーネントのハンドルをクリックします。関連付けられている 3D モデルが削除されます。
