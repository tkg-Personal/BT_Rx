# オブジェクト情報を表示する

**[表示]** コマンドは、キャンバス上で選択したオブジェクトをハイライト表示し、そのプロパティを表示します。

- 電子デザイン回路図ドキュメントの **[デザイン]** ツールバーで、 **[ビュー] > [表示]** ![表示のアイコン](../images/icon/ecd/show.png) をクリックします。

ツールバーで[表示]ツールがハイライト表示されます。

- キャンバスで、表示するオブジェクトをクリックします。

オブジェクトのプロパティを含む情報ボックスが表示されます。

- 別のオブジェクトを表示するには、キャンバスでそのオブジェクトをクリックします。

- キーボードで、[Esc]を押して **[表示]** ツールを終了します。
