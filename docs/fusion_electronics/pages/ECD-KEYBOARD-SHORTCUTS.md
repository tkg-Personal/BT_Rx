# Electronics のキーボード ショートカットを割り当てる

EAGLE の ASSIGN コマンドと同様に、Electronics のコマンドを組み合わせるカスタム キーボード ショートカットを追加します。**[ショートカット]**は、回路図ドキュメントと 2D PCB ドキュメント両方の[デザイン]タブで使用できます。

キーをショートカットに割り当てると、入力したキーの組み合わせが Fusion によって予約されている場合、または別のショートカット定義と競合する場合に、メッセージが表示されます。
