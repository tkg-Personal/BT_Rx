# ラベルの外観を調整する

[ラベル]をクリックすると、[ラベル]ダイアログを使用して、次のラベル設定を変更できます。

- ネットが別の回路図シートに続き、ラベルでもう一方のシートの接続点の位置を記録する場合は、[外部参照]をオンにします。

- [回転]は、右クリックごとに角度の回転(反時計回りで 0、90、180、または 270 度)を設定します。

- [ミラー]では、ラベル ハンドルがラベル文字の先頭にあるか、末尾にあるかを設定します。

- [サイズ]は、ラベル文字のポイント サイズまたは高さをコントロールします。

- [比率]はベクトル フォントにのみ適用され、文字の太さを文字サイズのパーセンテージで設定します。

- [フォント]では、ベクトル、プロポーショナル、固定、カスタムのフォントを選択できます。

- 前のフィールドでカスタム フォントを選択すると、フォント ファミリでは多様なフォントが使用できます。
