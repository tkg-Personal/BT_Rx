# ODB++ ファイルをエクスポートする

- キャンバスで回路基板を開き、[製造]ツールバーで**[ODB++ をエクスポート]** ![ODB++ をエクスポート アイコン](../images/icon/ecd/ecd-export-odb.png) をクリックします。

- [ODB++ 出力をエクスポート]ダイアログで、次の内容を指定します。

出力のファイル名。

- ファイル タイプ(.tgz、.zip、または.tar)。

- ファイルの保存場所。

- **[OK]**をクリックします。

## Ansys ODB++ 出力要件

エクスポートしないもの:

- シルクスクリーン

- はんだマスク

- 貼り付けマスク

- ロード エラーの原因となる可能性があるため、True Type フォントのテキストは使用しない

エクスポート:

- レイヤー

信号

- 基板材質

- ドリル

- ルート レイヤー

- 次のような ECAD データ

コンポーネント レイヤー(トップ 1、ボトム 16)

- EDA/データ(ネットおよびフットプリント)

- CAD ネットリスト(netlists/cadnet/netlist)

- 銅層の厚さ

'.copper_weight' 属性は 'attrlist' ファイルに書き出され、計測単位は出力単位に応じて μm (ミクロン)または oz/sqft になります。

- 絶縁層の厚さ

'.layer_dielectric' 属性は 'attrlist' ファイルに書き出され、計測単位は出力単位に応じて mm またはインチになります。

## カスタム CAMJOB ODB++ テンプレートを作成する

- CAM プロセッサ インタフェースを使用して、既定の ODB++ テンプレートから CAMJOB テンプレートを作成します。

- 調整を行ったら、上記のガイドラインに従って、ODB++ 出力ファイルが保存されるフォルダに保存します。

- その後は、既定のテンプレートではなく ODB++ CAMJOB テンプレートを使用して、ODB++ 出力を生成します。
