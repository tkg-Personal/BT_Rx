# スケッチから PCB を作成する

この機能により、プリント基板(PCB)のプロファイル スケッチ、3D 図面、および 2D 基板デザインの間にリンクを作成することで、電子デザイン プロセスがさらに自動化されます。スケッチまたは基板デザインに対する変更を 3D モデルに直接プッシュすることができます。

## サンプル ワークフロー

- Fusion の**[デザイン]**ワークスペースで、PCB の 2D スケッチを正確な寸法とジオメトリで作成します。**[スケッチ パレット]**で、**[スケッチを終了]**をクリックします。

![2d-sketch](../images/how-to/2d-sketch.png)

- ツールバーで、**[作成] > [PCB を作成] > [関連付けのある PCB を作成]**をクリックします。

![PCB を派生](../images/how-to/ecd-create-pcb.png)

関連付けのある PCB はスケッチにリンクされ、スケッチが変更されると更新されます。または、元のスケッチにリンクされていない独立した PCB を作成することもできます。

新しい 3D 図面が開き、既定の厚さの基板が表示されます。

![3D PCB](../images/how-to/derived-board.png)

- 3D 図面ツールバーで、**[修正] > [2D PCB にプッシュ]**をクリックします。

![2D にリンク](../images/how-to/ecd-3d-pcb-link-to-2d.png)

新しい 2D PCB 図面が作成され、これを使用してライブラリから基板にコンポーネントを追加できます。

![リンクされた 2D PCB](../images/how-to/2d-pcb-linked.png)

既存の 2D PCB デザインがある場合、関連するコマンド**[修正] > [2D PCB にリンク]**を使用できます。

- 2D PCB 図面のツールバーで、**[切り替え] > [3D PCB にプッシュ]**をクリックして、2D デザインが 3D パッケージでどのように表示されるかを確認できます。

これらのリンクが作成されたら、必要に応じてスケッチを更新し、2D PCB にコンポーネントを追加できます。3D PCB が他のドキュメントのいずれかと同期していないことを示す黄色の三角形の警告が表示された場合は、警告をクリックして同期を復元します。

3D モデルからスケッチまたはボードを編集するには、タイムラインでボード ノードを右クリックし、編集する項目を選択します。

![タイムラインから編集](../images/how-to/ecd-edit-from-timeline.png)
