# ボード形状に円弧を追加する

- [PCB 設計]ツールバーで、 **[アウトライン円弧]** ![アウトライン円弧のアイコン](../images/icon/ecd/outline-arc.png) をクリックします。

- 表示されるダイアログで、線幅、円弧の方向、および線分キャップの形状の調整を検討します。

- キャンバスの下に表示されるプロンプトに従って、クリックして円弧の始点を定義します。この始点は、線分またはスプラインの端点です。

- カーソルをドラッグして、円弧の直径と中心を定義します。

- クリックして円弧の終点を定義します。
