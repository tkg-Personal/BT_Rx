# ゲートの入れ替え

ゲートは、コンポーネント定義で同じピン数と同じスワップ レベルを持つ同等のゲートである場合にのみ入れ替えることができます。

- [回路図デザイン]ツールバーで、**[修正] > [ゲートの入れ替え]** ![ゲートの入れ替えのアイコン](../images/icon/ecd/gate-swap.png) をクリックします。

- 入れ替える最初のゲートをクリックします。

- 入れ替える 2 番目のゲートをクリックします。選択したゲートを入れ替えることができない場合、エラー メッセージが表示されることがあります。
