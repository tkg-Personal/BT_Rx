# ネットまたはトレースをマイタ接続する

ネットまたはトレースのパスを移動します。

- [回路図デザイン]ツールバーで **[やり直し] > [マイタ]** をクリックするか、[PCB 設計]ツールバーで **[やり直し] > [マイタ]** ![マイタのアイコン](../images/icon/ecd/miter.png) をクリックします。

- [マイタ]ダイアログで、角度付きのネットまたはトレース用の直線のマイタ ワイヤ、または曲線のネット用の丸いマイタ ワイヤのいずれかを選択します。丸いマイタ ワイヤを使用する場合は、必要に応じて曲線の[半径]値を設定します。

- ネットまたはトレースをクリック アンド ドラッグして移動し、他のネットまたはトレースの移動を続行します。
