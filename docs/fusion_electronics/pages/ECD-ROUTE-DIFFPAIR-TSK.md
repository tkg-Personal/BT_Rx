# 差動ペアを配線する

信号のペアの名前は、一方に接尾辞 _p、他方に接尾辞 _n が付くことを除いて同じです。2 つの信号は、可能な限り近い位置に平行に配線されます。また、同じ長さか、非常に近い長さになるように配線されます。

- [PCB 設計]ツールバーで、 **[差動ペアを配線]** ![手動配線のアイコン](../images/icon/ecd/route-differential-pair.png) をクリックします。エアワイヤのペアがハイライト表示されます。

- [差動ペアを配線]ダイアログで、選択されているオプションを確認および修正します。

**[トレースの曲げ]**は、トレースの新しい各セグメントの角度または形状をコントロールします。[Ctrl]キーを押しながら右クリックすると、オプションを切り替えることができます。

- ダイアログの下半分にあるコントロールを使用して、トレースおよびビアの物理プロパティを指定します。トレース幅は、コマンド ライン フィールドに数値を入力することによっても変更できます。

- いずれかのエアワイヤをクリックすると、両方とも選択されて配線が開始されます。

- カーソルをドラッグして最初のルーティング セグメントを開始し、クリックしてコーナーまたは曲げを配置し、作業を続行します。

- 最後のセグメントを削除する場合は、[Backspace]を押します。

- 任意の時点で、[Enter]を押すことで自動的にトレースを完了することができます。
