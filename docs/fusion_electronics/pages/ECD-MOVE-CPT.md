# オブジェクトを移動する

[移動]をクリックすると、[移動]ダイアログが開き、次のオブジェクト配置オプションが表示されます。

- 回転角度を 90 度ステップで選択します。

- マウスの右ボタンをクリックすると、90 度単位で反時計回りに回転します。

- [ミラー]をクリックして、コンポーネントを左から右に反転します。

- 3 ボタン マウスを使用している場合は、マウスの中ボタンをクリックしてオブジェクトを反転させます。プレビューを更新するには、マウスの移動が必要になる場合があります。

- 矢印キーを使用して、グリッド増分を移動します。[Alt] + 矢印キーを使用して、交互グリッド増分を使用して移動します。

これらのオプションは、移動するオブジェクトを選択する前または選択した後にアクティブにすることができます。
