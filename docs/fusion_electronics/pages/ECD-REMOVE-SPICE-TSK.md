# SPICE モデルを削除する

SPICE モデルは、そのモデルが回路デザインでマッピングされている場合、シミュレーション回路デザインのコンポーネントから削除できます。モデルがライブラリでマッピングされている場合、そのライブラリでコンポーネントを編集する必要があります。

## 回路図回路の場合

- [回路図デザイン]ツールバーで、**[シミュレート] > [モデルを削除]** ![モデルを削除のアイコン](../images/icon/ecd/remove-sim-model.png) をクリックします。

- SPICE モデルを削除するコンポーネントをクリックします。

## ライブラリ コンポーネントの場合

- [ライブラリ デバイス]ツールバーで、**[モデルを削除]** ![モデルを削除のアイコン](../images/icon/ecd/remove-sim-model.png) をクリックします。

- SPICE モデルを削除するコンポーネントをクリックします。
