# オブジェクトの位置と情報

**[表示] ** ツールは、キャンバス内のオブジェクトをハイライト表示して、その位置を確認できるようにします。オブジェクトの詳細は、ステータス バーに一覧表示されます。ツールで信号とネット全体をハイライト表示することができます。バスを選択すると、そのバスに属するすべてのネットもハイライト表示されます。

**Forward&Back Annotation** が自動的にアクティブになると、このツールで基板内に配置したオブジェクトは回路図でもハイライト表示され、その逆も同様です。
