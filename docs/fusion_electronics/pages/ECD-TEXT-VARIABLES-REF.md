# テキスト変数

| 
 | 
 | 
 |

| 
**変数** | 
**意味** | 
 |

| 
>NAME | 
コンポーネント名(最終的には + ゲート名)。下記の「注 1」を参照 | 
 |

| 
>VALUE | 
コンポーネント値/タイプ、「注 1」を参照 | 
 |

| 
>PART | 
コンポーネント名、「注 2」を参照 | 
 |

| 
>GATE | 
ゲート名、「注 2」を参照 | 
 |

| 
>MODULE | 
モジュール名(モジュール シートのみ) | 
 |

| 
>SHEET | 
1/3 という形式の回路図のシート番号、「注 3」を参照 | 
 |

| 
>SHEET_HEADLINE | 
シート説明のヘッドライン | 
 |

| 
>SHEET_TOTAL | 
>SHEETNR_TOTAL/>SHEETS_TOTAL という形式の階層回路図のシート番号 | 
 |

| 
>SHEETS | 
シートの合計数、「注 3」を参照 | 
 |

| 
>SHEETS_TOTAL | 
モジュール シートを含めたシートの合計数 | 
 |

| 
>SHEETNR | 
現在のシート番号、「注 3」を参照 | 
 |

| 
>SHEETNR_TOTAL | 
モジュール シートを含めた現在のシート番号 | 
 |

| 
>ASSEMBLY_VARIANT | 
アセンブリ バリアントの名前 | 
 |

| 
>DRAWING_NAME | 
図面名 | 
 |

| 
>LAST_DATE_TIME | 
最終変更時間 | 
 |

| 
>PLOT_DATE_TIME | 
印刷作成時間 | 
 |

注

1) パッケージと記号のみ
2) 記号のみ
3) 記号または回路図のみ

文字 > で始まる文字はいずれも、属性のプレースホルダ テキストとして解釈されます。ATTRIBUTE コマンドを参照してください。
