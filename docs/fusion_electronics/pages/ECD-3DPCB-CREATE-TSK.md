# 独立した PCB を作成する

- **[ファイル] > [新規デザイン]** をクリックします。

- [デザイン]作業スペースで、**[作成] > [スケッチを作成]**をクリックし、スケッチ ツールを使用して 2D ボードのアウトラインを作成します。

- ツールバーの右端にある**[スケッチを終了]**をクリックします。

- **[作成] > [PCB を作成] > [独立した PCB を作成]** ![3D PCB を作成のアイコン](../images/icon/pcb/create.png) をクリックして、スケッチを選択します。

- または、既存のソリッドの面を選択することもできます。

- **[独立した PCB を作成]**ダイアログで、必要に応じて原点を設定し、X 軸を指定するか、X 軸または Z 軸を反転します。

- **[円を NPTH に変換]**をオンにして、スケッチ円から非めっき貫通穴を作成します。

- このチェック ボックスをオンにした後に、指定した直径以下の円を NPTH に変換するには、**[最大 NPTH 直径]**値を設定します。

- **[独立した PCB を作成]**ダイアログで、**[OK]**をクリックします。3D PCB が作成され、作業スペースが Electronics 3D PCB に変わります。ここで 2D PCB にリンクして、ボードに穴を追加できます。
