# [形式をコピー]を使用する

- 回路図または基板を開いた状態で、**[修正] > [形式をコピー]** ![形式をコピーのアイコン](../images/icon/ecd/ecd-copy-format.png) をクリックします。

- プロパティのコピー元のコンポーネントまたはネットをクリックします。ダイアログが開き、コピーするプロパティ(コンポーネントの角度、ネットの幅やスタイルなど)を選択できます。

- プロパティを適用する、同じタイプのオブジェクトをクリックします。
