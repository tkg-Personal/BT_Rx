# チュートリアル: PCB レイアウトを作成する

プリント基板(PCB)レイアウト プロセスは、芸術であり科学でもあります。100 人のエンジニアに回路図を渡すと、それぞれに独自の工夫が施された 100 種類の PCB レイアウトが戻ってくることになります。

このチュートリアルでは、次の手順を使用して、回路図デザインから始めて、PCB を作成します。

- PCB 形状を定義します。

- コンポーネントを配置します。

- 接続を配線します。

![完成した PCB](../images/tutorial/pcb-layout/pcb-from-schematic.png)
