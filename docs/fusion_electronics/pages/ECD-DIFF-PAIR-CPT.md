# 差動ペアを使用する

差動ペアは、可能な限り近い位置に平行に配線される信号のペアです。また、同じ長さか、非常に近い長さになるように配線されます。これは、通常、回路基板のノイズ耐性の改善に使用されます。

このペアは、最初に同じネット名(正の場合は接尾辞 _p、負の場合は接尾辞 _n が付きます)を持つ回路図で定義されます。回路基板のデザインでは、この命名規則は差動ペアを示すものとして認識され、**[差動ペアを配線]**コマンドを使用して一緒に配線できます。
