# 固定パネルをスタックまたはタブ付けする

固定パネルを、ニーズに合わせて[電子デザイン]作業スペースに配置します。

## スタック パネルに変更するには

   ![タブ パネル](../images/overview/ecd-panel-tabs.png)

- パネルのタイトル バーで、任意のタブを右クリックし、 **[スタックに変換]** ![スタック パネルに変換のアイコン](../images/icon/ecd/info-panels-stack.png) を選択します。

## スタック パネルからタブ パネルに変更するには

   ![タブ パネル](../images/overview/ecd-panels-stacked.png)

- 任意の固定パネルで、パネルのタイトル バーを右クリックし、 **[タブに変換]** ![タブ パネルに変換のアイコン](../images/icon/ecd/info-panels-tabbed.png) を選択します。
