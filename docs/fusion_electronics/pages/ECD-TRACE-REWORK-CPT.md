# トレースの編集と再作業

次のツールは機能上関連しており、多くの場合、効率的にトレースを編集するために一緒に使用されます。これらのツールは、障害物の周囲または希望するパスにトレースを配線する場合に役立ちます。

**[マイタ]** ![マイタのアイコン](../images/icon/ecd/miter.png) は、セグメントを追加せずにネットまたはトレースのパスを変更するために使用します。セグメントを 1 回クリックすると新しいルートにジャンプしますが、最適ではない可能性があります。ルートを動的にコントロールして、より予測可能な結果を得るために、[マイタ]を使用してクリック アンド ドラッグする方法をお勧めします。

**[分割]** ![分割のアイコン](../images/icon/ecd/split.png) は、1 つまたは複数のセグメントをネットまたはトレースに追加するために使用します。パスの分割に関するスキルを高めるために、特に曲線を使用する場合は、いくつかの実験が必要です。

**[スライス]** ![スライスのアイコン](../images/icon/ecd/slice.png) は、選択した点でまたは点の左側または右側に、パス セグメントを分割およびリップアップするために使用します。スライスは、類似したルートをたどる複数のパスにわたって切断できます。トレースは、手動での再配線用のエアワイヤに戻ります。左右のセグメントのリップアップは、スライス線の方向に基づいています。始点に立ち、終点の方を向いていると想像してみてください。この方向では、右側のセグメントは自分の右側にあります。

**[リップアップして配線]** ![リップアップして配線のアイコン](../images/icon/ecd/ripup-&-route.png) では、2D 基板または 3D 基板上でコンポーネントを移動した後で乱れた状態のトレースが、1 回のクリックで自動的にクリーンアップされます。通常は、すべてまたはほとんどのクリーンアップが実行されますが、追加の手動作業が必要な場合もあります。
