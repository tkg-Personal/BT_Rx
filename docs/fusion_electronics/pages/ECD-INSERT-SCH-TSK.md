# 回路図シートを回路図ドキュメントにインポートする

- 回路図ドキュメントで、 **[デザイン]** タブの**([配置] > [回路図を挿入])** ![回路図を挿入のアイコン](../images/icon/ecd/insert-schematic.png) をクリックします。

- **[参照]** ダイアログで、挿入する回路図に移動して選択します。

- **[選択]** をクリックします。  

- **[ファイルから貼り付け]** ダイアログでは、インポートしたネットの名前を変更したり、既定値を受け入れることができます。

- **[OK]** をクリックします。

インポートした回路図シートが、現在の回路図ドキュメントに追加されます。
