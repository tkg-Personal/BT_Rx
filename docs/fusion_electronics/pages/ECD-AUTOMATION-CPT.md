# 自動化スクリプト

Electronics では、多数のユーザ言語プログラム(ULP)が提供され、CAM ジョブをスクリプトに変換して DXF データにエクスポートする、またはパレットの色を設定するなどの一般的なタスクを自動化するいくつかのスクリプトが用意されています。これらのスクリプトにアクセスするには、ドキュメントを開いた状態で[自動化]をクリックすると、ULP およびスクリプトのアイコンが表示されます。それぞれのアイコンを使用して、使用可能なスクリプトのフォルダを開きます。

![自動化パネル](../images/how-to/automation-menu.png)

独自の ULP の記述方法については、リファレンスのトピック「ユーザ言語プログラミング」を参照してください。
