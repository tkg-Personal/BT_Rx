# 電子デザイン部品表の列をカスタマイズする

電子デザイン部品表に表示される列をカスタマイズし、ワークフローのニーズに合わせて列の順序を変更することができます。

## 表示される列を選択する

- [電子デザイン]作業スペースで、 **[部品表]** ![BOM のアイコン](../images/icon/bom-icon.png) を開きます。

- 部品表の右上にある **[列の設定]** ![列の設定アイコン](../images/icon/hig/settings.png) をクリックします。

 ![部品表の列](../images/concepts/bom-columns.png)

- 表示または非表示にする列名を選択または選択解除します。

- (省略可能)開いているドキュメントに定義したカスタム属性を選択します。

## 列の順序を変更する

- 部品表で、列ヘッダを左または右にドラッグして順序を変更します。
