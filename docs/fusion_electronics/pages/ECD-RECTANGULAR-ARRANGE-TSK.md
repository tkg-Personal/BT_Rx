# オブジェクトを長方形に配置する

選択したオブジェクトを XY グリッドに配列します。

- 開いている電子デザイン回路図または 2D PCB ドキュメントで、配置するオブジェクトを選択します。

- **[デザイン]** タブで、**([修正] > [配置])** ![配置のアイコン](../images/icon/ecd-arrange-rect.png) をクリックして、 **[配置]** ダイアログを開きます。

- **[配置]** ダイアログの **[形状]** セクションで、 **[長方形配置]** ![長方形配置のアイコン](../images/icon/ecd-arrange-rect.png) をクリックします。

- キャンバス内をクリックして始点を指定するか、ダイアログ オプションを使用します。

- **[配置]** ダイアログで、列数、X 方向の間隔、X 方向を指定し、 **[Enter]** を押して新しい値の効果をプレビューします。

- 行数、Y 方向の間隔、Y 方向を指定し、 **[Enter]** を押して新しい値の効果をプレビューします。

- キャンバス内をクリックするか、ダイアログの値を変更して、 **[Enter]** を押し、さらに形状を調整します。

- 形状と位置が適切であれば、 **[完了]** をクリックします。
