# QuickRoute Signal

- [PCB 設計]ツールバーで、 **[QuickRoute Signal]** ![QuickRoute Signal のアイコン](../images/icon/ecd/quick-route-signal.png) をクリックします。

- [QuickRoute Signal]ダイアログで、配線するレイヤー、およびトレースとビアの属性を確認または選択します。

- 配線する信号をクリックします。

- 信号のすべてのセグメントが正常に配線されていることを確認します。
