# オートルータを使用する

- [PCB 設計]ツールバーで、 **[QuickRoute] > [オートルータ]** ![オートルータのアイコン](../images/icon/ecd/autorouter.png) をクリックします。

- [オートルータのメイン セットアップ]ダイアログで、使用する信号レイヤーの推奨配線方向を設定します。

– 水平

- | 垂直

- / 45°の対角

- \ 135°の対角

- * なし

- auto 自動じどう

- 作用力のパラメータ、ルーティング グリッド、およびトポロジのルータである TopRouter を使用するかどうかを指定します。

- 配線するエアワイヤのサブセットを選択するには、**[選択]**をクリックしてから、デザインのエアワイヤをクリックします。

- **[続行]**をクリックして、[ルーティング バリアント]ダイアログを開きます。配線後、複数バリアントのルーティング パスの進行状況がここで示されます。それぞれに完了度を示すパーセント値が示されます。

- 必要に応じて、**[>>]**をクリックしてさらに設定を表示し、バスの使用、リップアップ操作の最大数、最適化の実行回数、コスト要因に関する設定を変更します。

- 設定が完了したら、**[開始]**をクリックしてオートルータを実行します。
