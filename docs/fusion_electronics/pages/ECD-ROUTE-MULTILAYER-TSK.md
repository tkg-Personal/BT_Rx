# 複数の画層間で配線する

多層配線に必要な設定を有効にした後は、配線の作成を開始できます。

- (**[デザイン] > [配線]** ![配線のアイコン](../images/icon/ecd/route-manual.png))をクリックして[手動配線]ダイアログを開き、キャンバス内のコンポーネントをクリックして配線を開始します。

- コンポーネントから離れた場所にマウス カーソルを置くと、接続の描画が開始されます。

 ![配線の開始](../images/how-to/route01.png)

始点

- カーソル

接続を描画する際、実際の基板上での接続をブロックするようにコンポーネントを交差させることはできません。

- 別のレイヤーに配線する位置で、マウスの中央ボタンをクリックして[レイヤー]ダイアログを開きます。

 ![レイヤー ダイアログ](../images/how-to/layer-dialog.png)

- 配線先のレイヤーを選択し、**[OK]**をクリックします。

 Electronics は、ルートで接続される各レイヤーのラベルを付けた電位ビアを設定します(この例では、「1-2」)。

 ![最初のビア](../images/how-to/route02.png)

 ユーザがクリックするまでビアは作成されません。クリックする前に、スペース バーを押して、他の可能性のあるレイヤー配線オプションを切り替えることができます。これには、ビアを作成せずに描画を続行するオプションも含まれます(この図の 3 番目の部分)。

 ![ビアを切り替える](../images/how-to/route03.png)

- クリックして、目的のビアを作成します。

 ![新しいビア](../images/how-to/route04.png)

- 描画を続けるには、マウス カーソルを新しいビアから離します。

 ![新しいビアを描画する](../images/how-to/route05.png)

- 必要に応じて他のビア点を追加し、完了したら[Esc]を押します。

 ![次のビア](../images/how-to/route06.png)

 接続線の色は、どのレイヤー上に配置されているかを示します。たとえば、上の図では、青色の接続はレイヤー 2 上にあります。各接続点は、元の接続の開始レイヤーとそのビアのターゲット レイヤーを示します。
