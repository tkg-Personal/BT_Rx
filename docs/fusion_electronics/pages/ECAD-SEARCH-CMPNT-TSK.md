# 一般公開されているコンポーネントを検索する

**電子デザイン ライブラリ マネージャ**を使用して、一般公開されている数百のライブラリを検索し、デザインに必要なコンポーネントを見つけます。

## 前提条件

- [電子デザイン ライブラリ マネージャを開く](ECD-OPEN-LIBMGR-TSK.htm)

## 手順

- ライブラリ マネージャで、 **[パブリック ライブラリ]** タブをクリックします。

- 左上の検索ボックスに、使用するコンポーネントの名前を入力します。

 次の条件で検索できます。

**ライブラリ名:** コンポーネント ライブラリの名前。

- **コンポーネント名:** 必要な特定のコンポーネント。

- **パーツ名:** バリアントと属性セットを含む完全なコンポーネント名。

- 検索結果を確認して、検索文字列に一致するライブラリを見つけます。

- コンポーネントをプレビューするには、ライブラリを選択し、右側のパネルの矢印ボタンを使用します。

ライブラリをまだダウンロードしていない場合は、 **[ダウンロード]** ![ライブラリのダウンロード アイコン](../images/icon/ecd/lbr-download.png) をクリックしてプレビューを表示します。

- **[アクティブ化]** 切り替えをクリックして、ライブラリとそのコンポーネントを配置パネルに追加します。

## 次のステップ

- **デザインでコンポーネントを使用する:** アクティブ化されたコンポーネントは、配置パネルに表示され、デザインで使用できるようになります。

- **カスタム バージョンを作成する:** パブリック コンポーネントにカスタム変更を加えるには、[ライブラリを分岐](ECAD-FORK-PUBLIC-LBRS-TSK.htm)して、現在のハブにカスタム コピーを作成します。
