# 新しいデザイン ドキュメントの使用を開始する

Fusion を起動すると、[ホーム]タブが開き、新しいプロジェクトを作成したり、既存のプロジェクトを開くことができます。既定では、無題の新しいデザイン ドキュメントが作成されます。

- Fusion ウィンドウの上部にある **[無題]** タブをクリックして、ビューを切り替えます。

- Fusion ウィンドウの左上隅で、 **[データ パネルを表示]** ![データ パネルのアイコン](../images/how-to/data-panel-icon.png) をクリックしてデータ パネルを開きます。

- 電子デザイン用のプロジェクトがない場合は、新しいプロジェクトを作成します。

- 各デザインのフォルダを作成します。

- Fusion ウィンドウの上部で、 **[ファイル]** ![新規ファイルのアイコン](../images/icon/common/file.png) **> [新規電子デザイン]** をクリックして、空白の新しい電子デザイン ドキュメントを作成します。

このドキュメントから、回路図、2D PCB、および 3D PCB を作成できます。

- **[共通]** タブで、左側の **[新しい回路図]** ![新しい回路図](../images/icon/ecd/new-schematic.png) をクリックして空白のシートから開始するか、 **[回路図を参照]** ![回路図を参照](../images/icon/ecd/reference-schematic.png) をクリックして既存の回路図にリンクします。

![[作成]パネル](../images/overview/create-panel-common.png)

図 1 は、 **[共通]** タブの新しいドキュメント コマンドを示しています。

- **[共通]** タブで、 **[新しい PCB]** ![新しい回路図](../images/icon/ecd/new-pcb.png) をクリックして最初から開始するか、 **[PCB ドキュメントを参照]** ![新しい回路図](../images/icon/ecd/reference-pcb.png) をクリックして既存の PCB にリンクします。

- **[保存]** をクリックして、デザインおよび関連ドキュメントに名前を付けます。
