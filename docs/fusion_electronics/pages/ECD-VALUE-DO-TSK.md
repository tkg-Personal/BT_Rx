# コンポーネント値を変更する

- [PCB ドキュメント]ツールバーで、**[値]** ![値のアイコン](../images/icon/ecd/value.png) をクリックします。

- コンポーネントのオブジェクト ハンドルをクリックします。ダイアログに現在の値がハイライト表示されます。

![値変更のアニメーション](../images/how-to/ecd-chg-comp-value.gif)

- コンポーネントの新しい値を入力し、[OK]をクリックします。
