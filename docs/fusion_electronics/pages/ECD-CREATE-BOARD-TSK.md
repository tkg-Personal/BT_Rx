# 2D PCB ドキュメント

関連する回路図の作業を開始すると、いつでも 2D プリント基板(PCB)を作成することができます。

まず、回路図のコンポーネントが配置された空の基板が生成され、エアワイヤと呼ばれる白い線で結合されます。次に示すように、コンポーネントを基板にドラッグします。

![コンポーネントのドラッグの GIF](../images/how-to/drag-component.gif)

ボード上のコンポーネントを移動すると、「ratsnest」というアルゴリズムによって、さまざまな接続点の間で可能な限り直接にエアワイヤ パスが最適化されます。

コンポーネントの配置グリッドは、既定で 50 mil (1.27 mm)に設定されています。別の配置グリッドを使用する場合は、ウィンドウの下部にある[グリッド]アイコン ![グリッド アイコン](../images/icon/ecd/grid-settings.png) をクリックします。

両方のファイルが開いている場合、基板は **Forward&Back Annotation** エンジンによって回路図にリンクされます。一方のファイルに対して行った変更は、他方のファイルにも自動的に適用されます。

回路図から基板を生成した後、続けて回路図にコンポーネントを配置すると、基板内の参照パッケージがレイアウト エディタの現在のグリッド設定内に配置されます。

> **Note:** 
注:

 回路図の編集時に 2D PCB が開かれていない場合、**Forward&Back Annotation** は機能せず、2D PCB ドキュメントと回路図ドキュメントが同期されません。電気的ルール チェック(ERC)によって提供されるエラー メッセージを参考に、手動で相違点を解決することができます。
