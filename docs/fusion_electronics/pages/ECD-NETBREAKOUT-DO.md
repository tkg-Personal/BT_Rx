# ネットを部分切断する

- [回路図デザイン]ツールバーで、**[接続] > [ネットを部分切断]** ![ネットを部分断面アイコン](../images/icon/ecd/net-breakout.png)を選択します。**[ピンを部分切断]** ダイアログが表示されます。

- 使用するピンの命名方法を指定します。

- [最小のワイヤ長]を指定します。値は 3 と 50 の間の整数でなければなりません。

- **[OK]** をクリックします。

- キャンバスで、部分切断するピンを選択します。ピンは、任意のコンポーネントから選択できます。
