# 3D PCB

3D PCB は、プリント基板のソリッド モデル表現を格納する Fusion デザイン ドキュメントです。[デザイン]作業スペースで表示および編集するボードのアウトライン、レイヤー スタックの厚さ、コンポーネントが表示されます。PCB をコンテキスト内で配置して検証できます。筐体内に収まるようにしたり、機械パーツのクリアランスを確認したり、変更を 2D PCB と同期したりすることができます。

3D PCB は電子デザインから生成され、リンクされたままになるため、ECAD と MCAD の間で変更をプルすることができます。

## 3D PCB に含まれるもの

- **ボード アウトライン スケッチ:** PCB アウトラインを定義するための推奨される方法です。

- **ボードのボディ:** スタックアップから厚さを表示します。

- **3D パッケージ:** コンポーネントの MCAD 対応ジオメトリです。

- **参照キャンバス:** パフォーマンス指向の作成オプションを選択した場合の銅箔とマスクの視覚的表現です。

## 一般的な活用事例

- **早い段階で筐体内に収まるようにする:** 詳細なレイアウトを決定する前に、高さの予算と禁止領域を設定します。

- **コネクタとポートの位置合わせ:** PCB マウント I/O をインダストリアル デザインの開口部と位置合わせします。

- **熱および機械的な計画:** ヒート シンク、シールド、およびスタンドオフを配置し、エアフローとアセンブリのクリアランスを確保します。

## アセンブリの統合

3D PCB を機械アセンブリに統合するには、主に次の 2 つの方法があります。

- **機械的なコンテキストでボードを定義する:** デザインのスケッチまたは面から開始して、ボードを配置する位置とサイズを決定します。3D PCB を作成し、それを基に 2D PCB を作成します。

- **3D PCB をアセンブリに挿入する:** 製品アセンブリ内に 3D PCB を挿入するか、インプレイス編集します。

## 制限事項および留意事項

- **精度対速度:** 銅箔をキャンバスとしてレンダリングすると作成速度は高まりますが、これは物理的なジオメトリではないため、正確な質量や体積の解析が必要な場合は、完全なソリッドを使用します。

- **パッケージの完全性:** 3D パッケージのないパーツは単純なボックスとして表示されます。信頼性の高いクリアランス チェックを行うには、3D パッケージを割り当てます。

- **ドキュメント管理:** 2D ドキュメントと 3D ドキュメントは共に開いておき、頻繁に同期します。
