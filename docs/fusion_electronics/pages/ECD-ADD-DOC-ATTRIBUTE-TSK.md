# キャンバスにドキュメント属性を追加する

- 回路図または 2D PCB ドキュメントの **[描画]** パネルで、 **[テキスト]** ![テキストのアイコン](../images/icon/ecd/text.png) コマンドをクリックします。

- ダイアログの**[テキストを入力]**領域をクリックします。

- より大きいを表す記号「>」の後に属性名を入力します。たとえば、>COMPANY です。

- その他のテキスト パラメータを指定します。

- キャンバス上にカーソルを配置します。属性値のプレビューが表示され、キャンバスに配置するためにカーソルにアタッチされます。

- クリックしてドキュメント属性テキストを配置します。必要な数だけ属性のインスタンスを配置します。

- **[OK]**をクリックします。
