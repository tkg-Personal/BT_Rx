# 機械設計から PCB を作成する

**[デザイン]** 作業スペースの **[PCB]** ツールを使用して、Fusion のスケッチおよびモデル内の機械的ジオメトリを参照する PCB を作成します。

## リンクされた PCB を作成する

- これにより、外部コンポーネントとして PCB が作成され、アクティブなデザインに挿入されます。これは、PCB を機械的アセンブリと統合する場合に推奨される方法です。ボード アウトラインは PCB アウトライン スケッチで描画する必要があり、アセンブリ コンテキストに追加されたオブジェクトを参照することができます。

- Fusion は、新しいデザインをその親と同じ場所に自動的に保存します。

親コンポーネントが選択されていない場合、既定の場所は**データ パネル**のアクティブなプロジェクトとフォルダに一致します。

- 親コンポーネントがまだ保存されていない場合、既定の場所は**データ パネル**のアクティブなプロジェクトとフォルダに一致します。

- 新しいコンポーネントは現在のデザインに参照され、**ブラウザ**では、選択した親コンポーネントの下に表示されます。

- **[アクティブ化]** 設定がオンの場合、コンポーネントが自動的にアクティブになり、 **[インプレイス編集]** セッションが開始され、現在のデザインのコンテキストで新しいコンポーネントの作業を開始できるようになります。

## PCB を派生

- これは、アクティブなデザインのスケッチ プロファイルから PCB を派生し、3D PCB でアウトライン スケッチの代わりにこの外部スケッチを使用します。このオプションを使用するには、ソース デザインを保存する必要があります。

- 派生した PCB およびソース デザインは、アセンブリのコンポーネントとして使用するために、3 番目のデザインに追加する必要があります。

## リンクされていない PCB を作成する

- ソース デザインにリンクされていないスケッチまたは面から PCB を作成します。
