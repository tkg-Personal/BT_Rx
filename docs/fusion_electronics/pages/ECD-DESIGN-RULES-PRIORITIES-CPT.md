# デザイン ルールの優先度

Fusion は各カテゴリのデザイン ルールをその優先度に従って評価します。クリアランスのチェック、銅箔幅の値、またはビア ドリル サイズを必要とする手動配線など、インタラクティブなワークフロー中に、最高優先度ルールまたは最小値ルールから開始します。

ルールの優先度を設定して、厳密なルールを最初に、その後により厳密でない範囲を適用します。

すべての範囲 -

- ルールで許可されているオブジェクトと一致する「すべての」範囲の使用を避けます。たとえば、銅箔クリアランスの場合、既存の銅オブジェクトに一致するものです。

- 「すべて」の範囲を対象とするルールが必要な場合は、そのルールに最低優先度、最小値、および推奨値を設定します。

- ライブ/手動 DRC の場合、すべてのクリアランス ルールを評価します。優先度の高いルールに最大値を設定します。これにより、厳密なルールのみがオブジェクトのサブセットと一致する場合に違反が生成されます。

- 銅箔幅やドリル サイズなどの単項ルールの場合、最初に一致したルールの後にチェックが停止します。これらのルールの優先順位は、単項ルールの場合と同じように処理します。

- **一般ルール**の値は、最小のカスタム ルールの値よりも小さくします。または、使用可能であればチェック ボックスを使用して **[一般ルール]** を無効にします。こうすると、カスタム ルールのみが使用されます。
