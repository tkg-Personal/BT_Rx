# デザイン基本設定

**デザイン基本設定**を使用して、基板デザインに規格を適用します。これらには、トラック幅、異なる電位間のクリアランス値、ドリル穴の直径、および文字サイズを含めることができます。これらをデザイン ルール ファイル**(EDRU)**として保存することができます。

基板とその製造に関連するすべてのパラメータは、**デザイン基本設定**で指定されます。正しい規格値については基板の製造元に確認してください。データが指定されない場合、Fusion ではパラメータが自動的に指定されます。

基板のデザイン後にデザイン ルールの順守を確認するには、デザイン ルール チェック(DRC)ルーチンを実行します。これにより、見つかったエラーが特定されます。
