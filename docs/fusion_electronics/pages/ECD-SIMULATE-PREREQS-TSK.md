# シミュレーションのセットアップ タスク

[シミュレーション]コマンドを使用するには、いくつかの前提条件を満たす必要があります。

- *ngspice-digital* ライブラリと *ngspice-simulation* ライブラリのコンポーネントを使用して、回路図を作成します。これらのコンポーネントは既に SPICE モデルにマッピングされており、正確なシミュレーションを行うにはこの設定が必要です。

- 抵抗、静電容量などの適切な値を回路図内のデバイスに割り当てます。シミュレーションに適した場所に、電圧プローブまたはフェーズ プローブを追加します。

- [回路図デザイン]ツールバーで、**[シミュレーション] > [モデルを追加]**をクリックし、[モデルを追加]ダイアログで、SPICE モデルをマッピングしていないすべてのパーツにマッピングします。

- **[シミュレーション]** ![[シミュレーション]アイコン](../images/icon/ecd/simulate.png) をクリックします。

- [シミュレーション]ダイアログで、シミュレーションのタイプとして[動作点]、[DC スイープ]、[AC スイープ]、または[非定常]を選択します。

- シミュレーション タイプにその他の値を設定してから、**[シミュレーション]**ボタンをクリックします。
