# Fusion Electronics のナビゲーション バー ツール

ナビゲーション バーは、ドキュメント ツール バーでも使用できる、頻繁に使用するツールへのすばやいアクセスを提供します。ほとんどのツールは固定されていますが、作業中の電子デザイン ドキュメントに応じて変更されるものもあります。ナビゲーション バーは、[電子デザイン]ウィンドウの下部にあります。
