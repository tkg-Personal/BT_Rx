# 3D 回路基板を筐体に挿入する

**前提条件**:

- 3D 筐体デザイン ドキュメントが **[デザイン]** 作業スペースで開かれていること

- データ パネルが開いていること

 ![筐体](../images/how-to/eip/enclosure.png)

 図 1: **[デザイン]** 作業スペース内の筐体の例。基板ジオメトリをスケッチするための 4 つのスタンドオフ ポストの上に平面が表示されています。

**手順**:

- 筐体の 2D PCB をまだ作成していない場合は、 **[デザイン]** 作業スペースで、 **[ソリッド]** ツールバーのツールを使用して、モデルの筐体内に回路基板ジオメトリをスケッチします。

次の手順で使用するために、回路基板デザインのジオメトリを記録します。

- **[電子デザイン]** 作業スペースで、必要なジオメトリを使用して 2D PCBを作成します。

![2D PCB を編集](../images/how-to/eip/2d-pcb-1.png)

- **[2D PCB]** ドキュメントの **[デザイン]** ツールバーで、**[切り替え] > [3D PCB]**をクリックして、3D PCB デザイン ドキュメントを作成します。

![3D PCB](../images/how-to/eip/3d-pcb.png)

- Fusion ウィンドウの上部にあるタブを使用して、開いている筐体デザイン ドキュメントに切り替えます。

- **データ パネル**の 3D PCB ドキュメントを右クリックし、 **[現在のデザインに挿入]** を選択して基板を筐体デザイン ドキュメントに取り込みます。

![PCB を挿入](../images/how-to/eip/insert-3d-pcb.png)

- マニピュレータ ハンドルを使用して基板を移動および回転させ、正しい位置に配置することで、筐体内の開口部に対するコンポーネントの位置合わせを確認することができます。
