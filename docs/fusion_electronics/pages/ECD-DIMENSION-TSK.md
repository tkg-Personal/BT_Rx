# 寸法を追加する

- [PCB ドキュメント]ツールバーで、 **[寸法]** ![寸法のアイコン](../images/icon/ecd/dimension.png) をクリックします。

- [寸法]ダイアログで、追加する寸法線のタイプ、文字サイズ、および寸法ラベルのその他の属性を選択します。

- 角度寸法を追加するには、角度のタイプを選択した後で、角度の頂点をクリックし、一方の側をクリックし、円弧を他方の側または選択した角度計測値まで回転し、端点をクリックします。
