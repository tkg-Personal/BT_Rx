# ユーザ言語プログラムを実行する

- [回路図自動化]または[PCB オートメーション]のツールバーで、**[ULP を実行]** ![ULP を実行のアイコン](../images/icon/ecd/run-ulp.png) をクリックします。

- [ULP]ダイアログで、[名前]列と[説明]列を使用して、実行する ULP を検索します。たとえば、ex- で始まる ULP などです。

- ULP をダブルクリックして実行します。
