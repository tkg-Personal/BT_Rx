# アクティビティ 2: 2 つのデザインのバリアントのコンポーネントを比較する

このアクティビティでは、**Tubescreamer C Schematic** と Tuberscreamer Electronics Design の 2 番目の PCB のコンポーネントを比較します。

## 前提条件

- アクティビティ 1 を完了していること。

- **Tubescreamer C Board** は閉じ、**Tubescreamer C Schematic** は開いたままにします。

## 手順

- 必要に応じて **[データ パネルを表示]** をクリックし、 **[サンプル] > [電子設計サンプル] > [シンクロナイザのチュートリアル]** にナビゲートして、**Tubescreamer D Electronics Design** 基板を開きます。

- 前のアクティビティの場所に保存します。 

- 前のアクティビティから **Tubescreamer C Schematic** を開きます。

- **[検証] > [検証] > [同期]** ![同期のアイコン](../images/icon/ecd/synchronize.png) をクリックします。

- [シンクロナイザ]ダイアログで、左側の列のソースとして **Tubescreamer C Schematic** を残し、右側の列のターゲットとして **Tubescreamer D Electronics Design** を選択します。 

![[同期]ダイアログ](../images/tutorial/synchronizer/synchro-c-d.png)

この表示は、Tubescreamer D を作成するために Tubescreamer C に適用されたデザイン変更を調べるために役立ちます。 

> **Note:** 注: 列に含まれている情報が少ない場合、この基板または回路図のコンポーネントを含むライブラリが有効になっていないことを意味します。

- デザイン変更を確認したら、[同期]ダイアログで **[閉じる]** をクリックして、2 つのデザインを変更せずに残します。
