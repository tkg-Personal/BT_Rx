# ネットまたはトレースを分割する

- [回路図デザイン]ツールバーで **[やり直し] > [分割]** をクリックするか、[PCB 設計]ツールバーで **[やり直し] > [分割]** ![分割のアイコン](../images/icon/ecd/split.png) をクリックします。

- [分割]ダイアログで、曲げタイプ(角度付きまたは曲線)、およびマイタ タイプ(直線または丸)を確認または選択します。丸いマイタを使用する場合は、必要に応じて曲線の[半径]値を設定します。

- 分割を開始する位置でパスをクリックし、パスを新しい方向にドラッグします。

- 方向を変更する場所でクリックします。

- ダブルクリックして再配線を終了します。
