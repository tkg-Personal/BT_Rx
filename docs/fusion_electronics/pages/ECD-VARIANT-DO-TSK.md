# アセンブリ バリアントを作成する

- **[回路図]** > **[ドキュメント]** タブ ツールバーで、 **[アセンブリ バリアント]** ![バリアントのアイコン](../images/icon/variant-icon.png) をクリックします。または、 **[デザイン マネージャ]** の[アセンブリ バリアント]リストの右側にある、 **[アセンブリ バリアントを編集]** をクリックします。

- **[メイン アセンブリ バリアント]**ダイアログで、[新規作成]をクリックします。

- **[新規アセンブリ バリアント]**ダイアログで、新しいバリアントにわかりやすい名前を割り当てます。

- 必要に応じて、バリアントを選択して**[コピー元]**を選択し、新しいバリアントを作成します。

- **[OK]** をクリックします。

- **[メイン アセンブリ バリアント]**ダイアログのバリアントの下で、左側の列を使用して、コンポーネントを選択解除することによって削除します。

- **[値]** 列で、コンポーネント値に必要な変更を加えます。

- 完了したら、 **[OK]** をクリックします。
