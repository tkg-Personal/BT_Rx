# CAM ファイルをエクスポートする

- キャンバスで回路基板を開き、[製造]ツールバーで**[CAM エクスポート]** ![CAM エクスポートのアイコン](../images/icon/ecd/cam-export.png) をクリックします。

- [CAM エクスポート ファイル リスト]ダイアログで、次の詳細を確認します。

ファイルの保存場所として、そのファイル フォルダ パスを指定できるかどうか。

- すべてのファイルが存在し、使用できるファイル名を使用しているかどうか。

- CAM テンプレートが使用できるかどうか。これは、ダイアログの下部に一覧表示されています。

- これらの詳細が適切な場合は、**[OK]**をクリックし、[ZIP CAM アーカイブを作成]ダイアログで**[保存]**をクリックします。
