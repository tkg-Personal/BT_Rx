# シミュレーションのデジタル信号ソースを設定する

ソースは SPICE 互換デジタル ソースである必要があります。

- [回路図デザイン]ツールバーで、 **[シミュレート] > [デジタル ソース設定]** ![デジタル ソースのアイコン](../images/icon/ecd/digital-source-setup.png) をクリックします。  

- [デジタル ソース設定]ダイアログで、時間と N$2 の値のテーブルを設定するか、これらの値を持つ CSV ファイルをロードします。
