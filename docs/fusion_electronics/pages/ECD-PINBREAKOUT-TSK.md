# ピンを部分切断

![ピンを部分切断のアニメーション](../images/how-to/ecd-breakout-pins-anim.gif)

- コンポーネント ピンからネットを延長するには、[回路図デザイン]ツールバーで、 **[接続] > [ピンを部分切断]** ![ピンの部分切断を追加](../images/icon/ecd/pin-breakout.png) アイコンを選択します。

- コンポーネントを選択すると、使用可能なピンがリストされた状態で**[ピンを部分切断]**ダイアログが表示されます。

- コンポーネントが選択されていない場合は、ピンを部分切断するコンポーネントを選択します。**[使用可能なピン]**に拡張可能なピンが表示されます。

- **[使用可能なピン]**リストで、部分切断するピンを選択します。チェック ボックスをオンにするか、項目を選択して右クリックし、**[すべて選択]**をクリックします。ピンの選択解除でも同じように、**[すべて選択解除]**を選択します。

- **[ラベル形式]**ドロップダウンから、使用するピン ラベルの方法を選択します。

- 別のコンポーネントのピンを部分切断するには、次のコンポーネントを選択します。前の操作が適用され、次のコンポーネントのピンが**[使用可能なピン]**リストに表示されます。

> **Note:**   注: ピンの選択を解除するには、ピンの選択を解除するか、**[すべて]**を 2 回クリックします。

- 作業が終了したら、**[完了]**をクリックします。
