# リンクされていない PCB を作成する

ソース デザインにリンクされていないスケッチまたは面から PCB を作成するには、 **[リンクされていない PCB を作成]** ツールを使用します。

- **[デザイン]** 作業スペースで、 **[作成] > [PCB を作成] > [リンクされていない PCB を作成]** を選択します。

 **[リンクされていない PCB を作成]** ダイアログが表示されます。

- **[リンクされていない PCB を作成]** ダイアログで、プロファイルまたは平面を選択します。

- スケッチ プロファイルから点をクリックして、原点を選択します。

- スケッチ プロファイルからスケッチ線分またはエッジをクリックして、**X** 軸を選択します。

- **[X 軸を反転]** をクリックして、**X** 軸の方向を反転します。

- **[Z 軸を反転]** をクリックして、**Z** 軸の方向を反転します。

> **Note:**  注: Y 軸は直接選択されず、X 軸および Z 軸から推測されます。

- PCB ボディに円が穴として作成されないようにするには、 **[円を NPTH に変換]** のチェックボックスをオフにします。

- **[最大 NPTH 直径]** を指定して、穴の作成をこのサイズ以下に制限します。
