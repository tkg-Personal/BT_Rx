# QuickRoute ツールについて

選択した信号を開始から終了まで配線するには、**[QuickRoute Airwire]**を使用します。

同じ名前の信号セグメントのグループを配線するには、**[QuickRoute Signal]**を使用します。このコマンドの制限は、16 より多いセグメントを持つ信号では実行できないことです。

**[QuickRoute Multiple]**を使用して、チップの側面にある平行なパッドなどの関連する一連のパッドから、一連の信号を配線します。

最小数の曲げとセグメントで信号を最も単純なパスに再配線するには、**[QuickRoute Smooth]**を使用します。

関連するパッドのセットから信号のグループの配線をガイドするには、**[QuickRoute Guided]**を使用します。このオプションを使用して、基板上の別のコンポーネントの周囲を移動する場合などに、トレース コーナーが発生する場所を詳細にコントロールできます。また、ある画層から別の画層にトレースを進めることもできます。
