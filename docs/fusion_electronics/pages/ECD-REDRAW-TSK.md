# デザインを再描画する

エラー メッセージやその他のダイアログでデザイン ウィンドウが煩雑になり、デザインの表示がよく見えなくなっている場合は、このメニュー オプションを使用します。

[回路図デザイン]または[PCB 設計]ツールバーで、**[表示] > [再描画]** ![再描画のアイコン](../images/icon/ecd/zoom-redraw.png) を選択します。デザイン キャンバスが再描画されます。
