# QuickRoute Smooth を使用してトレースをスムーズにする

- [PCB 設計]ツールバーで、 **[QuickRoute Smooth]** ![QuickRoute Smooth のアイコン](../images/icon/ecd/quick-route-smooth.png) をクリックします。

- スムーズにして単純化するトレースをクリックします。配線をスムーズにすることができる場合は、即座に変更されます。変更が発生しない場合、配線は既に可能な限りスムーズになっています。
