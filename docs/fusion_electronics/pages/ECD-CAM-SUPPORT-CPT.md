# コンピュータ支援製造(CAM)のサポート

コンピュータ支援製造(CAM)とは、ソフトウェアを使用して工作機械や関連機械を制御し、ワークピースを製造することです。PCB 設計において、CAM とはデザイン データから製造データを生成するプロセスを指します。このデータは、ボードの製造とコンポーネントの組み立てに使用されます。

Electronics では、一般的な画層設定を行うボードに使用する一連の CAM テンプレート(ジョブ ファイル)が事前設定されています。CAM プロセッサは、現在の基板に合った CAM テンプレートを自動的に選択します。ただし、必要に応じて、カスタム ジョブ ファイルを作成してロードすることができます。

ビデオ: PCB エディタで CAM プロセッサを使用してプリント基板の製造データを生成する方法の概要。
