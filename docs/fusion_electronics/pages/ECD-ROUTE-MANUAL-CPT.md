# 電子デザインの手動配線

手動配線は自動配線よりも時間がかかりますが、設計者がボードの配線を完全にコントロールすることができます。これは、重要な信号を配線してから残りの信号を自動配線する場合によく使用されます。また、自動配線が困難なボードの密集した領域の配線にも手動配線が有効です。

手動配線ツールには次のものがあります。

- 手動配線

- 差動ペア配線

- 複数配線
