# BT Audio Rx — Bluetooth オーディオレシーバー PCB 仕様書

## 概要

Bluetoothで音楽を受信し、3.5mmイヤホンジャックからステレオ出力するポータブルデバイス。LiPoバッテリー駆動、USB-C充電。1枚のPCBに全回路を実装する。

---

## ブロック図

```
USB-C (5V) ──→ MCP73831 (充電IC) ──→ LiPo Battery (J3 JST PH)
                                           │
                                    DW01A + FS8205A (保護IC)
                                           │
                                    AP2112K-3.3 (LDO → 3.3V)
                                           │
                                       BM83 (Bluetooth)
                                        │        │
                                    AOHPL      AOHPR
                                        │        │
                                   R4+C7(LPF) R5+C8(LPF)
                                        │        │
                                     3.5mm Jack (Tip/Ring/Sleeve)
```

## 回路接続詳細

### 1. USB-C充電入力 (J1)

```
J1:VBUS ──→ U2:VIN (MCP73831 pin 4)
J1:GND  ──→ GND
J1:CC1  ──→ R1 (5.1kΩ) ──→ GND   ※UFPとして認識させる
J1:CC2  ──→ R2 (5.1kΩ) ──→ GND
```

### 2. 充電回路 (MCP73831)

```
U2:VIN  ← J1:VBUS (5V)
U2:VSS  → GND
U2:VBAT → BATT_RAW ネット (バッテリー+)
U2:STAT → R7 → LED4 (充電中: LOW → LED点灯)
U2:PROG → R3 (2kΩ) → GND  ※充電電流 ≈ 500mA
```

デカップリング:
- C1 (4.7µF): VIN-GND間
- C2 (4.7µF): VBAT-GND間

### 3. バッテリー保護回路 (DW01A + FS8205A)

```
BATT_RAW ──→ U3:VCC (DW01A pin 1)
BATT_RAW ──→ U3:TD  (DW01A pin 6) ※テスト/遅延ピン
U3:OD    ──→ Q1:G1  (FS8205A 過放電制御)
U3:OC    ──→ Q1:G2  (FS8205A 過電流制御)
U3:CS    ──→ Q1:S1/S2 (PACK_NEG_SENSE ネット)
U3:GND   ──→ GND (システムGND側)

Q1:D1/D2 ──→ BATT_RAW側
Q1:S1/S2 ──→ PACK_NEG_SENSE ──→ J3:pin2 (バッテリー-)
```

### 4. バッテリーコネクタ (J3)

```
J3:pin1 (+) ──→ BATT_RAW ネット
J3:pin2 (-) ──→ PACK_NEG_SENSE ネット
```

### 5. LDO (AP2112K-3.3)

```
U4:VIN ← BAT_IN (保護回路出力、BATT_RAW経由)
U4:EN  ← VIN に接続 (常時ON) or プルアップ
U4:GND → GND
U4:VOUT → VDD_3V3 ネット (3.3V)
U4:NC  → 未接続
```

デカップリング:
- C3 (10µF): VIN-GND間
- C4 (10µF): VOUT-GND間

### 6. Bluetoothモジュール (BM83)

```
U1:VDD系ピン ← VDD_3V3 (3.3V)
U1:GND系ピン → GND
U1:AOHPL     → R4 (1kΩ) → AUDIO_L ネット
U1:AOHPR     → R5 (1kΩ) → AUDIO_R ネット
```

デカップリング: C5, C6 (100nF) を VDD-GND間に配置
BM83のデータシートに従い、必要なプルアップ/プルダウン抵抗を追加すること。

### 7. オーディオ出力 RCローパスフィルタ

```
AUDIO_L ──→ R4 (1kΩ) ──┬──→ J2:Tip (L)
                        │
                   C7 (4.7nF) → GND

AUDIO_R ──→ R5 (1kΩ) ──┬──→ J2:Ring (R)
                        │
                   C8 (4.7nF) → GND

J2:Sleeve ──→ GND
```

**カットオフ周波数:** $f_c = \frac{1}{2\pi RC} = \frac{1}{2\pi \times 1000 \times 4.7 \times 10^{-9}} \approx 33.86 \text{ kHz}$

可聴域 (20Hz〜20kHz) を通過させ、高周波ノイズを除去。

### 8. LED

```
電源LED (LED3):
VDD_3V3 → R6 (1kΩ) → LED3:A → LED3:K → GND

充電LED (LED4):
J1:VBUS → R7 (1kΩ) → LED4:A → LED4:K → U2:STAT
※STAT=LOWで点灯 (充電中), STAT=Hi-Zで消灯 (充電完了)
```

## PCB設計方針

### 基板仕様
- **サイズ目標:** 40mm × 35mm 以下（なるべくコンパクトに）
- **層数:** 2層 (F.Cu + B.Cu)
- **銅箔厚:** 1oz (35µm)
- **板厚:** 1.6mm
- **表面処理:** HASL or ENIG

### 部品配置ガイドライン
- **J1 (USB-C):** 基板上端
- **J2 (3.5mmジャック):** 基板下端
- **J3 (バッテリーコネクタ):** 基板左端 (充電IC近く)
- **U1 (BM83):** 基板中央〜右側、**アンテナ部分を基板端に配置**（アンテナ下にGNDベタ・部品を置かない）
- **U2 (MCP73831):** J1の近く
- **U4 (AP2112K):** U1の近く
- **受動部品:** 親ICの近くにクラスタリング

### 配線ルール
- **電源トレース幅:** 0.3mm以上
- **信号トレース幅:** 0.15mm以上
- **クリアランス:** 0.15mm以上
- **オーディオ配線:** L/Rを対称配置、GNDと並走
- **GNDベタ:** B.Cu全面をGNDベタで埋める
- **ビア:** 電源はビア複数本でインピーダンス低減

## 既知の問題・注意点

1. **R8を削除すること** — 両端がBATT_RAWに接続されておりショート状態
2. **BM83のデータシートを必ず確認** — ピンアサイン、必須外付け部品、アンテナクリアランス
3. **技適** — BM83モジュールは技適取得済みだが、アンテナパターンを変更しないこと
4. **バッテリー保護回路の動作確認** — DW01A+FS8205Aのリファレンス回路とピン接続を再確認
5. **MCP73831のサーマルパッド** — SOT-23-5なのでサーマルパッド不要だが、放熱用GNDビアを推奨
